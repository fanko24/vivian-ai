# 家长指导：第3课 和AI做朋友

## 课前准备（最多3项）

1. 确认 ChatGPT 或豆包能正常对话
2. 测试"用小朋友能听懂的方式告诉我月亮为什么有圆有弯"看AI回答质量
3. 准备几个有趣的问题备用

## 课堂指导建议

1. **先对比再总结**：让孩子自己对比"模糊问题"和"好问题"的回答差异，自己总结出规律
2. **引导追问**：如果孩子不知道怎么追问，可以提示"AI说了什么你觉得有趣？对哪个部分想了解更多？"
3. **允许跑题**：对话不一定按计划进行，跟着孩子的兴趣走就好
4. **记录精彩对话**：帮孩子截图保存有趣的问答，以后可以回顾
5. **鼓励表达**：如果孩子不习惯打字，可以由家长代打，孩子口述

## 常见问题

**AI的回答太长太复杂怎么办？**
教孩子加上"请用小朋友能听懂的方式再讲一遍"。

**孩子不知道问什么怎么办？**
从孩子最近感兴趣的事物切入——动画片、动物、学校发生的事都可以。

**对话变得很无聊怎么办？**
换一个全新的话题试试，或者让孩子试试"请用讲故事的方式告诉我"。

## 延伸活动

1. 全家玩"好问题比赛"：一人出模糊问题，其他人帮它改成好问题，看谁改得最好
2. 让孩子用今天学的"好问题"方法去问家人一个问题，对比家人的回答和AI的回答
