# 第3课：AI画画师

## 课程信息
- 模块：模块一 · 认识AI朋友
- 时长：40分钟
- 主题：用文字生成图片，当一个小小AI画家

## 学习目标
1. 学会用"主体 + 细节 + 风格"描述想要的画
2. 能生成3张自己喜欢的AI画
3. 理解AI画画就像玩"猜画"游戏

## 准备清单
- [ ] 电脑或平板（已联网）
- [ ] ChatGPT（chat.openai.com）或豆包（doubao.com）已登录
- [ ] 画纸和彩笔（课后画画用）

## 时间安排
| 环节 | 时长 | 内容 |
|------|------|------|
| 魔法时刻 | 3分钟 | AI画出"穿宇航服的小猫在月球上" |
| 故事时间 | 5分钟 | 薇薇安用文字画画的故事 |
| 知识小课堂 | 7分钟 | 描述画面的秘诀 |
| 动手体验 | 20分钟 | 生成图片 + 尝试变化 |
| 总结分享 | 5分钟 | 展示最满意的作品 |

## 推荐工具
- 主要：[ChatGPT](https://chat.openai.com/)
- 备选：[豆包](https://www.doubao.com/)
