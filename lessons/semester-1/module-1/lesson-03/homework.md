# 课后作业

## 必做任务（10-15分钟）

### 生成3张"梦想世界"的画

想象一个你梦想中的世界，用AI画出3张图片！

比如：
- 梦想中的家是什么样子？
- 梦想中的宠物是什么样子？
- 梦想中去玩的地方是什么样子？

要求：
- 每张画都要用"三步法"来描述（主体 + 细节 + 风格）
- 把生成的图片保存下来（截图）
- 在每张画旁边写上你输入给AI的那句话

## 选做任务（15-20分钟）

### 用4张画讲一个小故事

用AI画4张图，连起来讲一个完整的故事！

1. 第1张：故事的开头（比如"从前有一只小兔子住在森林里"）
2. 第2张：发生了什么（比如"有一天它发现了一扇神秘的门"）
3. 第3张：最精彩的部分（比如"门后面是一个糖果世界"）
4. 第4张：故事的结尾（比如"小兔子和糖果朋友们开心地生活在一起"）

把4张画按顺序贴在纸上，在下面写上每张画的故事内容。
