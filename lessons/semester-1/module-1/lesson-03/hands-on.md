# 动手实践：AI画画师

## 30秒魔法时刻

看看AI能画出什么！

（家长在ChatGPT或豆包中输入："画一只穿着宇航服的小猫在月球上"）

几秒钟之后，一只穿着宇航服的可爱小猫就出现在月球上了！是不是超级酷？

---

## 练习一：简单描述 vs 详细描述

1. 输入简单描述："画一只小猫"
2. 看看AI画出了什么
3. 再输入详细描述："画一只橘色的胖小猫，毛茸茸的，戴着红色蝴蝶结，坐在花园的花丛里，卡通风格"
4. 比较两幅画，哪幅更像你想要的？

## 练习二：用"三步法"画你想要的画

用"主体 + 细节 + 风格"来描述：

1. 想一个你最想画的东西（动物、城堡、海洋……）
2. 用三步法写出描述
3. 输入给AI，等几秒钟看结果
4. 如果不满意，加更多细节再试一次

比如：
- 主体：一条鲸鱼
- 细节：蓝色的大鲸鱼，背上驮着一座小房子，在天空中飞
- 风格：梦幻的水彩画风格

## 练习三：同一个主题，不同风格

1. 选一个主题，比如"森林里的小兔子"
2. 先说："画森林里的小兔子，简笔画风格"
3. 再说："画森林里的小兔子，油画风格"
4. 最后说："画森林里的小兔子，像日本动画一样"
5. 同一只兔子，三种完全不同的样子！
