# 第1课：AI魔法秀

## 课程信息
- 模块：模块一 · 认识AI朋友
- 时长：35分钟
- 主题：第一次遇见AI，感受它的三大魔法

## 学习目标
1. 知道AI是"住在电脑里的超级聪明朋友"
2. 体验AI的三大能力：聊天、画画、讲故事
3. 学会安全使用AI的第一条口诀

## 准备清单
- [ ] 电脑或平板（已联网）
- [ ] ChatGPT（chat.openai.com）或豆包（doubao.com）已登录
- [ ] 家长提前用AI生成一张"彩虹上的可爱独角兽"图片备用

## 时间安排
| 环节 | 时长 | 内容 |
|------|------|------|
| 魔法时刻 | 3分钟 | 展示AI生成的独角兽图片 |
| 故事时间 | 5分钟 | 薇薇安遇见AI的故事 |
| 知识小课堂 | 7分钟 | AI是什么，能做什么 |
| 动手体验 | 15分钟 | 聊天 + 画画 + 讲故事 |
| 安全口诀 | 5分钟 | 学习安全使用口诀 |

## 推荐工具
- 主要：[ChatGPT](https://chat.openai.com/)
- 备选：[豆包](https://www.doubao.com/)
