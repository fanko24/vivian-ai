# 第2课：和AI做朋友

## 课程信息
- 模块：模块一 · 认识AI朋友
- 时长：40分钟
- 主题：学会和AI好好说话，成为真正的朋友

## 学习目标
1. 学会问AI好问题的三个诀窍
2. 能和AI围绕一个话题连续聊5个来回
3. 感受"说清楚"和"说不清楚"带来的不同结果

## 准备清单
- [ ] 电脑或平板（已联网）
- [ ] ChatGPT（chat.openai.com）或豆包（doubao.com）已登录
- [ ] 上节课的作业（问AI的三个问题）

## 时间安排
| 环节 | 时长 | 内容 |
|------|------|------|
| 魔法时刻 | 3分钟 | AI回答月亮为什么有圆有弯 |
| 故事时间 | 5分钟 | 薇薇安学说话的故事 |
| 知识小课堂 | 7分钟 | 怎样问出好问题 |
| 动手体验 | 20分钟 | 三个提问练习 |
| 总结分享 | 5分钟 | 分享今天的发现 |

## 推荐工具
- 主要：[ChatGPT](https://chat.openai.com/)
- 备选：[豆包](https://www.doubao.com/)
