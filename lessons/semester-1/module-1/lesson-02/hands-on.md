# 动手实践：和AI做朋友

## 30秒魔法时刻

看AI怎么回答这个问题：

"你能用小朋友的方式告诉我，月亮为什么有时圆有时弯吗？"

（家长输入这个问题，让孩子看AI的回答）

是不是说得又清楚又好懂？这就是问了一个好问题的结果！

---

## 练习一：模糊 vs 清楚

1. 先输入一个模糊的问题："告诉我关于动物的事"
2. 看看AI怎么回答
3. 再输入一个清楚的问题："为什么长颈鹿的脖子那么长？"
4. 比较两次回答，哪个更有趣、更好懂？

## 练习二：一次问一个

1. 选一个你感兴趣的话题，比如"太空"、"海洋"或"恐龙"
2. 围绕这个话题，一次只问一个问题
3. 等AI回答后，根据它的回答再问下一个问题
4. 连续聊5个来回，看看你能了解到什么

比如聊"海洋"：
- 第1问："大海里最大的动物是什么？"
- 第2问："蓝鲸有多大呀？"
- 第3问："蓝鲸每天吃多少东西？"
- ……一个接一个地问下去！

## 练习三：让AI用小朋友的方式说话

1. 随便问AI一个问题
2. 如果回答太长太难懂，就说："请你用8岁小朋友能听懂的话再说一遍"
3. 再试试加一句："能不能用打比方的方式告诉我？"
4. 感受一下，加上这句话之后AI的回答有什么变化
