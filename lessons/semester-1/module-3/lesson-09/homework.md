# 课后任务：AI科学探索

## 必做任务：我的科学探索日记（10-15分钟）

用AI探索2个你好奇的科学问题，把学到的记录下来。

**任务步骤**：
1. 想两个你一直想问的"为什么"
2. 打开AI工具（ChatGPT或豆包）
3. 用"请用小朋友能听懂的话告诉我"来提问
4. 每个问题至少追问一次

**记录方式（选一种你最喜欢的）**：
- **画画**：画出你学到的科学知识
- **语音**：录一段话讲讲你学到了什么
- **写字**：用自己的话写3-5句小结
- **表格**：

| 我的问题 | AI的回答 | 我又追问了什么 | 我学到了什么 |
|---------|---------|--------------|------------|
| | | | |
| | | | |

## 选做任务：动手做实验（15-20分钟）

选一个AI推荐的安全小实验，在家长帮助下动手做！

**任务步骤**：
1. 从课上练习四记录的实验中选一个
2. 和家长一起准备材料（必须是家里安全的物品）
3. 在家长陪同下做实验
4. 把实验过程和结果记录下来

**记录方式**：
- 拍几张实验过程的照片
- 画一画实验前后的变化
- 说说实验结果和你想的一样吗

**安全提醒**：所有实验都必须有家长在场！不能用火、电、化学品。

---

完成作业后，把你的科学发现分享给家人吧！
