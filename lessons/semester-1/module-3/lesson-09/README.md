# 第9课：AI科学探索

## 课程信息
- **模块**：模块三 - AI学习伙伴
- **时长**：45分钟
- **核心体验**：用AI探索你好奇的科学问题

## 学习目标
1. 学会用AI探索自己好奇的科学问题
2. 能通过追问深入了解一个话题
3. 用自己的方式记录和分享学到的知识

## 准备工作
- [ ] 平板电脑或电脑（已联网）
- [ ] AI工具已登录（ChatGPT或豆包）
- [ ] 纸和彩笔（用于记录和画画）
- [ ] 家长陪伴

## 时间安排
| 环节 | 时间 | 内容 |
|------|------|------|
| 故事开场 | 5分钟 | 薇薇安的为什么 |
| 动手实践 | 25分钟 | 探索科学问题 |
| 知识讲解 | 10分钟 | 科学家是怎么思考的 |
| 课后任务 | 5分钟 | 布置作业 |
