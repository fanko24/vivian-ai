# 动手实践：AI科学探索

## 30秒魔法时刻

打开AI工具（ChatGPT或豆包），输入："为什么天空是蓝色的？请用8岁小朋友能听懂的话告诉我。"
看AI用多有趣的方式回答这个经典问题！

---

## 练习一：选一个你的"为什么"（5分钟）

**目标**：找到你最想探索的问题

1. 想想你自己一直好奇的事情
2. 把它写下来或说出来
3. 如果一下子想不出来，从下面挑一个：
   - 为什么星星白天看不见？
   - 为什么海水是咸的？
   - 为什么花有不同颜色？
   - 为什么猫咪会发出咕噜声？
4. 打开AI，输入你的问题（记得加上"请用小朋友能听懂的话"）

## 练习二：让AI当你的科学老师（8分钟）

**目标**：学会用AI深入了解一个话题

1. 读一读AI对你第一个问题的回答
2. 找到回答里你最感兴趣的一点
3. 根据这一点，问AI一个新问题
4. 再从第二个回答里找新问题，继续问
5. 看看你能不能连续追问3次

**示例**：
- 第1问："为什么海水是咸的？"
- 第2问："盐是从哪里来的？"
- 第3问："河流怎么把盐带到海里的？"

## 练习三：记录你的发现（7分钟）

**目标**：用自己的方式整理学到的知识

1. 选一个你今天觉得最有趣的答案
2. 用你喜欢的方式记录下来：
   - 画一幅画解释它
   - 用自己的话写3-5句小结
   - 录一段语音讲给别人听
3. 给你的记录取一个有趣的名字

## 练习四：小小实验计划（5分钟）

**目标**：让AI帮你设计一个安全的小实验

1. 问AI："关于我今天学到的知识，有没有简单的安全小实验？"
2. 告诉AI："这个实验需要爸爸妈妈能帮忙，材料要家里有的"
3. 把AI建议的实验步骤记下来
4. 和家长商量什么时候可以做这个实验

**注意**：任何实验都必须在家长陪同下进行！

---

记住安全口诀哦！不要告诉AI你的真名、学校和家庭住址。
