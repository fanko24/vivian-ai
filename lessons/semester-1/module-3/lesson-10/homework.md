# 课后任务：我的AI成就展

## 必做任务：举办你的AI成就展（10-15分钟）

这是最后的毕业任务！把你的AI之旅展示给全家人看。

**任务步骤**：
1. 从课上练习三中选一种展示方式，完善你的展示
2. 邀请全家人来参加你的"AI成就展"
3. 向家人介绍你的3件最佳作品
4. 告诉家人你这学期学到的最重要的3件事

**展示小建议**：
- 可以把作品摆在桌子上或贴在墙上
- 可以当"小老师"教爸爸妈妈一个AI技巧
- 最后别忘了鞠个躬说"谢谢大家"

## 选做任务：给未来的自己写封信（15-20分钟）

想想未来的你想用AI做什么？

**任务步骤**：
1. 在纸上或用AI帮忙，写一封给"未来的自己"的信
2. 写下你现在最想学的新本领
3. 写下你对AI的想法和期待
4. 请家长帮你保存这封信，一年后再打开

**可以写的内容**：
- "亲爱的未来的我，我现在8岁，学会了……"
- "我以后想用AI做……"
- "我觉得AI最酷的地方是……"
- "我希望AI将来能……"
