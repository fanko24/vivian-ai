# 第10课：我的AI成就展

## 课程信息
- **模块**：模块三 - AI学习伙伴（最后一课）
- **时长**：45分钟
- **核心体验**：回顾所有作品，庆祝学习成就，展望未来

## 学习目标
1. 回顾和整理这学期的所有AI作品
2. 选出自己最满意的3件作品并分享
3. 感受自己的成长，对未来充满期待

## 准备工作
- [ ] 准备好之前所有的AI作品（画、故事、音乐、视频等）
- [ ] 平板电脑或电脑（已联网）
- [ ] AI工具已登录（ChatGPT或豆包）
- [ ] 纸、彩笔或打印设备
- [ ] 家长陪伴

## 时间安排
| 环节 | 时间 | 内容 |
|------|------|------|
| 故事开场 | 5分钟 | 薇薇安的毕业典礼 |
| 动手实践 | 25分钟 | 整理作品、制作成就展 |
| 知识讲解 | 10分钟 | 回顾所学、颁发证书 |
| 课后任务 | 5分钟 | 布置最终作业 |
