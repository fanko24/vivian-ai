# 家长指导手册：第8课 - AI学习助手

## 课前准备

1. 准备好一台已联网的平板或电脑，确保AI工具（ChatGPT/豆包）已登录
2. 帮孩子找一道最近不会做的题目，课上会用到
3. 了解孩子最近学习中遇到的困难，方便课堂引导

## 课堂引导

1. **先思考再求助**：当孩子想直接问AI答案时，引导她先说出自己的想法。哪怕想法不完全对，思考的过程最重要
2. **对比好问题坏问题**：让孩子亲手尝试两种不同的问法，感受AI回答的区别。让她自己说出哪种更好
3. **鼓励表达**：AI讲解完后，让孩子用自己的话复述一遍。如果她说不清楚，说明还没完全理解
4. **不要急于纠正**：如果孩子算错了，让她先问AI，体验自己发现错误的过程
5. **关注习惯养成**：本课的核心不是学知识，而是养成"先思考再求助"的好习惯

## 常见问题

**孩子只想抄AI的答案怎么办？**
正常现象，可以温和引导："你觉得直接抄答案，下次遇到同样的题会做吗？让AI教你怎么想，你就永远都会了。"

**AI的回答太复杂孩子看不懂怎么办？**
帮孩子追问AI："能用更简单的话再说一遍吗？我8岁。"或者家长用更通俗的话转述。

**孩子什么题都想去问AI怎么办？**
表扬孩子善于提问，同时提醒她先自己想30秒。可以用计时器，养成"想一想让AI"的习惯。

## 延伸活动

1. **家庭"好问题"比赛**：每周选一天，全家分享自己问AI的好问题，评选"最佳提问奖"
2. **学习方法海报**：和孩子一起做一张"先思考再求助"的提示贴纸，贴在学习桌前
