# 动手实践：AI学习助手

## 30秒魔法时刻

打开AI工具（ChatGPT或豆包），输入："为什么1+1=2？"
看AI怎么用有趣的方式解释这个最简单的数学问题！

---

## 练习一：先思考再求助（5分钟）

**目标**：学会正确的学习步骤

1. 找一道你最近不会做的题目（数学、语文都可以）
2. 先自己想一想，把你的想法说出来或写下来
3. 打开AI，输入："这道题我有点不明白，能教教我怎么想吗？"
4. 读一读AI的讲解，用自己的话再说一遍
5. 试试做一道类似的题目，看你是不是真的学会了

## 练习二：好问题坏问题（5分钟）

**目标**：学会向AI提好问题

1. 看看下面的两组问法
2. 先试"坏问题"：输入"帮我把这道题做出来"，看看AI怎么回答
3. 再试"好问题"：输入"这道题我不会，能教教我思路吗？"，看看回答有什么不同
4. 和家长说说，两种回答哪个更能帮你学习

**坏问题示例**：
- "给我答案"
- "帮我写完"
- "这道题答案是几"

**好问题示例**：
- "能教教我怎么做吗？"
- "我哪里想错了？"
- "有没有更简单的方法？"

## 练习三：AI小检查员（5分钟）

**目标**：用AI检查自己的作业

1. 拿出你今天做的一道题（已经做好的）
2. 告诉AI："我算出来答案是XX，能帮我看看对不对吗？"
3. 如果AI说你做对了，给自己一个表扬
4. 如果AI说你做错了，问它："我哪里想错了？"
5. 自己改正错误，记住正确的方法

## 练习四：探索AI学习超能力（5分钟）

**目标**：发现AI更多的学习本领

1. 试着问AI："你能帮我背诵古诗吗？"看看它怎么帮你
2. 试着问AI："英语单词'beautiful'怎么记？"看看它的记忆小妙招
3. 试着问AI："为什么雨后会有彩虹？"看看它怎么解释
4. 选一个你最喜欢的功能，下次学习时试试

---

记住安全口诀哦！不要告诉AI你的真名、学校和家庭住址。
