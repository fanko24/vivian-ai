# 第8课：AI学习助手

## 课程信息
- **模块**：模块三 - AI学习伙伴
- **时长**：40分钟
- **核心体验**：学会用正确的方式让AI帮助自己学习

## 学习目标
1. 知道AI是学习伙伴，不是答案机器
2. 学会"先思考，再求助"的学习方法
3. 能用AI检查作业、理解难题

## 准备工作
- [ ] 平板电脑或电脑（已联网）
- [ ] AI工具已登录（ChatGPT或豆包）
- [ ] 一道最近不会做的题目
- [ ] 家长陪伴

## 时间安排
| 环节 | 时间 | 内容 |
|------|------|------|
| 故事开场 | 5分钟 | 薇薇安的数学题 |
| 动手实践 | 20分钟 | 学会和AI一起学习 |
| 知识讲解 | 10分钟 | 好习惯vs坏习惯 |
| 课后任务 | 5分钟 | 布置作业 |

让我们一起学会和AI做学习伙伴吧！
