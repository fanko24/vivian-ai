# 第5课：AI会说话

> 让AI朗读你的故事，用语音和AI对话

## 课程信息

- **时长**：40分钟
- **工具**：豆包App（手机）
- **难度**：⭐⭐

## 学习目标

1. 体验AI语音交互（语音输入和语音输出）
2. 理解AI能把文字变成声音，也能听懂声音
3. 用语音模式和AI进行有趣的对话

## 准备清单

- [ ] 手机安装好豆包App
- [ ] 网络连接流畅
- [ ] 安静的环境（语音对话需要）
- [ ] 家长陪伴

## 时间安排

| 环节 | 时间 |
|------|------|
| 故事引入 | 5分钟 |
| 知识讲解 | 8分钟 |
| 动手实践 | 22分钟 |
| 分享展示 | 5分钟 |
