# 课后作业：AI会说话

## 必做任务：语音聊天记录（10分钟）

用语音模式和AI聊3个问题，记录最有趣的对话。

**任务步骤**：
1. 打开ChatGPT App或豆包App的语音模式
2. 用语音问AI 3个问题（可以是任何你好奇的事情）
3. 截图或记下最有趣的一组问答

**问题参考**：
- "如果我变成一只小鸟，我会看到什么？"
- "你能不能用故事的方式告诉我星星是怎么来的？"
- "今天我学了什么新东西，你能帮我记住吗？"

**记录表格**：

| 我问的问题 | AI的回答（简单记一下） | 我觉得有趣吗 |
|-----------|---------------------|-------------|
| | | ⭐⭐⭐⭐⭐ |
| | | ⭐⭐⭐⭐⭐ |
| | | ⭐⭐⭐⭐⭐ |

## 选做任务：让AI朗读你创作的内容（15分钟）

写一段你喜欢的内容，让AI朗读给你听。

**任务步骤**：
1. 写4-8句话（可以是诗歌、小故事、或给家人的话）
2. 在ChatGPT或豆包中输入这段文字
3. 点击朗读功能，听AI读你的文字
4. 如果可以的话，保存朗读的音频

**可以写的内容**：
- 一首关于春天的诗
- 一个小动物的故事
- 一段想对爸爸妈妈说的话

**思考问题**：
- 听AI读你的文字，和你自己读有什么不同？
- 你觉得AI朗读的声音好听吗？为什么？
