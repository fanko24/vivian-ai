# 动手实践：AI视频魔术

## 30秒魔法时刻

> 家长先来！打开豆包（doubao.com），按以下步骤操作：

1. 点击左侧导航栏的【视频生成】（Seedance 2.0）
2. 在输入框输入："一只小猫在花园里追蝴蝶，阳光明媚，卡通风格"
3. 点击生成按钮 → 等待1-3分钟
4. 点击播放

看！这段视频没有人拍，是AI用一句话"想"出来的！你也可以！

---

## 第一步：写出视频描述（8分钟）

**目标**：学会用四要素写出好描述

1. 从下面选一个主题：海底世界、太空旅行、森林冒险
2. 填写四要素：
   - **谁**：一个什么样的角色？
   - **在做什么**：角色在干什么？
   - **在哪里**：什么场景、什么背景？
   - **什么感觉**：明亮？神秘？温暖？
3. 把四要素连成一段描述

**示例**：
> "一只小海龟在大海里慢慢游泳，周围是彩色珊瑚和热带鱼，画面宁静美好，水彩画风格。"

## 第二步：用豆包生成你的第一个视频（12分钟）

**目标**：用AI把文字变成视频

1. 打开豆包（doubao.com）
2. 点击左侧导航栏的【视频生成】（Seedance 2.0）
3. 在输入框粘贴你写的描述
4. 可以调整下方参数（保持默认也行）
5. 点击【生成】按钮
6. 耐心等待1-3分钟
7. 生成完成后点击播放，看看效果！

> 如果豆包视频生成不可用，可以用可灵AI（klingai.com）：打开网站 → 登录 → 点击【文生视频】→ 输入描述 → 点击生成

## 第三步：多试几个主题（10分钟）

**目标**：体验不同主题的视频效果

1. 选一个完全不同的主题，写好描述
2. 在豆包视频生成页面输入新描述 → 点击生成
3. 再换一个主题，再生成一个
4. 三段视频都生成后，放在一起比较

**更多主题参考**：
- "一个小朋友坐在彩虹上滑滑梯，天空是粉色的，童话风格"
- "一只企鹅在沙漠里跳舞，画面搞笑有趣，3D动画风格"

> AI视频只有3-5秒是正常的，就像一个小动画片段！

---

## 成果整理

保存你的AI视频作品：

1. 在豆包视频生成页面，点击视频右下角的下载按钮
2. 或者点击分享 → 保存到本地
3. 如果用的是可灵AI，点击视频下方的【下载】按钮
4. 建一个文件夹叫"第9课-AI视频"，把2-3段视频都放进去

**恭喜你！你有了2-3段AI生成的短视频作品！**

记住安全口诀哦！
