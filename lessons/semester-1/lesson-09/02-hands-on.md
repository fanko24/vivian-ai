# 动手实践：AI视频魔术

## 30秒魔法时刻

> 课前准备：请家长提前在豆包App或可灵AI（klingai.com）中输入"一只小猫在花园里追蝴蝶，阳光明媚，卡通风格"，生成一段短视频并保存好。

家长播放准备好的AI短视频。

"这段视频没有人拍，是AI用一句话'想'出来的！你也可以！"

---

## 练习一：写出视频描述（8分钟）

**目标**：学会用四要素写出好描述

1. 从下面选一个主题：海底世界、太空旅行、森林冒险
2. 填写四要素：谁 + 在做什么 + 在哪里 + 什么感觉
3. 把四要素连成一句话
4. 检查：别人读你的话，能想象出画面吗？

示例："一只小海龟在大海里慢慢游泳，周围是彩色珊瑚，宁静美好的感觉。"

## 练习二：生成你的第一个视频（12分钟）

**目标**：用AI工具把文字变成视频

1. 打开豆包App或可灵AI（klingai.com）
2. 找到"视频生成"功能
3. 把你写的描述粘贴进去
4. 点击"生成"，耐心等待（可能需要1-3分钟）

> 小提示：豆包App和可灵AI都有中文界面，操作方便。视频生成需要一点时间，不要着急。

## 练习三：多试几个主题（10分钟）

**目标**：体验不同主题的视频效果

1. 选一个完全不同的主题（比如动物变美食、太空变海洋）
2. 写好描述，生成视频
3. 再换一个主题，再生成一个
4. 三个视频放在一起看，哪个最有趣？

> AI视频只有3-5秒是正常的，就像一个小动画片段！

记住安全口诀哦！
