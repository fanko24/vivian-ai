# 课后作业：AI视频魔术

## 必做任务：生成2个主题视频（10分钟）

选2个你喜欢的主题，用AI各生成一个短视频。

**推荐主题**：
- 可爱的动物（小猫玩毛线、小狗追球、小兔子吃胡萝卜）
- 美丽的风景（日落海滩、雪山星空、春天的花园）
- 奇妙的想象（会飞的鲸鱼、糖果世界、机器人跳舞）

**步骤**：
1. 为每个主题写一句描述（谁 + 在做什么 + 在哪里 + 什么感觉）
2. 在豆包App或可灵AI中生成视频
3. 保存你最喜欢的2个视频

**记录你的描述**：
| 视频编号 | 我的描述 | 我觉得效果怎么样 |
|----------|---------|-----------------|
| 第1个 | | ⭐⭐⭐⭐⭐ |
| 第2个 | | ⭐⭐⭐⭐⭐ |

## 选做任务：四场景故事视频（15分钟）

用4个短视频讲一个小故事！

1. 想一个简单的4步故事（比如：小熊找蜂蜜）
2. 为每个场景写一句描述：
   - 场景1：小熊在森林里闻到了甜甜的味道
   - 场景2：小熊跟着味道走啊走
   - 场景3：小熊发现了一棵大树上有蜂巢
   - 场景4：小熊开心地吃到了蜂蜜
3. 用AI分别生成4段视频
4. 按顺序播放，就是一个完整的小故事

**思考问题**：
- 为什么有时候AI生成的视频和我想的不太一样？
- 描述要怎么写才能让AI更好地理解？
