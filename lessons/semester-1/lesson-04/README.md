# 第4课：AI绘画大师

> 进阶绘画技巧，让AI画出更精彩的画面

## 课程信息

- **时长**：45分钟
- **工具**：豆包（doubao.com）
- **难度**：⭐⭐

## 学习目标

1. 学会用"好描述公式"让AI画出更精准的画面
2. 尝试水彩、油画、卡通、中国画等不同风格
3. 创作一组主题系列作品

## 准备清单

- [ ] 打开豆包
- [ ] 想好一个喜欢的主题
- [ ] 建好保存图片的文件夹

## 时间安排

| 环节 | 时间 |
|------|------|

| 知识讲解 | 10分钟 |
