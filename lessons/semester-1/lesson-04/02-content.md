# AI绘画大师的秘密

## 1. 好描述公式

想让AI画出你心里的画面？记住这个公式：

**好描述 = 画什么 + 在干什么 + 在哪里 + 什么风格**

举个例子：
- 太简单："一只猫"
- 刚刚好："一只橘猫在花园里追蝴蝶，水彩画风格"

就像跟朋友说你想吃什么——说"想吃东西"和说"想吃草莓味的冰淇淋"，哪个更容易帮你买到？

## 2. 魔法风格词

换一个风格词，画面就完全不同：

| 风格词 | 效果 |
|--------|------|
| 水彩画 | 柔和透明，像晕染开的感觉 |
| 油画 | 色彩浓郁，有厚厚的质感 |
| 卡通 | 可爱夸张，像动画片 |
| 中国画 | 水墨淡雅，意境悠远 |

## 3. 创意组合术

把两个不相关的东西放在一起，就能创造惊喜：
- 动物 + 职业 -> "穿西装的企鹅厨师"
- 食物 + 建筑 -> "汉堡包形状的房子"
- 季节 + 动物 -> "冬天里围着围巾的小鹿"
