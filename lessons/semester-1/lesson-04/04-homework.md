# 课后作业：AI绘画大师

## 必做任务：主题系列创作（10分钟）

选一个你喜欢的主题，创作一组3张系列画。

**推荐主题**：
- 动物音乐家（比如：猫咪弹钢琴、小狗拉小提琴、兔子吹喇叭）
- 甜品王国（比如：蛋糕城堡、冰淇淋山、巧克力河）
- 四季精灵（比如：春天的花精灵、夏天的太阳精灵……）

**步骤**：
1. 确定你的主题和3张画的画面
2. 用"好描述公式"写出每张画的描述
3. 在豆包中生成图片
4. 把3张图保存到一个文件夹

**记录你的描述词**：
| 图片编号 | 我的描述词 | 为什么这样写 |
|----------|-----------|-------------|
| 第1张 | | |
| 第2张 | | |
| 第3张 | | |

## 选做任务：风格大挑战（15分钟）

同一个画面，试试不同的风格，看看有什么不同！

1. 选一个你喜欢的画面描述
2. 分别用"水彩画风格""油画风格""卡通风格""中国画风格"生成4张图
3. 把4张图放在一起看，你最喜欢哪个风格？

**思考问题**：
- 为什么不同风格的画面感觉完全不同？
- 如果你要画"雨天的小猫"，你会选什么风格？为什么？

## 提交方式

保存图片文件到文件夹即可，不需要打印。
