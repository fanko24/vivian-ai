- 想聊什么就聊什么

## 作业2：让AI讲个故事 ⭐⭐⭐

### 任务

让AI讲个你喜欢的故事！

### 选择故事

让Yanny想想：
- "我想听什么故事？"
- "关于什么的？小猫？小狗？恐龙？"
- "是童话故事还是冒险故事？"

### 让AI讲故事

说：
```
小爱同学，讲个关于（Yanny选的）的故事
```

### 记录故事

| 故事主题 | 故事讲了什么？ | Yanny觉得有趣吗？ |
|---------|---------------|------------------|
|         |               |                  |

### 喜欢的话

- 可以让AI再讲一遍
- 可以让AI多讲几个
- 可以让AI讲不同的故事

## 家长提示

### 1. 控制使用时间

- 每次语音聊天不要超过10分钟
- 保护嗓子，不要一直说话
- 注意保护听力

### 2. 鼓励使用

- 鼓励Yanny每天和AI说几句话
- 让AI帮忙做一些小事
- 享受和AI互动的乐趣

### 3. 记录有趣的对话

- 记录Yanny和AI的有趣对话
- 以后可以回忆

### 4. 真实交流

- 告诉Yanny，AI虽然有趣，但真实的朋友和家人才是最重要的
- 不要只和AI说话，也要多和家人朋友交流

### 5. 安全提醒

- 不要告诉AI家里的地址、电话等
- 如果AI说奇怪的话，告诉爸爸妈妈
