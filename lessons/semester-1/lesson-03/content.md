# 第三课内容：AI语音聊天——和AI说说话

## 🎯 趣味开场（5分钟）

### 你知道AI也会说话吗？

之前我们用文字和AI聊天，今天我们要用**声音**和AI聊天！

> "我不仅能看懂你的文字，还能听懂你的声音！你可以直接对我说话，我会用声音回答你哦！"

**这就是AI语音聊天！** 🗣️

### 什么是AI语音聊天？

AI语音聊天就是：
- 你**说** → AI**听** → AI**想** → AI**说**

```
👧 Yanny说话 → 🤖 AI听懂 → AI回答 → AI用说话回答
```

**和之前有什么不同？**
- 之前：打字 → AI回答（看文字）
- 今天：说话 → AI回答（听声音）

## 📱 认识AI语音工具（8分钟）

### 可以使用哪些工具？

| 工具 | 特点 | 推荐 |
|------|------|------|
| 智能音箱（小爱/天猫精灵/小度） | 最方便，直接说话 | ⭐⭐⭐⭐⭐ |
| ChatGPT语音版 | 需要手机App | ⭐⭐⭐ |
| 豆包语音 | 电脑或手机 | ⭐⭐⭐⭐ |

### 强烈推荐：智能音箱

**推荐理由**：
- 家里可能已经有了
- 不用操作，直接说话
- 孩子用起来最方便

**如果家里没有智能音箱**：
- 可以用手机下载"豆包"App，里面有语音功能
- 或下载ChatGPT App，使用语音功能

### 怎么使用智能音箱

```
┌─────────────────────────────────┐
│           小爱音箱              │
├─────────────────────────────────┤
│                                  │
│   唤醒词 + 问题                 │
│                                  │
│   "小爱同学，今天天气怎么样？"   │
│                                  │
│   ⬆️ AI回答你的问题              │
│                                  │
└─────────────────────────────────┘
```

**常用唤醒词**：
- 小爱音箱："小爱同学"
- 天猫精灵："天猫精灵"
- 小度音箱："小度小度"

## 📝 本课重点词汇

- **语音 (Voice)**: 说话的声音
- **唤醒词 (Wake Word)**: 让AI注意你的话，比如"小爱同学"
- **智能音箱 (Smart Speaker)**: 会说话的音箱，能听懂你说的话
- **语音识别 (Speech Recognition)**: AI听懂你说的话
- **语音合成 (Speech Synthesis)**: AI用声音说话
