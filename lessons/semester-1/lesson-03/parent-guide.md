# 第三课：家长指导手册

> AI语音聊天——和AI说说话

## 授课目标

通过本节课，让孩子能够：
1. 体验AI语音对话的神奇
2. 学会用语音和AI交流
3. 感受不用打字就能聊天的方便

## 时间分配建议

| 环节 | 时间 | 注意事项 |
|------|------|---------|
| 趣味开场 | 5分钟 | 用"不用打字就能说话"吸引兴趣 |
| 认识AI语音工具 | 8分钟 | 展示智能音箱，讲解使用方法 |
| 动手实践：语音聊天 | 25分钟 | 让孩子主导说话 |
| 总结回顾 | 7分钟 | 讨论感受，预告下节课 |

## 授课要点

### 1. 趣味开场（5分钟）

**如何引入**：
- 用"不用打字就能说话"吸引兴趣
- 问问孩子有没有见过会说话的音箱

**互动话术**：
- "你知道吗？不用打字，直接说话就能和AI聊天！"
- "家里的智能音箱，你可以直接对它说话！"
- "想不想试试？"

### 2. 认识AI语音工具（8分钟）

**唤醒智能音箱**：
- 让孩子看看家里的智能音箱
- 教孩子如何唤醒

**讲解唤醒词**：
- 小爱音箱："小爱同学"
- 天猫精灵："天猫精灵"
- 小度音箱："小度小度"

**使用方法**：
```
唤醒词 + 想说的话
例如："小爱同学，今天天气怎么样？"
```

**简单演示**：
- 家长先示范一次唤醒和对话
- 让孩子看清楚怎么说

**重点讲解**：
- 说话要清楚
- 说完等待AI回应
- AI用声音回答

---

### 3. 动手实践：语音聊天（25分钟）

**活动一：第一次和AI说话（5分钟）**

让孩子自己说：
```
小爱同学，你好！
```

**家长引导**：
- 清楚地说出唤醒词
- 等待AI回应
- "AI会回答什么？"

**注意事项**：
- 说话要清楚
- 不要着急
- 等待AI处理

---

**活动二：问AI简单问题（6分钟）**

让孩子用语音问问题：
```
小爱同学，今天天气怎么样？
```
```
小爱同学，现在几点了？
```

**家长引导**：
- "AI能听懂你说的话吗？"
- "AI回答得对吗？"
- "还有什么问题想问？"

---

**活动三：让AI讲故事（6分钟）**

让孩子说：
```
小爱同学，给我讲个故事！
```

**安静听故事**：
- 让孩子安静地听
- 不要打断AI

**故事后讨论**：
- 故事有趣吗？
- AI的声音好听吗？

---

**活动四：和AI玩个游戏（5分钟）**

让孩子说：
```
小爱同学，我们来玩猜谜游戏吧！
```

**观察**：
- AI会出什么题？
- 孩子能猜出来吗？
- 玩得开心吗？

---

**活动五：AI帮Yanny做什么（3分钟）**

让孩子让AI帮忙：
```
小爱同学，帮我定个5分钟后的闹钟
```

**讨论**：
- AI真的帮到忙了吗？
- 觉得AI方便吗？

---

### 4. 总结回顾（7分钟）

**问Yanny**：
1. 和AI说话感觉怎么样？
2. 用语音聊天方便吗？
3. AI的声音怎么样？
4. 喜欢和AI语音聊天吗？

**预告下节**：
- "下次课我们要学习文生视频——让故事动起来！"

## 互动建议

### 增加趣味性

1. **比一比**: 看谁能先唤醒AI
2. **游戏时间**: 和AI多玩几个游戏
3. **故事时间**: 让AI讲睡前故事
4. **助手时间**: 让AI帮忙做小事

### 引导思考

- "AI能听懂你说的话吗？"
- "AI的声音和真人的声音有什么不同？"
- "你觉得AI方便吗？"

### 避免误区

- ❌ 不要对孩子要求太高
- ❌ 不要批评孩子说话不清楚
- ❌ 不要让孩子一直说话
- ✅ 鼓励孩子尝试
- ✅ 表扬孩子的每一次成功
- ✅ 保护孩子的嗓子

---

## 注意事项

### 1. 控制使用时间

- 每次语音聊天不要超过10分钟
- 不要让孩子长时间说话
- 保护嗓子

### 2. 安静环境

- 在安静的地方进行语音对话
- 避免噪音干扰
- AI才能听得更清楚

### 3. 真实交流

- 告诉孩子，AI虽然有趣，但真实的朋友和家人才是最重要的
- 不要只和AI说话，也要多和家人朋友交流

### 4. 安全提醒

- 不要告诉AI家里的地址、电话等
- 如果AI说奇怪的话，告诉爸爸妈妈

---

## 常见问题处理

**Q: 孩子说唤醒词，AI没有反应怎么办？**
A: 让孩子清楚地说，靠近音箱一点，再试一次

**Q: 孩子说话不清楚，AI听不懂怎么办？**
A: 不要着急，鼓励孩子慢慢说，家长可以帮忙

**Q: AI回答了错误的内容怎么办？**
A: 告诉孩子AI也会犯错，不要太在意

**Q: 孩子想一直和AI说话怎么办？**
A: 要控制时间保护嗓子，约定每次聊5-10分钟就好
