# 第3课：和AI做朋友

## 课程信息
- 模块：模块一 · 认识AI朋友
- 时长：40分钟
- 主题：学会跟AI更好地聊天，成为AI的好朋友

## 学习目标
1. 知道"好问题"和"模糊问题"的区别
2. 学会一次只问一个问题，说清楚自己想问什么
3. 能和AI进行一段5轮以上的有趣对话

## 准备清单
- [ ] 电脑或平板（已联网）
- [ ] 豆包（doubao.com）已登录
- [ ] 家长陪伴

## 时间安排
| 环节 | 时长 | 内容 |
|------|------|------|
| 魔法时刻 | 3分钟 | 看AI如何妙答有趣问题 |
| 故事时间 | 5分钟 | 薇薇安学会问好问题的故事 |
| 知识小课堂 | 7分钟 | 好问题的三个秘诀 |
| 动手体验 | 20分钟 | 对话练习 |
| 总结分享 | 5分钟 | 分享最有趣的对话 |
