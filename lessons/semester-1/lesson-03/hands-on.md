# 第三课：动手实践

> 今天Yanny要用声音和AI聊天！

---

## 活动一：第一次和AI说话（5分钟）

### 任务

让Yanny试着唤醒智能音箱，和AI说第一句话！

### 唤醒AI

**如果家里有智能音箱**：

1. 找到家里的智能音箱（小爱/天猫精灵/小度）
2. 清楚地说出唤醒词：
   - "小爱同学"
   - "天猫精灵"
   - "小度小度"

3. 等待AI回应（AI会说"我在"或"在的"）

### 第一次对话

让Yanny说：

```
你好！
```

**观察**：
- AI回答了什么？
- AI的声音怎么样？
- Yanny觉得有趣吗？

### 记录下第一次语音对话

```
Yanny说：你好！
AI说：（这里记录AI的回答）
```

## 🌟 实践小贴士

1. **清楚说话**: 唤醒词要清楚地说，不然AI听不到
2. **耐心等待**: AI处理语音需要时间，不要着急
3. **安静环境**: 在安静的地方说话，AI听得更清楚
4. **多鼓励**: 每次成功对话，都要表扬Yanny
5. **控制时间**: 25分钟足够了，保护嗓子
