# 动手实践：AI声音魔术师

## 30秒魔法时刻

> 家长先来！打开剪映App，按以下步骤操作：

1. 点击【开始创作】→ 选择任意一张照片 → 点右下角【添加】
2. 底部工具栏 → 点【文本】→【添加文本】→ 输入"你好，我是AI！"
3. 点击文字轨道 → 底部工具栏 → 点【文本朗读】
4. 随便选一个声音 → 点击播放

听！文字变成了声音，像真人在说话一样！

---

## 第一步：克隆你的声音（10分钟）

**目标**：让AI"学会"你的声音

1. 打开剪映App → 点击【开始创作】
2. 选择任意一张照片 → 点右下角【添加】进入编辑界面
3. 底部工具栏向左滑动 → 找到【文本】→ 点击
4. 点击【添加文本】→ 输入："大家好，我是薇薇安！"
5. 点击文字轨道（文字会变成选中状态）
6. 底部工具栏 → 找到【文本朗读】→ 点击
7. 在弹出的声音面板 → 点击顶部【我的】标签
8. 点击【克隆音色】→ 点击【录制音频】
9. 按住录音按钮，自然地朗读（5-10秒）：

> "大家好，我叫薇薇安。今天天气真好，我在学用AI。"

10. 松开按钮 → 点击【完成】→ 给声音起个名字 → 保存
11. 等待AI学习（约10-30秒）
12. 选择你刚保存的音色 → 点击播放试听

**小提示**：在安静的房间录制，手机离嘴巴约15厘米

## 第二步：用你的声音读故事（7分钟）

**目标**：用克隆声音朗读第6课的故事

1. 还是在刚才的编辑界面
2. 点击文字轨道 → 修改文字内容
3. 把第6课你编的故事粘贴进来
4. 如果故事很长，可以分成多段文字
5. 点击【文本朗读】→【我的】→ 选择你克隆的音色
6. 点击播放，听听AI用你的声音读故事

## 第三步：试试不同的声音（5分钟）

**目标**：体验剪映的多种AI声音

1. 点击【文本朗读】→ 浏览不同的声音分类
2. 试试"小朋友的声音"、"讲故事的声音"
3. 同一段文字换不同声音读，比一比哪个最有趣

---

## 成果整理

保存你的声音克隆作品：

1. 确认视频里已经用克隆声音朗读了文字
2. 点击右上角【导出】→ 选择分辨率 → 点击【导出】
3. 等待导出完成 → 视频自动保存到手机相册
4. 把视频发给爸爸妈妈，让他们闭上眼睛猜猜是不是你读的

**恭喜你！你有了一段AI用你声音朗读的视频！**

记住安全口诀哦！不能用AI声音去骗人或恶作剧。
