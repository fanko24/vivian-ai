# 第7课：AI视频魔术

> 用文字生成会动的画面

## 课程信息

- **时长**：50分钟
- **工具**：豆包视频 / 可灵AI（klingai.com）
- **难度**：⭐⭐⭐

## 学习目标

1. 学会用文字描述让AI生成短视频
2. 理解"AI视频 = 文字变成会动的画面"
3. 尝试不同主题的视频创作

## 准备清单

- [ ] 打开豆包App或可灵AI
- [ ] 准备好你想拍的视频想法
- [ ] 确保网络流畅（视频生成需要网络）

| 环节 | 时间 |
|------|------|
| 故事引入 | 5分钟 |
| 知识讲解 | 8分钟 |
