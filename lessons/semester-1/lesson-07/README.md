# 第7课：AI声音魔术师

> 让AI模仿你的声音，体验声音克隆的神奇

## 课程信息

- **时长**：45分钟
- **工具**：豆包App（手机端）
- **难度**：⭐⭐⭐

## 学习目标

1. 体验AI声音克隆，了解AI如何"学会"一个人的声音
2. 用克隆的声音朗读文字，感受声音技术的神奇
3. 知道声音克隆的边界，不能用AI声音去骗人

## 准备清单

- [ ] 手机安装好豆包App
- [ ] 网络连接流畅
- [ ] 安静的录音环境（关窗、关电视）
- [ ] 家长陪伴

## 时间安排

| 环节 | 时间 | 内容 |
|------|------|------|
| 30秒魔法时刻 | 1分钟 | AI现场模仿声音 |
| 故事引入 | 5分钟 | 薇薇安的变声魔术 |
| 知识讲解 | 8分钟 | AI声音克隆的秘密 |
| 动手实践 | 26分钟 | 录音、克隆、朗读 |
| 作品分享 | 5分钟 | 展示最满意的AI声音 |
