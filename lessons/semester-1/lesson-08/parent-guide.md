# 家长指导：AI小老师

## 本课要点

这是最后一课，综合了学习助手、科学探索和毕业回顾三部分内容。

## 核心概念（可简单解释给孩子）

1. **AI是学习伙伴**：AI教方法，不是给答案
2. **先思考再求助**：自己先想，再问AI
3. **科学探索**：提问→找答案→再追问
4. **成长回顾**：整理作品，感受进步

## 课前准备

1. 确认ChatGPT或豆包可用
2. 准备一道孩子最近不会做的题目
3. 找出孩子之前所有的AI作品（画、音乐、视频等）

## 课堂指导建议

### 练习一（用AI辅助学习）
- 找孩子最近学习中遇到的真实问题
- 引导孩子先自己想，再问AI
- 强调"学方法"不是"抄答案"

### 练习二（科学探索）
- 鼓励孩子问自己真正好奇的问题
- 引导孩子追问，不要停留在第一个答案
- 可以和孩子比赛谁能追问更多次

### 练习三（选出TOP3）
- 认真听孩子解释为什么选这3件作品
- 这是孩子回顾成长的重要时刻
- 多鼓励，多肯定

## 毕业仪式

最后5分钟是毕业仪式时间：
1. 让孩子展示TOP3作品
2. 填写毕业证书
3. 全家人鼓掌庆祝
4. 可以拍一张"毕业照"留念

## 延伸建议

课程结束后，孩子可以：
1. 继续用AI辅助学习
2. 创作更多AI作品
3. 尝试新的AI工具
4. 保持好奇心，继续探索
