# 动手实践：AI绘画大师

## 30秒魔法时刻

在ChatGPT中输入：

> "一座用糖果建造的城堡，周围飘着彩虹色的泡泡，梦幻风格"

看看AI用30秒画出什么！惊讶吗？

---

## 练习一：好描述公式（10分钟）

**目标**：练习用公式写出好描述

1. 从下面选一个主题：海洋世界、太空冒险、森林派对
2. 用公式填写：画什么 + 在干什么 + 在哪里 + 什么风格
3. 输入ChatGPT，等待画面生成
4. 看看画面和你想象的一样吗？哪里不同？

## 练习二：风格变变变（10分钟）

**目标**：同一个画面，尝试不同风格

1. 选一个你喜欢的画面（可以用练习一的）
2. 把风格词改成"水彩画风格"，生成一张
3. 再改成"卡通风格"，生成一张
4. 把几张图放在一起，你最喜欢哪个风格？

## 练习三：创意组合术（8分钟）

**目标**：用意想不到的组合创造惊喜

1. 从第一列选一个词：企鹅、西瓜、恐龙、彩虹
2. 从第二列选一个词：医生、城堡、宇航员、花园
3. 把两个词组合成一句话描述，加上风格词
4. 输入ChatGPT，看看AI怎么理解你的创意！

> 小提示：如果用豆包，操作方法一样，把描述输入对话框就行啦。

记住安全口诀哦！
