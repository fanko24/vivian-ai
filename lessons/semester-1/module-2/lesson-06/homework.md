# 课后作业：AI视频魔术

## 必做任务：生成3个主题视频（10-15分钟）

选3个你喜欢的主题，用AI各生成一个短视频。

**推荐主题**：
- 可爱的动物（小猫玩毛线、小狗追球、小兔子吃胡萝卜）
- 美丽的风景（日落海滩、雪山星空、春天的花园）
- 奇妙的想象（会飞的鲸鱼、糖果世界、机器人跳舞）

**步骤**：
1. 为每个主题写一句描述（谁 + 在做什么 + 在哪里 + 什么感觉）
2. 在可灵AI或即梦中生成视频
3. 保存你最喜欢的3个视频

## 选做任务：四场景故事视频（15-20分钟）

用4个短视频讲一个小故事！

1. 想一个简单的4步故事（比如：小熊找蜂蜜）
2. 为每个场景写一句描述：
   - 场景1：小熊在森林里闻到了甜甜的味道
   - 场景2：小熊跟着味道走啊走
   - 场景3：小熊发现了一棵大树上有蜂巢
   - 场景4：小熊开心地吃到了蜂蜜
3. 用AI分别生成4段视频
4. 按顺序播放，就是一个完整的小故事

可以提交：视频文件、描述文字、或播放给家长看
