# 第7课：AI故事大王

> 模块二：AI创作工坊 | 第7课

## 课程信息

- **时长**：50分钟
- **工具**：ChatGPT（chat.openai.com）/ 豆包（doubao.com）
- **难度**：⭐⭐⭐

## 学习目标

1. 学会用"角色+困难+解决+结局"构思一个完整故事
2. 用AI帮助写故事并生成配图
3. 制作一本属于自己的迷你图画书

## 准备清单

- [ ] 打开 ChatGPT 或豆包
- [ ] 准备4张A4纸（做迷你书）
- [ ] 准备彩笔和胶水

## 时间安排

| 环节 | 时间 |
|------|------|
| 故事引入 | 5分钟 |
| 知识讲解 | 10分钟 |
| 动手实践 | 30分钟 |
| 分享展示 | 5分钟 |
