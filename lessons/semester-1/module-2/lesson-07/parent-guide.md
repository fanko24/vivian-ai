# 家长指导：第7课 AI故事大王

## 课前准备

1. 提前准备好4张A4纸、彩笔和胶水
2. 可以提前做一本示范小书（用AI生成4页故事+插图），课上展示
3. 确认ChatGPT或豆包的文字生成和图片生成功能正常

## 课堂指导建议

1. **展示示范**：先让孩子看做好的小书样本，激发创作热情
2. **引导构思**：用"角色-困难-解决-结局"四个步骤引导，不要直接给答案
3. **控制长度**：提醒孩子"4句话就够了"，避免故事太长来不及完成
4. **统一风格**：提醒孩子每张插图用相同的风格词，保证画风一致
5. **帮助组装**：折纸和粘贴部分孩子可能需要帮助，但封面设计让她自己做

## 常见问题

**Q：孩子想不出故事怎么办？**
A：可以给她几个开头选："一只怕黑的小猫……""小熊找不到妈妈……"让她选一个接着想。

**Q：AI写的故事太长了怎么办？**
A：告诉孩子可以对AI说"请缩短到4句话"或"每句话只要一行"。

**Q：插图和文字不匹配怎么办？**
A：帮孩子把文字中最关键的视觉元素提取出来，重新写画面描述。比如把"小猫在树下哭"改成"一只灰色小猫坐在大橡树下，眼泪滴答滴答，水彩画风格"。

## 延伸活动

2. 把孩子的迷你书扫描保存，以后可以回顾成长的足迹
