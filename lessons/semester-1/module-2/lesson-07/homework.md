# 课后作业：AI故事大王

## 必做任务：制作你的迷你图画书（10-15分钟）

创作一本完整的4页迷你图画书，有文字有插图！

**步骤**：
1. 用"角色+困难+解决+结局"构思故事
2. 在ChatGPT或豆包中让AI帮你写出4句话
3. 为每句话生成一张AI插图
4. 打印或手抄文字和图片，做成一本小书

**故事主题参考**：
- 小猫第一次独自出门
- 小熊学骑自行车
- 小花籽寻找生长的地方
- 机器人交到了第一个朋友

## 选做任务：朗读你的故事（15-20分钟）

把你的故事读给家人听，还可以录下来！

1. 翻开你的迷你图画书
2. 一页一页地朗读，加上感情
3. 可以给不同角色用不同的声音
4. 让家长帮忙录音或录像

**加分项**：把录音和图画书一起保存，这就是你的第一个"有声绘本"！

## 提交方式

可以提交：做好的小书、图片文件、或朗读录音/录像
