# 第5课：AI音乐工厂

> 模块二：AI创作工坊 | 第5课

## 课程信息

- **时长**：45分钟
- **工具**：Suno（suno.com）/ 天工AI音乐（tiangong.cn）
- **难度**：⭐⭐⭐

## 学习目标

1. 学会写简单的歌词（4-8句）
2. 用AI音乐工具生成一首真正的歌曲
3. 理解歌曲由歌词、旋律、风格三部分组成

## 准备清单

- [ ] 打开 Suno 或 天工AI音乐
- [ ] 想好一个想写的主题（比如春天、小动物、好朋友）
- [ ] 准备好耳机或音箱

## 时间安排

| 环节 | 时间 |
|------|------|
| 故事引入 | 5分钟 |
| 知识讲解 | 8分钟 |
| 动手实践 | 27分钟 |
| 分享聆听 | 5分钟 |
