# 家长指导：第5课 AI音乐工厂

## 课前准备

1. 提前注册好Suno（suno.com）或天工AI音乐（tiangong.cn）账号
2. 准备好耳机或音箱，让孩子听清音乐效果
3. 可以提前生成一首示范歌曲，课上播放给孩子听

## 课堂指导建议

1. **引导写词**：孩子写歌词时，帮她想押韵的字，但不要替她写
2. **操作协助**：Suno是英文界面，孩子可能需要帮助操作；天工AI音乐是中文界面，更适合独立操作
3. **多试几次**：第一次生成的歌可能不理想，鼓励孩子换风格或改歌词再试
4. **认真聆听**：歌曲生成后，和孩子一起认真听完，说说喜欢哪里
5. **节奏引导**：写歌词时可以帮孩子拍手打节奏，感受韵律

## 常见问题

**Q：Suno需要付费吗？**
A：Suno有免费额度，每天可以生成几首。天工AI音乐也有免费体验额度。如果额度用完，可以换一个工具。

**Q：孩子写的歌词很幼稚怎么办？**
A：这正符合8岁孩子的水平！重要的是表达和创作的过程，不需要追求文学性。

**Q：生成的歌曲不好听怎么办？**
A：可以试试换个风格词，比如把"欢快"改成"温柔"，或者把"流行"改成"民谣"。不同风格效果差别很大。
