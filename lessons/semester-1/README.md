> 8节课，开启AI世界的奇妙旅程

---

## 学期概览

本学期通过8节课，从认识AI到创作再到学习应用，循序渐进地帮助孩子掌握AI的基本使用。

- **总课时**：8节
- **建议节奏**：每周1节
- **每节时长**：35-50分钟
- **适合年龄**：8岁左右

## 学习建议

### 给家长
1. **陪伴学习** - 建议全程陪伴，特别是前几节课
2. **鼓励探索** - 鼓励孩子大胆尝试，不怕犯错
3. **记录成长** - 用照片或视频记录孩子的学习过程
4. **控制时间** - 每次学习不超过45分钟，保护视力

### 给小朋友
1. **大胆提问** - 有什么问题都可以问AI
2. **发挥想象** - 用AI实现你的奇思妙想
3. **耐心等待** - AI有时候需要思考一下
4. **记住安全** - 不要告诉AI你的个人信息
