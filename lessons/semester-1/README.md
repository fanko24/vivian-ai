# 第一学期：AI魔法探险

> 12节课，开启AI世界的奇妙旅程

---

## 学期概览

本学期分为三个模块，从认识AI到创作再到学习应用，循序渐进地帮助孩子掌握AI的基本使用。

- **总课时**：12节
- **建议节奏**：每周1节
- **每节时长**：40-60分钟
- **适合年龄**：8岁左右

---

## 模块一：认识AI朋友（3周）

让孩子初次接触AI，建立对AI的基本认知，学会与AI进行简单互动。

| 课时 | 课程名称 | 核心体验 |
|------|----------|----------|
| [第1课](module-1/lesson-01/) | 和AI聊聊天 | 第一次与AI对话，感受AI的智能 |
| [第2课](module-1/lesson-02/) | AI画梦工厂 | 用文字生成图片，体验AI绘画 |
| [第3课](module-1/lesson-03/) | AI小歌手 | 与AI进行语音互动，探索AI的声音能力 |

---

## 模块二：AI创作工坊（5周）

引导孩子用AI进行各种创意创作，培养创造力和表达能力。

| 课时 | 课程名称 | 核心体验 |
|------|----------|----------|
| [第4课](module-2/lesson-04/) | 小小创作家 | 用AI创作故事，学习写作技巧 |
| [第5课](module-2/lesson-05/) | 创意画室 | 创意绘画创作，发挥无限想象力 |
| [第6课](module-2/lesson-06/) | 音乐魔法师 | 创作音乐，感受音乐的魅力 |
| [第7课](module-2/lesson-07/) | 配音小演员 | 角色配音表演，体验声音的艺术 |
| [第8课](module-2/lesson-08/) | 视频魔法师 | 制作简单视频，体验视频创作的乐趣 |

---

## 模块三：AI学习伙伴（4周）

将AI作为学习助手，培养良好的学习习惯和科学思维。

| 课时 | 课程名称 | 核心体验 |
|------|----------|----------|
| [第9课](module-3/lesson-09/) | 作业小帮手 | 学会合理使用AI辅助学习 |
| [第10课](module-3/lesson-10/) | 科学探险家 | 探索科学世界，培养科学思维 |
| [第11课](module-3/lesson-11/) | 我的AI作品集 | 整理作品，回顾成长历程 |
| [第12课](module-3/lesson-12/) | 成果展示会 | 展示学习成果，庆祝学习完成 |

## 学习准备

### 工具准备
- [ ] 电脑或平板电脑
- [ ] 网络连接
- [ ] AI工具账号（推荐：[豆包](https://www.doubao.com/) 或 [ChatGPT](https://chat.openai.com/)）

### 材料准备
- [ ] 作文本或纸张
- [ ] 彩笔
- [ ] 文件夹（保存作品）

### 心理准备
- [ ] 保持好奇心
- [ ] 大胆提问
- [ ] 耐心学习

---

## 学习建议

### 给家长
1. **陪伴学习** - 建议全程陪伴，特别是前几节课
2. **鼓励探索** - 鼓励孩子大胆尝试，不怕犯错
3. **记录成长** - 用照片或视频记录孩子的学习过程
4. **控制时间** - 每次学习不超过45分钟，保护视力

### 给小朋友
1. **大胆提问** - 有什么问题都可以问AI
2. **发挥想象** - 用AI实现你的奇思妙想
3. **耐心等待** - AI有时候需要思考一下
4. **记住安全** - 不要告诉AI你的个人信息
