# 动手实践：AI故事大王

## 30秒魔法时刻

打开豆包（doubao.com），在对话框输入：

> "请用3句话编一个故事：一只会飞的猫咪，它每天飞到天上去摘星星。"

看！AI马上就编出了一个小故事！

---

## 练习一：和AI一起编新故事（10分钟）

**目标**：给AI一颗"故事种子"，让它帮你编故事

1. 打开豆包（doubao.com）
2. 想好你的故事种子（人物+地点+事件）
3. 在对话框输入你的种子

**提示词示例**：
> "请帮我编一个故事。故事里有：一只小兔子（人物），在大森林里（地点），发现了一扇神秘的门（事件）。故事要适合小朋友，大概200字。"

4. 读一读AI写的故事，觉得怎么样？
5. 如果不满意，可以换一颗种子再试一次

**更多种子参考**：
- 一个小女孩 + 海底世界 + 找到了一颗会发光的珍珠
- 一个小机器人 + 月亮上 + 遇到了外星朋友
- 一朵小花 + 城市里 + 想找到回家的路

## 练习二：让AI续写故事（8分钟）

**目标**：自己写个开头，让AI接着往下编

1. 自己写一个故事开头（2-3句话）
2. 在豆包中输入开头，让AI续写

**提示词示例**：
> "我写了一个故事开头，请帮我续写后面的内容，要有趣、适合小朋友：'从前有一只小熊，它每天都会在树下等它的好朋友。可是今天，好朋友没有来……'"

3. 读一读AI续写的内容
4. 如果喜欢，可以继续让AI往下写

## 练习三：让AI改写故事结局（8分钟）

**目标**：拿一个熟悉的故事，让AI改写结局

1. 想一个你知道的故事（比如龟兔赛跑、三只小猪）
2. 让AI改写结局

**提示词示例**：
> "请把《龟兔赛跑》的故事改写一下，这次兔子没有睡觉，但是路上遇到了一件有趣的事，结局和原来不一样。请用小朋友能听懂的语言写。"

3. 读一读新结局，和原来的有什么不同？
4. 你也可以试试：
   - 让三只小猪和大灰狼做朋友
   - 让丑小鸭一直就是天鹅，只是不知道

## 作品分享（5分钟）

1. 选出你最满意的故事
2. 大声读给爸爸妈妈听
3. 也可以截图保存你的作品

记住安全口诀哦！
