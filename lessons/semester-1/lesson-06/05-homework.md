# 课后作业：AI音乐工厂

## 必做任务：创作你的第一首歌（10分钟）

写一首简短的歌词，用AI工具把它变成真正的歌曲！

**步骤**：
1. 选一个主题（小动物、季节、家人、友谊都可以）
2. 写4-8句歌词，每句5-7个字
3. 打开豆包App或天工AI音乐
4. 粘贴歌词，选择风格，生成歌曲
5. 保存你最满意的版本

**歌词参考**：
> 主题：我的小猫
>
> 小猫小猫喵喵叫
> 软软毛毛真可爱
> 花园里面追蝴蝶
> 抓到一只笑哈哈

**记录你的创作**：
| 我的歌词 | 我选的风格 | 我觉得怎么样 |
|----------|-----------|-------------|
| | | ⭐⭐⭐⭐⭐ |

## 选做任务：为喜欢的人写一首歌（15分钟）

选一个你爱的人（妈妈、爸爸、好朋友），为她/他写一首歌！

1. 想想这个人最让你开心的是什么
2. 写几句想对她/他说的话
3. 选一个温柔的风格
4. 生成歌曲后，放给她/他听

**提示**：这首歌可以当作一份特别的礼物！

**思考问题**：
- 为什么不同风格的歌曲给人感觉不同？
- 你最喜欢哪种风格？为什么？
