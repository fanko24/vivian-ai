# 第6课：AI故事大王

> 和AI一起编故事、续写故事、改写结局

## 课程信息

- **时长**：45分钟
- **工具**：豆包（doubao.com）
- **难度**：⭐⭐⭐

## 学习目标

1. 学会用AI一起编故事（开头、发展、结尾）
2. 能让AI续写故事或改写故事结局
3. 体验AI的文字创作能力

## 准备清单

- [ ] 电脑或平板，打开豆包（doubao.com）
- [ ] 网络连接正常
- [ ] 豆包账号已登录
- [ ] 安静的学习环境
- [ ] 家长陪伴

## 时间安排

| 环节 | 时间 | 内容 |
|------|------|------|
| 30秒魔法时刻 | 1分钟 | AI现场编故事 |
| 故事引入 | 5分钟 | 薇薇安和AI编故事 |
| 知识讲解 | 8分钟 | 故事的三个部分 |
| 动手实践 | 26分钟 | 编故事、续写、改结局 |
| 作品分享 | 5分钟 | 展示最满意的故事 |
