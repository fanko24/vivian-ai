# 第2课：AI画梦工厂

> 用文字生成图片，感受AI绘画的神奇

## 课程信息
- **时长**：45分钟
- **工具**：豆包（doubao.com）
- **难度**：⭐⭐

## 学习目标
1. 学会用文字描述想要的图片
2. 能生成自己喜欢的图画
3. 理解描述越具体，画面越好看

## 准备清单
- [ ] 平板电脑或电脑
- [ ] 豆包账号已登录（doubao.com）
- [ ] 家长陪伴

## 时间安排
| 环节 | 时间 | 内容 |
|------|------|------|
| 30秒魔法时刻 | 1分钟 | 家长演示AI绘画 |
| 故事引入 | 5分钟 | 魔法画笔的故事 |
| 动手实践 | 25分钟 | 学习用文字画画 |
| 知识讲解 | 8分钟 | 了解AI绘画 |
| 创意任务 | 6分钟 | 设计自己的画作 |
