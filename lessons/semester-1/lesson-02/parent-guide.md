# 第二课：家长指导手册

> 文生图——AI画出我的梦想

## 授课目标

通过本节课，让孩子能够：
1. 体验AI画图的神奇——用文字生成图片
2. 学会用文字描述让AI画出想要的画面
3. 激发创意和想象力

## 时间分配建议

| 环节 | 时间 | 注意事项 |
|------|------|---------|
| 趣味开场 | 5分钟 | 用"用文字画画"吸引兴趣 |
| 认识AI画图工具 | 10分钟 | 展示界面，简单讲解 |
| 动手实践：AI画图 | 35分钟 | 让孩子主导创作 |
| 作品展示分享 | 8分钟 | 鼓励孩子分享 |
| 总结回顾 | 2分钟 | 简要回顾，预告下节课 |

## 授课要点

### 1. 趣味开场（5分钟）

**如何引入**：
- 用"用文字变出图画"来吸引兴趣
- 问问孩子最想看什么

**互动话术**：
- "你知道吗？用文字就能让电脑画画！"
- "你想看什么样的猫？什么样的城堡？"
- "只要你用文字描述出来，AI就能画出来！"

### 2. 认识AI画图工具（10分钟）

**打开工具**：
- 带着孩子打开豆包画图/文生一格
- 简单介绍一下界面

**讲解界面**：
- 输入框：在这里输入你想画的描述
- 生成按钮：点击生成图片
- 图片显示区：生成的图片会显示在这里

**简单演示**：
- 家长先示范一次简单的画图
- 让孩子看清楚怎么操作

**重点讲解**：
- "描述越详细，AI画得越好"
- "可以想象任何东西，即使现实生活中没有"

### 3. 动手实践：AI画图（35分钟）

**活动一：第一次画图——简单开始（8分钟）**

让孩子自己输入：
```
一只可爱的小猫
```

**家长引导**：
- 等待AI生成，耐心一点
- 看看AI画得怎么样
- "和你想象的一样吗？"

**可以再试试**：
- "一朵红色的花"
- "一只小狗"

---

**活动二：画一些有趣的组合（10分钟）**

让孩子输入有趣的组合：
```
一只飞在天上的猫
```
```
一只会跳舞的小狗
```

**家长引导**：
- "AI能听懂这些奇怪的描述吗？"
- "这些画有趣吗？"
- "你自己能想到什么？"

---

**活动三：添加更多细节（8分钟）**

**对比练习**：

先输入简单的：
```
一只小猫
```

再输入有细节的：
```
一只毛茸茸的橙色小猫，坐在绿色的草地上，旁边有好多花
```

**对比两张图**：
- 哪一张更漂亮？
- 为什么有细节的更好？

**教孩子添加细节的秘诀**：
- **颜色**: 什么颜色的？
- **形状**: 什么形状的？
- **位置**: 在哪里？
- **动作**: 在做什么？

---

**活动四：画Yanny想象中的世界（7分钟）**

**引导提问**：
- "你想看什么样的城堡？"
- "你想象中的花园是什么样子的？"
- "你最喜欢的动物在哪里玩耍？"

**让孩子创作**：
让孩子自己输入想画的东西

**记录**：
画的主题：________________
为什么想画这个：________________

---

**活动五：画一幅"我和我的梦想"（2分钟）**

让孩子输入：
```
一个8岁的小女孩，她的梦想是（让Yanny补充）
```

**这是最珍贵的作品**：
- 一定要截图保存
- 给这幅画起个名字
- 记录这是Yanny的梦想

---

### 4. 作品展示分享（8分钟）

**让Yanny展示**：
- 挑选最喜欢的3-5张图
- 给每张图起个名字
- 告诉爸爸妈妈为什么喜欢

**问Yanny**：
1. 今天画得开心吗？
2. 你最喜欢哪一幅画？
3. AI画的图和你想象的一样吗？
4. 下次还想画什么？

### 5. 总结回顾（2分钟）

**回顾要点**：
- "今天我们用文字让AI画了画"
- "学会了添加细节让AI画得更好"
- "创作了自己的AI作品"

**预告下节**：
- "下次课我们要学习AI语音聊天——和AI说说话！"

## 互动建议

### 增加趣味性

1. **比一比**: 家长和孩子一起画，看谁画的好
2. **起名字**: 给每幅画起有趣的名字
3. **讲故事**: 根据画的图编个小故事
4. **做画册**: 把所有画整理成一个画册

### 引导思考

- "你觉得AI是怎么知道你画什么的？"
- "AI画的和你想象的一样吗？哪里不一样？"
- "你觉得AI能画出任何东西吗？"

### 避免误区

- ❌ 不要用复杂的术语
- ❌ 不要要求画得"对"，AI画画是创作
- ❌ 不要限制孩子的想象力
- ❌ 不要批评AI画得不好
- ✅ 鼓励孩子的创意
- ✅ 肯定孩子的想象力
- ✅ 表扬孩子的每一个作品

## 孩子注意力管理

### 如果孩子注意力不集中：

1. **休息一下**：让活动2分钟
2. **简化内容**：只画简单的图
3. **减少数量**：不用画很多，画1-2幅就好
4. **缩短时间**：从60分钟减到30分钟

### 如果孩子特别感兴趣：

1. **延长创作**：让多画一会儿
2. **记录想法**：把Yanny的想法记下来
3. **限制时间**：虽然感兴趣，也要控制时间保护眼睛
4. **安排下次**：告诉孩子下次还可以画

---

## 安全注意事项

1. **不要生成不当内容**：引导孩子画积极健康的内容
2. **保护眼睛**：每次画图不要超过10分钟
3. **适度使用**：不要沉迷于AI画图
4. **真实创作**：除了AI画图，也要鼓励孩子手绘

---

## 常见问题处理

**Q: AI画的图和孩子想象的不一样怎么办？**
A: 告诉孩子AI有自己的"想法"，每次画都不同，这正是有趣的地方

**Q: 孩子觉得AI画得不好怎么办？**
A: 告诉孩子AI画的是"艺术"，不是"对错"，鼓励孩子继续尝试

**Q: 孩子想一直画怎么办？**
A: 要控制时间保护眼睛，可以约定每天画1-2幅就好

**Q: AI画出了奇怪的内容怎么办？**
A: 换一个描述就好，不用太在意
