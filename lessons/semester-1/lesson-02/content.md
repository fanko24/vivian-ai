# 第二课内容：文生图——AI画出我的梦想

## 🎯 趣味开场（5分钟）

### 你见过会画画的电脑吗？

今天我们要做一件超级神奇的事情——**用文字让电脑画画**！

> "我能听懂你说的话，然后把你描述的东西画出来。不管是飞在天上的猫，还是彩虹城堡，我都能画出来！"

**这就是"文生图"AI！** 🎨

### 什么是"文生图"？

"文生图"就是：
- **文** = 文字（你说的话）
- **生** = 生成/制作
- **图** = 图片/图画

所以**文生图**就是：**用文字让AI生成图片**！

## 🎮 动手实践：用文字让AI画画（35分钟）

### 活动一：第一次画图——简单开始（8分钟）

让Yanny输入一个简单的描述：

```
一只可爱的小猫
```

或者：

```
一朵红色的花
```

**等待生成，然后观察**：
- AI画得怎么样？
- 和你想象的一样吗？
- 想不想再试试？

**可以尝试更多简单的**：
- "一只小狗"
- "一个苹果"
- "一座房子"
- "一颗星星"

---

### 活动二：画一些有趣的组合（10分钟）

让Yanny发挥想象力，画一些有趣的组合：

```
一只飞在天上的猫
```

```
一只会跳舞的小狗
```

```
一个彩虹色的房子
```

```
一朵会说话的花
```

**观察讨论**：
- AI能理解这些奇怪的描述吗？
- 画出来的图片有趣吗？
- 你最喜欢哪一张？

---

### 活动三：添加更多细节（8分钟）

告诉Yanny：描述越详细，AI画得越好！

**试试添加细节**：

| 简单的描述 | 添加细节后 |
|-----------|-----------|
| 一只小猫 | 一只毛茸茸的橙色小猫，坐在绿色的草地上 |
| 一个蛋糕 | 一个漂亮的生日蛋糕，上面有很多水果和蜡烛 |
| 一座城堡 | 一座高高的城堡，在蓝天白云下，有彩虹 |

让Yanny试试：
```
一只毛茸茸的橙色小猫，戴着红色的小帽子，坐在绿色的草地上，旁边有好多花
```

**讨论**：
- 添加细节后，AI画得更好了吗？
- 你觉得还有什么可以添加的？

---

## 📝 本课重点词汇

- **文生图 (Text-to-Image)**: 用文字生成图片的AI技术
- **描述 (Description)**: 用语言说明某样东西是什么样子的
- **细节 (Detail)**: 图片里的小部分，让图片更具体
- **生成 (Generate)**: 创造、制作出来
