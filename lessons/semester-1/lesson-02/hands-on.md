# 第二课：动手实践

> 今天Yanny要用文字变出美丽的图画！

---

## 活动一：第一次画图——简单开始（8分钟）

### 任务

让Yanny打开AI画图工具，输入第一个画图请求！

### 打开工具

**推荐使用豆包画图**：
1. 打开 https://www.doubao.com/
2. 点击"画图"
3. 开始创作！

### 第一幅画

让Yanny输入：

```
一只可爱的小猫
```

**然后点击"生成"按钮**

### 观察

等AI生成完后，和Yanny一起看看：
- 画得怎么样？
- 是你想象中的样子吗？
- 喜不喜欢？

### 可以再试几个

如果Yanny有兴趣，可以再试试：
- "一朵红色的花"
- "一只小狗"
- "一座房子"

## 活动三：添加更多细节（8分钟）

### 任务

让Yanny学习如何添加细节，让AI画得更好！

### 对比练习

先输入简单的：
```
一只小猫
```

再输入有细节的：
```
一只毛茸茸的橙色小猫，坐在绿色的草地上，旁边有好多花
```

**对比两张图**，讨论：
- 哪一张更漂亮？
- 为什么有细节的更好？
- 还可以加什么细节？

### 让Yanny试试

让Yanny自己尝试添加细节：

| 简单描述 | 添加细节后 | 效果怎么样？ |
|---------|-----------|-------------|
| 一个蛋糕 | 一个漂亮的生日蛋糕，上面有很多草莓和蜡烛 |             |
|         |            |             |
|         |            |             |

### 小技巧

告诉Yanny添加细节的秘诀：
- **颜色**: 什么颜色的？
- **形状**: 什么形状的？大大的？小小的？
- **位置**: 在哪里？在天上？在草地上？
- **动作**: 在做什么？飞着？坐着？跳舞？

## 活动五：画一幅"我和我的梦想"（2分钟）

### 任务

让Yanny画一幅特别的作品！

### 输入内容

让Yanny输入：

```
一个8岁的小女孩，她的梦想是
```

**然后让Yanny补充自己的梦想**

例如：
```
一个8岁的小女孩，她的梦想是成为一名画家，在阳光下画画
```

### 保存这幅画

**这是最珍贵的作品！**
- 截图保存
- 起个名字
- 记录这是Yanny的梦想

## 🌟 实践小贴士

1. **耐心等待**: AI生成图片需要时间，不要着急
2. **鼓励尝试**: 不管画得怎么样，都要表扬Yanny的创意
3. **保存作品**: 每幅画都要截图保存
4. **记录想法**: 记录下Yanny每幅画的想法
5. **控制时间**: 35分钟的实践足够了，保护眼睛
