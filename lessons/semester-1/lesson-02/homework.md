## 作业1：画一幅"我的梦想" ⭐⭐⭐

### 任务

用AI画一幅代表你梦想的画！

### 画什么

让Vivian想一想：
- "我长大想做什么？"
- "我最想去哪里？"
- "我最喜欢什么样的世界？"

### 输入描述

让Vivian输入：

```
一个8岁的小女孩，她的梦想是
```

**然后补充自己的想法**

### 给这幅画起个名字

画的名字：____________________

### 写几句话

在画旁边写上：

```
我的梦想是：_________________________

因为：______________________________
```

## 家长提示

### 1. 每次画几幅就好

不用一次画很多，每天1-2幅就够了

### 2. 保存所有作品

每画一幅，都要截图保存，以后可以回顾

### 3. 鼓励创意

不管画得怎么样，都要表扬Vivian的创意和想象力

### 4. 记录想法

让Vivian说说每幅画的想法，家长帮忙记录

### 5. 控制时间

每次画图不要超过10分钟，保护眼睛
