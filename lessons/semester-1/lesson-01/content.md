# 第一课内容：认识ChatGPT——会聊天的AI

## 🎯 趣味开场（5分钟）

### 你知道吗？

今天我们要认识一个特别的朋友——**AI聊天助手**！

> "我住在电脑里，什么都知道，会说话，会写故事，会画画，还能帮你写作业。猜猜我是谁？"

**答案：ChatGPT！** 🤖

### 它是什么？

想象一下，如果你有一个**超级聪明的朋友**，他：
- 什么都知道 📚
- 24小时都在 💤
- 会说话、会写故事、会画画 ✍️
- 永远不会不耐烦 😊

这个朋友就是**AI对话助手**！最出名的叫**ChatGPT**，国内有**豆包**、**Kimi**、**文心一言**等。

## 🎮 动手实践：和AI聊天（25分钟）

### 活动一：第一次打招呼（5分钟）

让Yanny自己输入：

```
👧 输入：你好！我是Yanny，今年8岁。
```

看看AI怎么回答！

**家长引导**：
- 看看AI说了什么？
- AI说得友好吗？
- 想不想和AI多说几句？

---

### 活动二：让AI讲故事（8分钟）

让Yanny向AI请求：

```
👧 输入：给我讲一个关于小猫的故事，要很有趣！
```

**观察AI的回答**：
- 故事怎么样？有趣吗？
- 和妈妈/爸爸平时讲的故事有什么不同？

**可以尝试更多**：
- "再讲一个！"
- "讲一个关于恐龙的故事"
- "讲一个关于魔法的故事"

---

### 活动三：问AI问题（8分钟）

让Yanny想想自己想知道什么，然后问AI：

**可以问的问题**（家长可以引导）：
- "为什么天是蓝的？"
- "猫为什么会喵喵叫？"
- "世界上最大的动物是什么？"
- "彩虹是怎么形成的？"
- "为什么月亮会跟着我走？"

**活动记录**：

| Yanny问的问题 | AI怎么回答的？ | Yanny觉得回答怎么样？ |
|--------------|----------------|---------------------|
|              |                |                      |
|              |                |                      |

---

### 活动四：和AI玩游戏（4分钟）

让Yanny说：

```
👧 输入：我们来玩猜谜游戏吧！你出题我来猜。
```

或者：

```
👧 输入：我们来玩"你说我画"游戏，你描述一个动物，我来画。
```

看看AI怎么回应！
