## 作业1：和AI聊聊天 ⭐⭐⭐

### 任务

找时间再打开ChatGPT或豆包，和AI聊聊天！

### 可以聊什么

**关于自己**：
- "我叫Vivian，我今年8岁，我最喜欢____"
- "我今天做了____事，我觉得很开心"
- "我喜欢____，你喜欢什么？"

**问问题**：
- "为什么______？"
- "你知道______吗？"
- "给我讲一个关于______的故事"

**玩游戏**：
- "我们来玩猜谜游戏吧"
- "你出题目我来猜"
- "我们玩'你说我画'吧"

### 记录表

| 你对AI说什么？ | AI怎么回答你？ | 你觉得有趣吗？ |
|---------------|----------------|---------------|
|               |                |               |
|               |                |               |
|               |                |               |

### 小建议

- 每次聊5-10分钟就好
- 可以每天和AI聊几句
- 想聊什么就聊什么！

## 作业2：画一幅"我和AI" ⭐⭐

### 任务

画一幅画，画出你和AI聊天的场景！

### 画画要点

- 画出AI的样子（它可以是任何样子！）
- 画出你自己
- 画出你们在做什么（聊天、讲故事、玩游戏...）
- 用你喜欢的颜色

### 写几句话

在画旁边写上：

```
我画的是：
___________________________

我觉得AI：
___________________________
```

## 作业3：给AI出一道题 ⭐⭐⭐

### 任务

想一个有趣的问题或任务，让AI来完成！

### 可以试试

**让AI写**：
- "帮我写一首关于小猫的诗"
- "帮我写一个童话故事"
- "帮我写一封信给我的好朋友"

**让AI创作**：
- "编一个绕口令"
- "讲一个笑话"
- "编一个谜语"

**让AI帮忙**：
- "给我推荐几本好看的书"
- "告诉我一些科学小知识"
- "教我一个小游戏"

## 家长提示

### 1. 每次聊几分钟就好

不用一次聊很久，每次5-10分钟就够了

### 2. 保存有趣的对话

截图保存Vivian和AI的有趣对话，以后可以回顾

### 3. 控制屏幕时间

注意保护眼睛，每次聊天后看看远处休息一下

### 4. 鼓励孩子表达

无论Vivian和AI聊什么，都要鼓励她表达自己的想法

### 5. 安全提醒

- 不要告诉AI家里的地址、电话等个人信息
- 遇到奇怪的内容告诉爸爸妈妈
