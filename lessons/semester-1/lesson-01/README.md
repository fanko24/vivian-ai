# 第1课：和AI聊聊天

> 第一次与AI对话，感受AI的智能

## 课程信息
- **时长**：40分钟
- **工具**：豆包（doubao.com）
- **难度**：⭐

## 学习目标
1. 学会使用AI工具进行对话
2. 理解什么是AI对话
3. 能提出有趣的问题与AI交流

## 准备清单
- [ ] 平板电脑或电脑
- [ ] 豆包账号已登录（doubao.com）
- [ ] 安静的学习环境
- [ ] 家长陪伴

## 时间安排
| 环节 | 时间 | 内容 |
|------|------|------|
| 30秒魔法时刻 | 1分钟 | 家长演示AI对话 |
| 故事引入 | 5分钟 | 认识新朋友AI |
| 动手实践 | 20分钟 | 学习与AI对话 |
| 知识讲解 | 8分钟 | 了解AI对话和安全口诀 |
| 课后整理 | 6分钟 | 整理和分享 |
