# 第一课：动手实践

> 今天Yanny要亲自和AI对话，感受AI的神奇！

---

## 活动一：第一次和AI打招呼（5分钟）

### 任务

让Yanny打开ChatGPT/豆包/Claude，输入第一句话！

### 输入内容

让Yanny输入（可以打字也可以用语音）：

```
你好！我是Yanny，今年8岁，很高兴认识你！
```

### 家长引导

1. 观察AI怎么回答
2. 和Yanny一起讨论："AI说得友好吗？"
3. 问Yanny："想不想再和它说几句？"

### 记录下第一次对话

```

AI: （这里记录AI的回答）
```

## 🌟 实践小贴士

1. **让Yanny自己输入**: 尽量让孩子自己打字或语音输入
2. **耐心等待**: AI回答需要一点时间，不要着急
3. **多鼓励**: 无论AI回答得怎么样，都要表扬Yanny的参与
4. **保存对话**: 可以截图保存有趣的对话内容
5. **控制时间**: 25分钟的实践时间足够了，不要让孩子太累
