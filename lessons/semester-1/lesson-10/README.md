# 第10课：AI小老师

> 用AI辅助学习，探索科学问题，完成毕业展示

## 课程信息

- **时长**：50分钟
- **工具**：豆包（doubao.com）
- **难度**：⭐⭐

## 学习目标

1. 学会用AI正确辅助学习（不是抄答案）
2. 用AI探索好奇的科学问题，学会追问
3. 回顾成长，选出最佳作品，完成"毕业"仪式

## 准备清单

- [ ] 打开豆包
- [ ] 准备一道最近不会做的题目（用于学习练习）
- [ ] 准备好之前所有的AI作品（画、音乐、视频等）
- [ ] 家长陪伴

## 时间安排

| 环节 | 时间 |
|------|------|
| 故事引入 | 5分钟 |
| 知识讲解 | 10分钟 |
| 动手实践 | 30分钟 |
| 毕业仪式 | 5分钟 |
