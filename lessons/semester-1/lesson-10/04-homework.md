# 课后作业：AI小老师

## 必做任务：我的学习记录（10分钟）

用AI帮你学习一个知识点，并记录过程。

**任务步骤**：
1. 找一道这周你学习中觉得难的问题
2. 用今天学的"好问题"方式问AI
3. 按照AI教的方法，自己再试一次
4. 记录你学到了什么

**记录表格**：
| 我的问题 | AI教我的方法 | 我学会了吗 |
|----------|-------------|-----------|
| | | ✓ / 还需要练习 |

## 选做任务：科学探索日记（15分钟）

用AI探索2个你好奇的科学问题，记录发现！

**任务步骤**：
1. 想两个你一直想问的"为什么"
2. 用AI探索，每个问题至少追问一次
3. 用你喜欢的方式记录（画画、写字、或录音）

**记录方式（选一种）**：
- **画画**：画出你学到的科学知识
- **写字**：用自己的话写3-5句小结
- **表格**：

| 我的问题 | AI的回答 | 我又追问了什么 | 我学到了什么 |
|---------|---------|--------------|------------|
| | | | |
| | | | |
