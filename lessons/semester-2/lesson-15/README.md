# 第15课：我的成长故事绘本

## 课程信息
- **模块**：第二学期 - Yanny的AI创作工作室
- **时长**：50 分钟
- **难度**：⭐⭐⭐
- **核心体验**：用真实生活照片讲一个关于自己的成长故事

## 学习目标
1. 学会从照片中选出有故事性的画面
2. 学会用口述让AI帮忙把故事整理成连贯的文字
3. 学会把照片和文字排版成一本像绘本的小书

## 准备工作
- [ ] 平板电脑或电脑（已安装豆包或能访问 doubao.com）
- [ ] 豆包账号已登录
- [ ] **从四个活动中各选1-2张有故事性的照片**（游泳、钢琴、羽毛球、画画）
- [ ] 准备好PPT软件或WPS
- [ ] 家长陪伴

## 时间安排

| 环节 | 时间 | 内容 |
|------|------|------|
| 30秒魔法时刻 | 1 分钟 | 看看用真实照片做的成长绘本 |
| 故事开场 | 5 分钟 | 薇薇安的成长故事 |
| 知识讲解 | 8 分钟 | 怎么用照片讲故事 |
| 动手实践 | 25 分钟 | 和豆包一起做成长绘本 |
| 分享展示 | 4 分钟 | 看看自己的绘本 |
| 课后整理 | 2 分钟 | 整理和保存作品 |

> 本课工具：**豆包** (doubao.com) + PPT/WPS

---

用你的真实照片，讲一个属于你自己的成长故事！
