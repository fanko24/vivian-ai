# 第15课：互动故事绘本

> 用豆包创作故事并配图，制作可打印的PDF绘本

## 课程信息

- **时长**：55分钟
- **工具**：豆包（doubao.com）+ PPT/WPS
- **难度**：⭐⭐⭐

## 学习目标

1. 把第一学期第6课编的故事拆分成多个场景
2. 用豆包AI绘画为每个场景生成风格统一的插图
3. 在PPT中排版，制作完整的绘本并导出PDF

## 准备清单

- [ ] 找到第一学期第6课编的故事文本（"会飞的小猫"）
- [ ] 打开豆包（doubao.com）
- [ ] 打开 PPT 或 WPS 演示
- [ ] 准备打印机（可选，没有也能完成）
- [ ] 家长陪伴

## 时间安排

| 环节 | 时间 |
|------|------|
| 30秒魔法时刻 | 3分钟 |
| 回顾故事，分场景 | 10分钟 |
| 知识讲解 | 5分钟 |
| AI绘画+PPT排版 | 30分钟 |
| 作品分享 | 7分钟 |
