# 第 15 课：原一的线上画展

## 课程信息
- **模块**：模块四 - 用 AI 创造（学期二）
- **时长**：50 分钟
- **难度**：⭐⭐⭐
- **核心体验**：把自己的画作变成一个线上画展网页

## 学习目标
1. 用 Claude Code 创建画廊风格的网页
2. 学习策展——挑选作品、安排顺序、撰写介绍
3. 体验"创作 → 策展 → 分享"的完整过程

## 准备工作
- [ ] 准备 6-8 幅自己最得意的画作
- [ ] 手机或相机给画作拍照
- [ ] Claude Code 已打开（claude.ai/code）
- [ ] 豆包网页已打开（doubao.com）
- [ ] 家长陪伴

> 💡 **本课工具**：Claude Code（写网页）+ 豆包（帮忙写文字）

## 时间安排
| 环节 | 时间 | 内容 |
|------|------|------|
| 30秒魔法时刻 | 5 分钟 | 看一个漂亮的线上画展 |
| 挑选与拍照 | 10 分钟 | 选画、拍照、写介绍 |
| 动手实践 | 25 分钟 | 用 Claude Code 制作画展网页 |
| 调整美化 | 5 分钟 | 修改布局和样式 |
| 分享与讨论 | 5 分钟 | 展示作品，互相欣赏 |

## 学习小贴士
1. 挑选画作时，选你最喜欢的，不一定要选"最好看"的
2. 每幅画写一句话介绍就够了，说说你画它的时候在想什么
3. 网页的风格要让看画的人觉得舒服
