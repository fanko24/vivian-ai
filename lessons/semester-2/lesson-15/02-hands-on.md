# 动手实践：制作互动故事绘本

## 30秒魔法时刻

爸爸妈妈先给Yanny看一本做好的AI绘本（PDF文件或打印装订版）：

> "看！这是一本用AI画的绘本故事书！每一页都有漂亮的插图和文字，像书店里卖的一样！等一下，你的故事也能变成这样一本书！"

---

## 第1步：准备故事，分成场景（8分钟）

### 回忆或创作故事

想一个你喜欢的故事。如果想不出，在 **doubao.com** 的聊天框里输入：

> "请帮我写一个儿童故事，主角是一只叫'棉花'的小猫，住在棉花糖做的云上，有透明的小翅膀。故事大约200字，用小朋友能听懂的语言。"

### 把故事分成4-6个场景

每个场景就是一个画面，写在纸上：

| 场景 | 发生了什么 | 画什么画面 |
|------|-----------|-----------|
| 场景1 | | |
| 场景2 | | |
| 场景3 | | |
| 场景4 | | |
| 场景5 | | |

**例子（《会飞的小猫棉花》）：**
- 场景1：棉花住在棉花糖云上，每天舔一口云当早餐
- 场景2：棉花飞到地上，遇到不会飞的小狗团团
- 场景3：团团说想飞，棉花让团团骑在自己背上
- 场景4：它们一起飞过彩虹桥，看到了美丽的风景
- 场景5：它们成了最好的朋友，每天一起飞翔

---

## 第2步：用豆包画每一页插图（15分钟）

这一步最重要！要给每个场景画一张插图，风格要统一。

### 打开图像生成

1. 打开 **doubao.com**，点击左侧导航栏里的 **"图像生成"**

### 保持风格一致

每张图的描述都加上：**"儿童绘本插画风格，水彩风格，色彩明亮可爱"**，主角外貌描述保持一样。

### 输入场景描述

在输入框里输入画面描述，点击 **"生成"**。

**场景1的例子：**
> "儿童绘本插画风格，水彩风格，一只白色毛茸茸的小猫，有透明的小翅膀，趴在一朵粉色棉花糖做的云上，舔着云朵，蓝天背景，色彩明亮可爱"

**场景2的例子：**
> "儿童绘本插画风格，水彩风格，一只白色毛茸茸的小猫，有透明的小翅膀，飞到了草地上，面前站着一只棕色的小狗，色彩明亮可爱"

### 保存每一张图片

1. 等图片出现，选一张最喜欢的
2. 右键点击图片 → **"图片另存为……"** → 保存到桌面
3. 用场景编号命名，如"场景1.png""场景2.png"
4. 每个场景都画一张。如果某张风格差别大，用相同描述词重画那张就好

---

## 第3步：在PPT中排版成绘本（15分钟）

### 新建PPT

1. 双击打开 **WPS演示** 或 **PowerPoint**，选择空白演示文稿

### 制作封面（第1页）

1. 点击 **"插入" → "图片"**，选场景1的图片，拉大铺满页面
2. 插入文本框，在画面上方输入书名（48-60号字），如 **"会飞的小猫棉花"**
3. 再插入文本框，下方输入 **"文/Yanny  图/AI绘画助手"**

### 制作内页（每个场景一页）

1. 点击 **"新建幻灯片"**，选空白版式
2. **上半部分**：插入对应场景的图片，调整到占页面60-70%，居中
3. **下半部分**：插入文本框，输入故事文字（2-3句话），字号24-28号，居中对齐
4. 每个场景重复以上步骤

### 制作封底（最后一页）

1. 新建一张幻灯片
2. 写上 **"作者的话"**（比如"希望你也喜欢棉花和团团的故事！"）和制作日期

### 保存并导出PDF

1. 点击 **"文件" → "保存"**，文件名叫故事名字
2. 点击 **"文件" → "导出"**（或"另存为"），选择 **"PDF"** 格式导出

---

## 成果整理与分享

### 打印装订成真正的绘本

如果有彩色打印机：
1. 打印PDF，按顺序排好
2. 用订书机在左边订起来，或打孔后用丝带绑起来
3. 就是真正可以翻的绘本了！

### 分享给家人

- 通过微信把PDF发给家人
- 带到学校给好朋友翻翻看
- 也可以读给弟弟妹妹听！

> 这是你自己创作的第一本绘本，好好保存起来！

---

## 成品标准

- [ ] 封面：书名 + 主角图 + 作者信息
- [ ] 4-6页内页，每页有插图 + 故事文字
- [ ] 封底：作者的话或小彩蛋
- [ ] AI插图风格统一，画面清晰
- [ ] 文字字号够大（至少24号）
- [ ] 已导出为PDF文件

> 记住安全口诀：绘本里不要写真实的家庭住址哦！
