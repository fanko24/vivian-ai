# 知识小课堂：用照片讲故事

## 1. 什么是"故事线"——让照片变成故事

单独的照片只是一张张图片。但当它们按照一定的顺序排列起来，就变成了一个故事。

就像一本绘本：
- 翻开第一页 → 看到开始
- 翻到中间 → 看到发生了什么
- 翻到最后 → 看到结局

排列照片的两种方式：

**按时间排**：从最早的照片到最近的照片
- 好处：能看到变化和成长
- 适合：想展示"从不会到会"

**按主题排**：把同一个活动的照片放在一起
- 好处：每个主题是一个完整的段落
- 适合：想展示"我会的东西真多"

**小秘诀**：不管用哪种方式，每张照片下面都要有一段文字，讲讲照片背后的故事，这样别人才看得懂。

## 2. 口述故事让AI整理——说出你的真实感受

做成长绘本最重要的不是文字写得多漂亮，而是故事够真实。

你可以这样做：
1. 看着照片，想到什么就说什么
2. 告诉豆包你当时的心情、发生了什么、后来怎样了
3. 让豆包帮你把说的内容整理得更通顺

口述时可以想这些问题：
- 当时发生了什么？
- 你是什么心情？
- 后来怎么样了？
- 从这件事你学到了什么？

**小秘诀**：不要让AI编故事，只让它帮你整理你说的话。真实的感受才是最打动人的！

## 3. 成长的主题——每段故事都有一个收获

好的成长绘本有一个共同点：每段故事都让人学到一些东西。

看看这些例子：
- 第一次下水很害怕 → 但我学会了 → **勇敢**
- 练琴练了好多遍 → 终于弹对了 → **坚持**
- 羽毛球动作练不好 → 但我没有放弃 → **努力**
- 画了一幅画 → 把心里想的画出来了 → **表达**

这些"我学会了……"就是成长的证明！

在绘本里，把这些话用 **不同颜色** 或 **加粗** 标记出来，别人一看就知道你从每件事中收获了什么。

**小秘诀**：最后的"作者的话"可以总结一下：回头看看这些经历，你觉得最大的收获是什么？

---

今天学的三个本领：排列故事线、口述真实感受、找到成长主题！
