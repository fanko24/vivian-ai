# 绘本制作三步法

## 1. 故事分场景

做绘本的第一步不是画画，而是**分场景**。

就像拍电影一样：
- 导演先写好剧本，分成一场一场的戏
- 我们先把故事分成一个一个场景
- 每个场景就是绘本里的一页

**怎么分？**
- 开头（介绍主角）→ 1-2个场景
- 中间（发生事情）→ 2-3个场景
- 结尾（解决问题）→ 1-2个场景

总共 **4-6个场景** 就够啦！

## 2. AI绘画提示词公式

让豆包画出风格统一的插图，要用这个公式：

**主角外貌 + 动作 + 环境 + 风格**

| 部分 | 示例 |
|------|------|
| 主角外貌 | 白色毛茸茸的小猫，有透明的小翅膀 |
| 动作 | 趴在粉色棉花糖云上，舔着云朵 |
| 环境 | 蓝天背景，阳光温暖 |
| 风格 | 儿童绘本插画风格，水彩风格 |

**关键技巧**：每张图的"风格"和"主角外貌"要保持一样，这样整本绘本看起来才统一！

## 3. PPT排版秘诀

绘本排版有3个要点：

1. **封面**：大标题 + 主角图 + "文/Yanny"
2. **内页**：上图下文，每页一张图 + 一段话
3. **封底**：写一句"作者的话"

页面方向用**横向（16:9）**，图片占 60-70%，文字在下方居中。

## 小结

记住三步口诀：**分场景 → 画插图 → 排版导出！**
