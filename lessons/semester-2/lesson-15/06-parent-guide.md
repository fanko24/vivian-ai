# 家长指导手册：第 15 课 - Yanny的线上画展

## 课前准备

### 材料准备
1. 帮孩子把画作找出来，放在方便挑选的地方
2. 准备手机拍照，确保 Claude Code 可正常使用
3. 把拍好的照片传到电脑上

### 心理准备
- 涉及孩子的真实作品，多鼓励，不要评价"画得好不好"
- 重点在于"策展"——挑选、排序、写介绍
- 让孩子自己做决定，家长只做技术辅助

## 课堂引导

### 拍照技巧
帮孩子给画作拍清晰的照片：选光线充足的地方，画平放，手机从正上方垂直拍摄，不要开闪光灯。

### 引导策展
1. **挑选**：让孩子自己决定选哪些画，不要替她选
2. **排序**：问"你想让大家先看哪幅？最后看哪幅？"
3. **写介绍**：引导问题——"你画这幅画的时候在想什么？""你最喜欢这幅画的哪个地方？"

### 使用 Claude Code
- 帮孩子上传照片，鼓励她自己跟 Claude Code 对话
- 生成的网页先让孩子看，让她说出想改什么
- 可以用豆包帮忙润色介绍文字，但意思要是孩子自己的

## 安全注意事项

1. 画展网页不要包含过多个人隐私信息
2. 不要出现学校名称、家庭住址等
3. 分享链接时建议只发给熟悉的人

## 学习观察

- **策展能力**：能否自主挑选和排序作品
- **表达能力**：写介绍时是否流畅自然
- **技术操作**：与 Claude Code 的交互是否熟练
- **审美意识**：对网页布局有没有自己的想法

---

**温馨提示**：这节课的核心是让孩子学会整理和展示自己的作品，多鼓励孩子表达对画作的感受！
