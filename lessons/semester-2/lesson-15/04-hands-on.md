# 动手实践：制作互动故事绘本

## 30秒魔法时刻

爸爸妈妈展示一本做好的AI绘本（PDF或打印版），让Yanny翻一翻：

"看，这是一个小朋友用AI画的绘本！每一页都有漂亮的插图和文字。等一下，你的故事也能变成这样一本书！"

---

## 第一步：回顾故事，分成场景（10分钟）

1. 打开第一学期第6课编的故事文本
2. 如果找不到文本了，可以让豆包帮你重新生成：

```
请帮我写一个儿童故事，主角是一只叫"棉花"的小猫，
它住在棉花糖做的云上，有一双透明的小翅膀。
故事要有开头、中间和结尾，大约200字。
```

3. 把故事分成 **4-6 个场景**，每个场景写在一张纸上

**分场景模板：**

| 场景编号 | 这个场景发生了什么 | 配什么画面 |
|---------|-------------------|-----------|
| 场景1 | | |
| 场景2 | | |
| 场景3 | | |
| 场景4 | | |
| 场景5 | | |
| 场景6 | | |

**举个例子（《会飞的小猫棉花》）：**
- 场景1：棉花住在棉花糖云上，每天舔一口云当早餐
- 场景2：棉花飞到地上，遇到不会飞的小狗团团
- 场景3：团团说想飞，棉花让团团骑在自己背上
- 场景4：它们一起飞过彩虹桥，看到了美丽的风景
- 场景5：它们成了最好的朋友，每天一起飞翔

> 小提示：场景之间要有连贯性，像翻动画片一样一页接一页。

---

## 第二步：用豆包AI绘画画插图（15分钟）

每个场景画一张插图，一共需要 4-6 张。

### 2.1 打开豆包AI绘画

1. 打开浏览器，输入 `doubao.com`，按回车
2. 登录豆包账号
3. 点击"AI绘画"或图片生成功能

### 2.2 写提示词

按照"主角 + 动作 + 环境 + 风格"的公式写提示词。

**场景1的提示词（示例）：**

```
儿童绘本插画风格，一只白色毛茸茸的小猫，有透明的小翅膀，
趴在一朵粉色棉花糖做的云上，舔着云朵，
蓝天背景，阳光温暖，色彩明亮可爱，水彩风格
```

### 2.3 生成并挑选

1. 把提示词粘贴到豆包的输入框
2. 点击"生成"，等几秒钟
3. 豆包会生成几张图，选一张你最喜欢的
4. 右键保存图片，命名如"场景1-棉花在云上.png"

### 2.4 重复操作

对每个场景重复 2.2 和 2.3 的步骤，改一下提示词里的内容就好。

**保持风格一致的小技巧：**
- 每张图的提示词都加上"儿童绘本插画风格，水彩风格"
- 主角的外貌描述保持一样（白色毛茸茸的小猫、透明小翅膀）
- 色调保持温暖明亮

### 2.5 检查图片

把所有图片放在一个文件夹里，检查一下：

| 场景 | 文件名 | 图片内容对不对 | 风格统一吗 |
|------|--------|--------------|-----------|
| 场景1 | | ✅ / ❌ | ✅ / ❌ |
| 场景2 | | ✅ / ❌ | ✅ / ❌ |
| 场景3 | | ✅ / ❌ | ✅ / ❌ |
| 场景4 | | ✅ / ❌ | ✅ / ❌ |
| 场景5 | | ✅ / ❌ | ✅ / ❌ |

> 如果某张图不满意，重新生成那一张就好，不用全部重画。

---

## 第三步：在PPT中排版成绘本（15分钟）

### 3.1 新建PPT

1. 打开 PPT 或 WPS 演示
2. 新建一个空白演示文稿
3. 页面方向设为**横向**（设计 → 幻灯片大小 → 横向 16:9）

### 3.2 制作封面（第1页）

封面要包含这些内容：

1. **背景**：用场景1的图片作为封面背景
2. **书名**：大大的字，写在画面上方
   - 字号：48-60号
   - 字体：选一个可爱的字体
3. **作者信息**：画面下方写"文/Yanny 图/AI绘画助手"

**具体操作：**
1. 点击"插入" → "图片" → 选择场景1的图片 → 插入
2. 拉大图片铺满整个页面
3. 点击"插入" → "文本框" → 在画面上方画一个文本框
4. 输入书名，如"会飞的小猫棉花"
5. 再插入一个文本框，在下方输入"文/Yanny  图/AI绘画助手"

### 3.3 制作内页（第2页到倒数第2页）

每个场景做一页，总共 4-6 页内页。

**每页内页的做法：**

1. 插入一张新幻灯片（空白版式）
2. **上半部分放插图**：
   - 插入图片 → 选择对应场景的图片
   - 调整大小，占页面的 60-70%
   - 居中摆放
3. **下半部分放文字**：
   - 插入文本框，放在图片下方
   - 输入这个场景的故事文字（2-3句话）
   - 字号：24-28号
   - 文字居中对齐

**排版检查清单：**
- [ ] 图片居中，大小合适
- [ ] 文字在图片下方，居中对齐
- [ ] 字号足够大（至少24号）
- [ ] 每页只有一张图和一段文字

### 3.4 制作封底（最后一页）

封底可以放这些内容：

1. **"作者的话"**：写一句你想对读者说的话
   - "希望你也喜欢棉花和团团的故事！"
2. **日期**：写上制作日期
3. **可以画一个小装饰**：用一张小的AI插图或简单的图案

---

## 第四步：导出PDF（5分钟）

1. 点击"文件" → "导出" → "创建PDF"
2. 选择保存位置（桌面），文件名如"会飞的小猫棉花-绘本.pdf"
3. 点击"保存"
4. 打开PDF翻一遍，看看效果！

> 如果家里有打印机，可以打印出来，用订书机或打孔+丝带装订，就是一本真正的绘本了！

---

## 成品标准

完成的绘本应该包含：

- [ ] 封面：书名 + 主角图 + 作者信息
- [ ] 4-6页内页，每页有插图 + 故事文字
- [ ] 封底：作者的话或小彩蛋
- [ ] AI插图风格统一，画面清晰
- [ ] 文字字号够大，容易阅读
- [ ] 排版整齐美观
- [ ] 已导出为PDF文件

---

## 分享你的作品

1. 把做好的绘本PDF发给家人看
2. 如果打印出来了，带到学校给好朋友翻翻
3. 也可以读给弟弟妹妹听！

> 记住安全口诀！绘本里不要写真实的家庭住址哦。
