# 动手实践：制作线上画展

## 30 秒魔法时刻

爸爸妈妈先展示一个做好的画展网页，让原一看到效果："等一下，你的画也能像这样在网上展出！"

---

## 第 1 步：挑选画作（10 分钟）

1. 把你的画都拿出来，铺在桌子上或地上
2. 从里面挑出 6-8 幅你最想展示的画
3. 不一定要选"最好看"的，选你最喜欢的就好

> 小提示：可以选不同类型的画——水彩、蜡笔、彩铅搭配着来。

## 第 2 步：给画作拍照

1. 找一个光线好的地方（白天窗边最好）
2. 把画平放，用手机从正上方拍，尽量拍得端正
3. 把照片传到电脑上

> 小提示：拍照时不要开闪光灯，避免反光。

## 第 3 步：写作品介绍

给每幅画写一句话介绍：

| 画作 | 介绍示例 |
|------|----------|
| 小猫画 | "这是我家橘子，它最喜欢趴在窗台上晒太阳。" |
| 风景画 | "那天海风很暖，我想把这种感觉留住。" |
| 花卉画 | "外婆家的月季花开了，我用水彩画了下来。" |

> 可以请豆包帮忙润色，但意思要是你自己的。

## 第 4 步：用 Claude Code 制作画展网页（20 分钟）

打开 Claude Code，告诉它：

```
帮我做一个画展网页，叫"原一的小画展"。
要求：温馨可爱的风格，浅色背景，每幅画配上介绍，
点击画可以看大图，每行 2-3 幅，最上面有标题。
```

然后把你拍的照片和写的介绍发给 Claude Code。

## 第 5 步：安排画作顺序

想想画的顺序：第一幅放你最喜欢的，最后一幅放最有意义的。不满意可以跟 Claude Code 说："帮我把第 3 幅和第 5 幅换一下位置。"

## 第 6 步：调整美化（5 分钟）

看看网页，想想还能改什么：背景颜色、字体大小、画的间距。跟 Claude Code 说你想改的地方。

## 完成检查

- [ ] 画展网页打开了，画都能看到
- [ ] 每幅画下面都有介绍
- [ ] 点击画能看大图
- [ ] 顺序和风格都满意

---

记住安全口诀！做好后先给爸爸妈妈看，确认没问题再分享给别人。
