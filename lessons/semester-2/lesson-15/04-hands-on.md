# 动手实践：制作线上画展

## 30 秒魔法时刻

爸爸妈妈先展示一个做好的画展网页，让Yanny看到效果："等一下，你的画也能像这样在网上展出！"

---

## 第 1 步：挑选画作（10 分钟）

1. 把你的画都拿出来，铺在桌子上或地上
2. 从里面挑出 **6-8 幅**你最想展示的画
3. 不一定要选"最好看"的，选你最喜欢的就好

**挑选小提示：**
- 数量：6 幅最少，8 幅最佳，不要超过 10 幅（太多会显得拥挤）
- 类型搭配：尽量选不同类型的画，比如：
  - 🖌️ 水彩画 1-2 幅（颜色柔和的）
  - 🖍️ 蜡笔画 1-2 幅（色彩鲜艳的）
  - ✏️ 铅笔/彩铅画 1-2 幅（线条细致的）
  - 其他类型的画 1-2 幅（剪纸、拼贴等都可以）
- 选画的时候可以问自己："我想让别人看到什么样的我？"

> 小提示：可以选不同类型的画——水彩、蜡笔、彩铅搭配着来。

## 第 2 步：给画作拍照

**拍照前准备：**
1. 找一个光线好的地方——白天窗边最好，自然光让颜色最真实
2. 把画平放在干净的桌面或地板上，下面垫一张白纸
3. 把画旁边的东西都拿走，只留画作本身

**拍照时注意：**
1. 打开手机相机，**关掉闪光灯**（闪光灯会让画反光、看不清颜色）
2. 手机拿在画的正上方，尽量和画面平行（不要歪着拍）
3. 画面要尽量填满手机屏幕，但四周留一点白边
4. 每幅画拍 2-3 张，选最清楚的那张
5. 拍完后放大看看——字和线条清楚吗？颜色跟原画像吗？

**把照片传到电脑：**
1. 用数据线连接手机和电脑，或者用微信/QQ"文件传输助手"发送
2. 在电脑上新建一个文件夹，叫"画展照片"
3. 把照片都放到这个文件夹里
4. 把照片重新命名，比如"01-小猫.jpg""02-海边.jpg"（方便后面排序）

> 小提示：拍照时不要开闪光灯，避免反光。

## 第 3 步：写作品介绍

先用自己的话填写这个模板，每幅画写一张：

```
作品名称：________________
用什么画的：________________（水彩/蜡笔/彩铅/其他）
我画的是什么：________________
我最喜欢这幅画的哪里：________________
```

**举个例子：**

| 画作 | 她先写的草稿 | 润色后变成 |
|------|-------------|-----------|
| 小猫画 | 我家橘子在晒太阳 | "这是我家橘子，它最喜欢趴在窗台上晒太阳。" |
| 风景画 | 海边很暖和 | "那天海风很暖，我想把这种感觉留住。" |
| 花卉画 | 外婆家的花开了 | "外婆家的月季花开了，我用水彩画了下来。" |

**用豆包帮你润色：**

1. 打开豆包（doubao.com）
2. 输入这段话：

```
我写了几句画画的介绍，帮我把每句话改得更好听、更生动，但不要改我的意思：
1. [把你写的话贴在这里]
2. [把你写的话贴在这里]
3. ...（每幅画的介绍都贴上）
```

3. 豆包会帮你改得更好听，你看看喜不喜欢
4. 不满意可以跟豆包说："第二句再活泼一点"或"第三句太长了，帮我缩短"

> 记住：意思要是你自己的，豆包只是帮你表达得更好听。

## 第 4 步：用 Claude Code 制作画展网页（20 分钟）

**打开终端：**
1. 爸爸妈妈按 `Cmd + 空格`（Mac）或点击搜索按钮，输入"终端"或"Terminal"
2. 打开终端应用，会看到一个黑色或白色的窗口，有一个闪烁的光标

**启动 Claude Code：**
1. 在终端里输入 `claude`，然后按回车键
2. 等几秒钟，出现欢迎信息和输入框，就可以开始说话了

**告诉 Claude Code 你的需求：**
1. 在 Claude Code 的输入框里，输入这段话：

```
帮我做一个画展网页，叫"Yanny的小画展"。
要求：温馨可爱的风格，浅粉色背景，每幅画配上介绍文字，
点击画可以看大图，每行展示 2-3 幅画，最上面有标题，
标题下面写一句欢迎语"欢迎来到我的画展！"。
```

2. 按 `Shift + Enter` 发送，等待 Claude Code 生成网页

**把画作照片给 Claude Code：**
1. 把"画展照片"文件夹拖到终端窗口里，终端会自动显示文件夹的路径
2. 或者告诉 Claude Code：

```
我的画展照片在这个文件夹：[把文件夹拖进来，会自动变成路径]
照片名字是 01-小猫.jpg 到 08-花园.jpg。
请把这些照片放到网页里，按照编号顺序排列。
```

3. 同时把作品介绍也告诉 Claude Code：

```
每幅画的介绍：
第1幅（01-小猫.jpg）："这是我家橘子，它最喜欢趴在窗台上晒太阳。"
第2幅（02-海边.jpg）："那天海风很暖，我想把这种感觉留住。"
...（把所有介绍都贴上）
```

**等待生成和预览：**
1. Claude Code 会在几秒到一分钟内生成网页
2. 看到提示后，在浏览器中打开生成的 HTML 文件查看效果
3. Mac 用户：在 Finder 中找到生成的文件，双击打开
4. 或者在终端输入 `open 画展网页.html`（Mac）/ `start 画展网页.html`（Windows）

> 如果遇到问题，直接跟 Claude Code 说哪里不对，它会帮你修好。

## 第 5 步：安排画作顺序

想想画的顺序：第一幅放你最喜欢的，最后一幅放最有意义的。

**可以跟 Claude Code 这样说：**
- "帮我把第 3 幅和第 5 幅换一下位置"
- "把那幅海边日落放在第一个，它是我的最爱"
- "最后一幅放全家福那幅，因为最有纪念意义"
- "把色彩最鲜艳的两幅放在开头，吸引大家的目光"

调完之后在浏览器里刷新页面，看看新顺序好不好看。

## 第 6 步：调整美化（5 分钟）

看看网页，想想还能改什么。跟 Claude Code 说你想改的地方。

**调整示例——你可以这样说：**
- "背景颜色换一种，浅黄色试试"
- "画的间距再大一点，不要太挤"
- "标题字再大一点，加个可爱的字体"
- "每幅画加一个圆角边框，像相框一样"
- "鼠标放到画上的时候，画稍微变大一点"
- "页面最下面加一行字：'谢谢来看我的画展！'"
- "大图查看的时候加个关闭按钮"
- "每幅画下面加上我是用什么画的信息（水彩/蜡笔/彩铅）"

**每次只改一两个地方**，改完刷新看看效果，满意了再改下一个。

## 成品标准

完成的画展网页应该包含：

- [ ] 展示 6-8 幅画作，每幅画都能清楚看到
- [ ] 每幅画下面有作品介绍（名称 + 一句话描述）
- [ ] 点击画作可以查看大图，有关闭按钮
- [ ] 页面最上方有标题（如"Yanny的小画展"）
- [ ] 排版美观，每行 2-3 幅画，间距适中
- [ ] 整体风格温馨可爱，颜色搭配舒服
- [ ] 画作顺序合理（最喜欢的在前面）
- [ ] 网页可以在浏览器中正常打开和浏览

---

## 分享你的作品

1. 告诉 Claude Code："请把所有图片内嵌到 HTML 文件中，让这个网页可以单独打开，不需要网络也能看"
2. 把生成的 HTML 文件通过微信发给奶奶、姥姥等家人
3. 家人在手机上点击文件 → 选择"用浏览器打开"就能看！

> 也可以在浏览器中打开网页，按 Ctrl+P（Mac 按 Cmd+P）保存为 PDF，打印出来做成纸质版！

---

记住安全口诀！做好后先给爸爸妈妈看，确认没问题再分享给别人。
