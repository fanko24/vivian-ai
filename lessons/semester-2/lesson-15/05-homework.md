# 课后任务：分享你的画展

## 必做任务：邀请三位观众（10 分钟）

把你做好的画展网页链接发给 3 个人（爷爷奶奶、好朋友、老师等），然后记录反馈：

| 观众 | 最喜欢哪幅画 | 说了什么 |
|------|-------------|---------|
| 第1位：______ | ______ | ______ |
| 第2位：______ | ______ | ______ |
| 第3位：______ | ______ | ______ |

> 想一想：大家的反馈有没有让你对自己的画有新的发现？

---

## 选做任务：添加留言墙（15 分钟）

给你的画展加一个"观众留言区"。跟 Claude Code 说：

```
帮我在画展网页最后加一个"留言墙"，
参观的人可以写一句话，风格要和画展搭配。
```

---
