# 动手实践：我的游泳日记

## 30秒魔法时刻

先看看这份AI帮忙做的游泳日记画册是什么样的！（家长展示一个提前做好的示例PPT或截图）

是不是很漂亮？今天Yanny也能做出一本属于自己的游泳画册！

---

## 第1步：打开豆包，聊聊游泳的故事（10分钟）

**打开豆包**：
1. 在浏览器地址栏输入 **doubao.com**，按回车键打开豆包网页
2. 如果还没登录，点击页面右上角的 **"登录"** 按钮，用手机号或微信扫码登录
3. 登录成功后，点击页面左上角的 **"开始新对话"**（或者聊天框上方的 **"+"** 按钮），打开一个全新的聊天窗口

**开始聊天**：在页面下方的聊天输入框里，输入下面这段话，然后点击输入框右边的 **发送按钮**（一个小箭头图标），或者直接按键盘上的 **回车键**：

> "豆包你好！我想把我的游泳比赛经历写成日记。我参加过3次游泳比赛，最近2次都拿了冠军。你能帮我整理成日记吗？我先跟你讲讲第一次比赛的情况……"

然后把你记得的比赛故事讲给豆包听。可以一段一段地讲：
- 第一次比赛：心情怎么样？游的是什么姿势？结果如何？
- 第二次比赛：和第一次有什么不一样？拿了冠军是什么感觉？
- 第三次比赛：最难忘的一个瞬间是什么？

**期待的结果**：豆包会回复一段整理好的日记文字，读起来通顺、像一篇小日记。

如果豆包写的太长或太难懂，你可以在聊天框里接着说：
> "请用一年级小朋友能看懂的语言，写简短一点。"

**保存日记文字**：
1. 用鼠标选中豆包写的日记文字（从第一个字拖到最后一个字，文字会变成蓝色）
2. 点击鼠标右键，选择 **"复制"**（或者按键盘 **Ctrl+C**，Mac电脑按 **Command+C**）
3. 打开电脑上的记事本（Windows）或文本编辑（Mac），点击右键选择 **"粘贴"**（或者按 **Ctrl+V** / **Command+V**）
4. 点击 **"文件" → "保存"**，文件名写成"游泳日记文字"，保存到桌面上

> 小提示：每次豆包写完一段，都复制保存一次，这样就不会丢失啦！

---

## 第2步：让豆包帮你画插图（8分钟）

从你的比赛经历中，选2-3个最喜欢的画面，让豆包帮你画出来。

**开始画图**：
1. 还是在刚才和豆包聊天的页面里（也可以新建一个对话）
2. 在聊天输入框里，输入你想要的画面描述，然后点击 **发送按钮**
3. 等待大约 **10-30秒**，豆包会生成图片并显示在聊天窗口里

比如你可以这样输入：

> 画面一："请画一个8岁小女孩站在游泳比赛的跳台上，她穿着泳衣戴着泳帽泳镜，表情很自信，背景是室内游泳馆，旁边有裁判准备吹哨"

> 画面二："请画一个扎马尾辫的小女孩在泳池里奋力游泳，水花四溅，她在做自由泳的动作，旁边泳道也有选手在游"

> 画面三："请画一个小女孩站在游泳比赛的领奖台上，她脖子上挂着金牌，开心地举起双手，观众在鼓掌"

**等待生成**：豆包会显示"正在生成中……"，耐心等一小会儿，图片就会出现。

**如果不满意**：在聊天框里告诉豆包哪里要改。比如：
> "请把表情画得更开心一点"
> "请加上水花的效果"

豆包会重新画一张。

**保存图片**：
1. 看到满意的图片后，用鼠标 **右键点击** 图片
2. 在弹出的菜单中选择 **"图片另存为……"**
3. 在弹出的保存窗口中，选择保存到 **桌面**
4. 把文件名改成好记的名字，比如"游泳图1""游泳图2""游泳图3"
5. 点击 **"保存"** 按钮
6. 每张图都这样保存一次

> 小提示：如果用手机或平板，长按图片，选择"保存到相册"就可以啦！

---

## 第3步：组装你的电子画册（7分钟）

现在你有了日记文字和插图，让我们把它们组装成电子画册！

**打开软件**：
1. 在电脑上找到 **WPS演示** 或 **PowerPoint**（PPT）的图标，双击打开
2. 如果看到"新建"页面，选择一个 **空白演示文稿**，点击创建

**做封面（第1页）**：
1. 点击幻灯片中间写着"单击此处添加标题"的地方，输入 **"Yanny的游泳日记"**
2. 点击下方"单击此处添加副标题"的地方，输入今天的日期，比如 **"2026年4月"**
3. 插入游泳插图：点击顶部菜单栏的 **"插入" → "图片"**，在弹出的窗口中找到桌面上的游泳图片，点击 **"插入"**
4. 用鼠标拖动图片到合适的位置（可以放在标题下方），拖动图片角上的小圆点可以调整大小

**做内容页（第2页起）**：
1. 点击左侧幻灯片缩略图区域，再点击顶部菜单的 **"新建幻灯片"**（或"插入"→"新幻灯片"），选择一个有标题和内容的版式
2. 在标题栏输入小标题，比如 **"我的第一次比赛"**、**"冠军！冠军！"**
3. 在内容区域：点击 **"插入" → "图片"**，选择对应的游泳插图插入
4. 在图片旁边或下方：点击 **"插入" → "文本框" → "横向文本框"**，然后在页面上拖出一个框，把之前保存的日记文字粘贴进去（从记事本里复制）
5. 重复以上步骤，为每一场比赛做一个页面

**调整排版**：
- 用鼠标点击文字，在顶部工具栏把字号改成 **28号或32号**，字大一点更好看
- 点击文字后可以在顶部选择字体，推荐用 **"微软雅黑"** 或 **"楷体"**，看起来更舒服
- 用鼠标拖动图片和文字框来调整位置，让页面看起来整齐
- 文字放在上面或左边，图片放在下面或右边

**别忘了保存**：点击左上角的 **"文件" → "保存"**（或按 **Ctrl+S** / **Command+S**），文件名叫"Yanny的游泳日记"，保存到桌面。

## 成品标准

完成的作品应该包含：

- [ ] 封面页（标题"Yanny的游泳日记" + 日期 + 游泳插图）
- [ ] 至少3页内容页（每页写一段比赛故事）
- [ ] 每页都有文字说明和配图
- [ ] 文字用小朋友能看懂的语言（一年级水平）
- [ ] 排版整齐，字号够大（28号以上）
- [ ] 每页都有小标题（比如"我的第一次比赛"）

---

## 分享你的作品

1. 把画册导出为 PDF：在 PPT 或 WPS 中点击"文件" → "导出" → 选择 PDF 格式
2. 也可以导出为图片：每页另存为图片，挑几张好看的发微信
3. 把文件通过微信发给奶奶、姥姥等家人，手机上直接打开就能看！

> 也可以用彩色打印机打印出来，做成纸质版送给家人！

做好了别忘了给爸爸妈妈看看你的作品！
