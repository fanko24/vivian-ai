# 第11课：我的游泳成长册

## 课程信息
- **模块**：第二学期 - Yanny的AI创作工作室
- **时长**：45 分钟
- **难度**：⭐⭐
- **核心体验**：用真实照片和AI帮忙，做一本游泳成长画册

## 学习目标
1. 学会从照片中挑选最好的素材
2. 学会把照片发给豆包，让AI帮忙写照片说明
3. 学会把照片和文字组合成电子画册

## 准备工作
- [ ] 平板电脑或电脑（已安装豆包或能访问 doubao.com）
- [ ] 豆包账号已登录
- [ ] **提前选好5-8张游泳训练照片**（从刚开始学到现在，越早的越好）
- [ ] 把照片传到电脑或平板上（方便操作）
- [ ] 准备好PPT软件或WPS（家长提前安装好）
- [ ] 家长陪伴

## 时间安排

| 环节 | 时间 | 内容 |
|------|------|------|
| 30秒魔法时刻 | 1 分钟 | 看看用真实照片做的游泳成长册 |
| 故事开场 | 5 分钟 | 薇薇安的游泳成长故事 |
| 知识讲解 | 8 分钟 | 怎么用AI帮忙整理照片 |
| 动手实践 | 25 分钟 | 和豆包一起做游泳成长册 |
| 分享展示 | 4 分钟 | 看看自己做的成长册 |
| 课后整理 | 2 分钟 | 整理和保存作品 |

> 本课工具：**豆包** (doubao.com) - 免费AI助手，能看懂照片并帮你写说明
