# 动手实践：我的游泳日记

## 30秒魔法时刻

爸爸妈妈先给Yanny看一份做好的游泳日记画册（PDF文件或打印版）：

> "看！这是一本用AI帮忙做的游泳日记画册。封面有漂亮的插图，每一页都记录了一场比赛的故事。今天，你也能做出一本属于自己的游泳画册！"

翻开画册，看看每一页：有AI画的游泳插图，有关于比赛的文字故事，排版整齐又好看。等一下，Yanny的游泳日记也会变成这样！

---

## 第1步：和豆包聊游泳故事（10分钟）

### 打开豆包

1. 打开浏览器，在地址栏输入 **doubao.com**，按回车
2. 如果还没登录，点击页面右上角的 **"登录"** 按钮，用手机号或微信扫码登录
3. 登录后看到聊天界面，底部有一个输入框

### 开始讲故事

在底部输入框里打字（或者用语音输入），输入下面这段话，然后点击输入框右边的 **发送按钮**（蓝色箭头）：

> "豆包你好！我想把我的游泳经历写成日记。我参加过游泳比赛，请你先问我几个问题，帮我回忆比赛的故事吧！"

豆包会问你一些问题，比如"你参加了什么比赛？""心情怎么样？"一个一个回答它。

### 让豆包整理成日记

聊完之后，输入：

> "请把我们刚才聊的整理成一篇日记，用一年级小朋友能看懂的语言，每篇100字左右。帮我写3篇：第一次学游泳、一次难忘的比赛、我最喜欢游泳的原因。"

### 保存日记文字

1. 用鼠标选中豆包写的日记文字，从第一个字拖到最后一个字，文字会变成蓝色
2. 点击鼠标右键，选择 **"复制"**（或者按键盘 **Ctrl+C**，Mac电脑按 **Command+C**）
3. 打开电脑上的记事本（Windows）或文本编辑（Mac），粘贴保存，文件名叫"游泳日记文字"

> 小提示：每篇日记都复制保存一次，这样就不会丢啦！

---

## 第2步：让豆包画插图（8分钟）

选2-3个最喜欢的画面，让豆包帮你画出来。

### 打开图像生成

1. 还是在豆包页面，点击左侧导航栏里的 **"图像生成"**
2. 页面会切换到图像生成界面，中间有一个输入框

### 输入画面描述

在输入框里输入你想要的画面描述，然后点击 **"生成"** 按钮。比如：

> 画面一："一个8岁小女孩站在游泳馆的泳池边，穿着泳衣戴着泳帽和泳镜，表情又紧张又兴奋，卡通可爱风格"

> 画面二："一个扎马尾辫的小女孩在泳池里奋力游泳，水花四溅，在做自由泳动作，卡通可爱风格"

> 画面三："一个小女孩站在领奖台上，脖子上挂着金牌，开心地举起双手，卡通可爱风格"

### 保存图片

1. 等大约10-30秒，图片会出现在页面上
2. 如果不满意，告诉豆包哪里要改，比如"请把表情画得更开心一点"
3. 看到满意的图片后，用鼠标 **右键点击** 图片，选择 **"图片另存为……"**
4. 保存到桌面，文件名改成"游泳图1""游泳图2""游泳图3"

---

## 第3步：在PPT中组装画册（10分钟）

现在有了日记文字和插图，让我们把它们组装成电子画册！

### 打开PPT

1. 在电脑上找到 **WPS演示** 或 **PowerPoint**（PPT）的图标，双击打开
2. 选择一个 **空白演示文稿**，点击创建

### 制作封面（第1页）

1. 点击页面中间写着"单击此处添加标题"的地方，输入 **"Yanny的游泳日记"**
2. 点击下方"单击此处添加副标题"的地方，输入日期，比如 **"2026年4月"**
3. 插入游泳插图：点击顶部菜单栏的 **"插入" → "图片"**，在弹出的窗口中找到桌面上的游泳图片，点击 **"插入"**
4. 用鼠标拖动图片到合适的位置，拖动图片角上的小圆点调整大小

### 制作内容页（第2页起）

1. 点击顶部菜单的 **"新建幻灯片"**（或"插入"→"新幻灯片"），选择有标题和内容的版式
2. 在标题栏输入小标题，比如 **"第一次学游泳"**、**"难忘的比赛"**
3. 插入图片：点击 **"插入" → "图片"**，选择对应的游泳插图
4. 插入文字：点击 **"插入" → "文本框" → "横向文本框"**，在页面上拖出一个框，把之前保存的日记文字粘贴进去
5. 每篇日记做一个页面，重复以上步骤

### 调整排版

- 用鼠标点击文字，在顶部工具栏把字号改成 **28号或32号**
- 推荐用 **"微软雅黑"** 或 **"楷体"** 字体
- 文字放在上面或左边，图片放在下面或右边

### 保存文件

点击 **"文件" → "保存"**（或按 **Ctrl+S** / **Command+S**），文件名叫"Yanny的游泳日记"，保存到桌面。

---

## 成果整理与分享

### 导出PDF

1. 在PPT或WPS中点击左上角 **"文件" → "导出"**（或"另存为"）
2. 在格式选项中选择 **"PDF"**
3. 选择保存到桌面，文件名"Yanny的游泳日记.pdf"，点击 **"保存"**
4. 打开PDF翻一遍，看看效果！

### 分享给家人

- 打开微信，把PDF文件发给奶奶、姥姥等家人，手机上直接打开就能看
- 如果家里有彩色打印机，可以打印出来，做成纸质版小画册
- 也可以导出为图片，挑几张好看的发到家庭群里

---

## 成品标准

- [ ] 封面页（标题"Yanny的游泳日记" + 日期 + 游泳插图）
- [ ] 至少3页内容页（每页一篇日记故事）
- [ ] 每页都有文字说明和配图
- [ ] 文字用小朋友能看懂的语言
- [ ] 排版整齐，字号够大（28号以上）
- [ ] 已导出为PDF文件

> 记住安全口诀：不告诉AI真实姓名和家庭住址哦！
