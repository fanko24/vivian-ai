# 动手实践：制作个人网站（毕业作品）

## 30秒魔法时刻

家长打开 Claude Code，输入：

> "帮我做一个个人网站，标题叫'你好，我是Yanny'，温馨可爱风格，有导航栏可以跳到每个板块，浅色背景。"

几秒钟后，一个漂亮的个人网站框架出现在屏幕上！

"看！这就是你今天要做的东西——一个完全属于你的个人网站！把你这一学期做的所有作品都放进去！"

---

## 第1步：规划网站结构（10分钟）

拿出一张A4纸，横着放，画出你的网站地图。

**参考网站地图：**

```
你好，我是Yanny！（首页）
  ├── 自我介绍（头像 + 一段话 + 我的爱好）
  ├── 游泳世界（第17课的比赛时间线）
  ├── 我的画作（第19课的线上画展）
  ├── 钢琴角（练习心得 + 喜欢的曲子）
  └── 羽毛球天地（小教练手册）
```

**跟爸爸妈妈讨论每个板块放什么：**
- 自我介绍：你想让别人第一眼了解你什么？
- 游泳世界：放第17课做的比赛时间线网页
- 我的画作：放第19课做的线上画展网页
- 钢琴角：放你喜欢的曲子和学钢琴的故事
- 羽毛球天地：简单写几句你打球的故事

> 小提示：板块不要超过5-6个，太多了网站会太长。

---

## 第2步：用豆包准备素材（10分钟）

### 2.1 写自我介绍

1. 打开浏览器，输入 `doubao.com`，按回车
2. 在对话框中输入：

> "我是一个8岁的女孩，叫Yanny，上一年级。我喜欢游泳、画画、弹钢琴和打羽毛球。我的性格是开朗、爱笑、喜欢交朋友。帮我写一段有趣的自我介绍，100字以内，要让人一看就觉得我很可爱！"

3. 看看豆包写的喜不喜欢，不满意就说：
   - "太长了，帮我缩短到50个字"
   - "加上我还会画画这个爱好"
   - "语气再活泼一点"
4. 满意后复制保存下来

### 2.2 做头像

1. 点击豆包左侧导航栏的 **"图像生成"**
2. 输入：

> "帮我画一张可爱的小女孩头像，卡通风格，短头发，穿着彩色衣服，背景是浅粉色，圆圆的脸蛋，大大的眼睛，风格温馨可爱，像绘本里的插画。"

3. 在图片上 **右键** → **"另存为"**，保存到桌面
4. 不满意就说："头发换成长头发"或"衣服换成蓝色"

### 2.3 准备其他素材

- 找1-2张你最满意的个人照片（游泳、画画、弹琴的都可以）
- 想一句网站的欢迎语，比如"欢迎来到我的小天地！"

---

## 第3步：用Claude Code制作网站（20分钟）

### 3.1 打开终端并启动 Claude Code

1. **Mac电脑**：按 `Cmd + 空格`，输入 `终端`，按回车
   **Windows电脑**：按 `Win + R`，输入 `cmd`，按回车
2. 在终端里输入 `claude`，按回车，等几秒钟

### 3.2 输入需求

在 Claude Code 里输入：

> "帮我做一个个人网站，叫「你好，我是Yanny」。
> 风格温馨可爱，浅色背景。
> 板块：首页（自我介绍+头像）、游泳世界、我的画作、钢琴角、羽毛球天地。
> 最上面要有导航栏，可以点击跳到每个板块。
> 首页显示我的自我介绍和头像，其他板块先留着空位，我后面再填充内容。
> 把所有代码写在一个 index.html 文件中。"

### 3.3 预览网页

在终端输入 `open index.html`（Mac）或在文件夹中双击 `index.html`，看看整体框架是不是你想要的。

---

## 第4步：整合之前的项目（15分钟）

现在要把第17课和第19课做的项目放到网站里。

### 4.1 找到之前的项目文件

| 来源课程 | 项目内容 | 放到哪个板块 |
|----------|---------|-------------|
| 第17课 | 游泳比赛时间线 | 游泳世界 |
| 第19课 | 线上画展 | 我的画作 |

> 第16课做的是PPT日历，不是网页，不能直接整合到网站里。

### 4.2 整合游泳时间线和画展

把之前的 HTML 文件 **拖到终端窗口** 里，告诉 Claude Code：

> "这是我的游泳时间线网页：[把第17课的HTML文件拖进来]
> 这是我的画展网页：[把第19课的HTML文件拖进来]
>
> 请把游泳时间线的内容放到「游泳世界」板块，
> 把画展的内容放到「我的画作」板块。
> 钢琴角和羽毛球天地我自己来写。"

### 4.3 填写钢琴角

告诉 Claude Code：

> "在「钢琴角」板块，写上我最喜欢的钢琴曲和学琴的故事。
> 我学钢琴已经___了，最喜欢的曲子是___。"

### 4.4 填写羽毛球天地

告诉 Claude Code：

> "在「羽毛球天地」板块，帮我写一些简单的内容：
> - 我什么时候开始学打羽毛球的
> - 我最喜欢的羽毛球动作
> - 给初学者3个小建议"

### 4.5 放上自我介绍和头像

把头像图片拖到终端窗口，告诉 Claude Code：

> "这是我的头像，请放到首页自我介绍旁边。
> 自我介绍的文字是：___（把豆包写的粘贴进来）"

**每整合一个板块，都去浏览器刷新看看效果。**

---

## 第5步：美化与调整（10分钟）

检查网站：每个板块都有内容吗？导航能跳转吗？风格统一吗？

**常见的调整提示词：**

| 想改什么 | 跟 Claude Code 说 |
|---------|-----------------|
| 导航不清晰 | "导航栏加上点击后变色的效果，固定在页面最上方" |
| 头像位置不对 | "头像放在自我介绍左边，圆形显示" |
| 板块之间没分隔 | "每个板块之间加一条漂亮的分隔线" |
| 想加欢迎横幅 | "页面最上面加一个大横幅，写上'欢迎来到我的小天地'" |
| 字体不够可爱 | "整体字体再圆润可爱一点" |
| 想加背景 | "背景加上淡淡的渐变色" |
| 想加图标 | "每个板块标题前面加一个小图标（游泳加浪花、画画加画笔……）" |
| 想加动画 | "加入一些小动画，比如页面滚动时内容慢慢出现" |
| 底部信息 | "页面底部加一行字：'这是Yanny用AI制作的个人网站'" |

**每次只改一两处，改完刷新看看效果。**

### 内嵌图片

调整满意后，告诉 Claude Code：

> "请把所有图片内嵌到 HTML 文件中，让这个网站可以单独打开，不需要网络也能看"

---

## 第6步：毕业庆祝！（5分钟）

网站完成了！从头到尾浏览一遍，请爸爸妈妈也看看。满意了就保存好——这就是你的毕业作品！请爸爸妈妈拿出毕业证书，一起填写。

---

## 成品标准

- [ ] 首页有自我介绍和头像
- [ ] 至少3个内容板块（游泳世界、我的画作、钢琴角等）
- [ ] 第17课游泳时间线已整合到"游泳世界"
- [ ] 第19课画展已整合到"我的画作"
- [ ] 最上方有导航栏，点击可以跳转到对应板块
- [ ] 整体风格统一（颜色、字体、间距协调）
- [ ] 每个板块都有实际内容，不是空白
- [ ] 所有图片已内嵌，文件可以独立打开
- [ ] 网页可以在浏览器中正常打开和浏览

---
