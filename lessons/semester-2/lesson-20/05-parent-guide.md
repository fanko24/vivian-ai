# 家长指导手册：第20课 - Yanny的个人网站（毕业课）

## 课前准备

- 这是整个 AI 启蒙课程的毕业课，准备好庆祝！
- 提前回顾之前做过的所有项目（第17-19课）
- 准备一张纸让孩子画网站结构图
- 提前抄写或打印毕业证书，课上一起填写

## 课堂引导

### 规划环节
- "你想在网站里放哪些内容？"
- "如果别人来看你的网站，你最想让他们看到什么？"
- 让孩子自己画网站地图，家长帮忙补充

### 回顾成长
引导孩子回顾："还记得第一次用 Claude Code 的样子吗？你觉得现在的你有什么不一样？"帮孩子认识到自己的进步。

### 使用工具
- 用豆包帮写自我介绍和画头像，提醒孩子内容要真实
- Claude Code 项目较大，可能需要分步进行，鼓励孩子多尝试

## 重要对话：AI 与你

引导话术：
1. "你觉得 AI 帮你做了什么？"
2. "有没有什么是只有你能做、AI 做不了的？"
3. "游泳是你自己游的，画是你自己画的——这些才是最了不起的！"

核心信息：AI 是工具，想法是你的。真正厉害的是会用工具的人。

## 毕业庆祝

1. 网站完成后，和孩子一起浏览每个板块
2. 拿出毕业证书，一起填写
3. 让孩子大声读出证书内容
4. 拍一张照片留念

## 安全注意事项

1. 个人网站不要出现过多隐私信息
2. 分享时只发给熟悉的人

## 学习观察

- **规划能力**：能否系统地思考网站结构
- **整合能力**：能否把多个项目串联起来
- **自我认知**：能否区分 AI 做的和自己做的事情
- **成长意识**：能否感受到自己的进步
