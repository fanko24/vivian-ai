# 第 16 课：Yanny的个人网站

## 课程信息
- **模块**：模块四 - 用 AI 创造（学期二·毕业课）
- **时长**：60 分钟
- **难度**：⭐⭐⭐⭐
- **核心体验**：把所有作品整合成一个个人网站，庆祝成长

## 学习目标
1. 规划和设计一个多板块的个人网站
2. 把之前做的多个项目整合到一个网站中
3. 回顾成长——从 AI 新手到 AI 小创作者

## 准备工作
- [ ] 回顾之前做过的所有网页项目（第 13-15 课）
- [ ] Claude Code 已打开（claude.ai/code）
- [ ] 豆包网页已打开（doubao.com）
- [ ] 准备好一颗庆祝的心
- [ ] 家长陪伴

> 💡 **本课工具**：Claude Code（做网站）+ 豆包（写文字、做图片）

## 时间安排
| 环节 | 时间 | 内容 |
|------|------|------|
| 30秒魔法时刻 | 5 分钟 | 看一个完整的个人网站示例 |
| 规划网站结构 | 10 分钟 | 设计网站的板块和内容 |
| 动手实践 | 30 分钟 | 用 Claude Code 制作个人网站 |
| 整合与美化 | 10 分钟 | 把之前的项目整合进来 |
| 毕业庆祝 | 5 分钟 | 颁发毕业证书！ |

## 学习小贴士
1. 这是最后一课，也是最大的项目，慢慢来不着急
2. 回头看看你做过的一切，会发现自己进步了好多
3. 这个网站是属于你的，想放什么就放什么
