# 第二学期：原一的AI创作工作室

> 8节课，用AI把兴趣变成作品

---

## 学期概览

本学期通过8个项目，让孩子用AI工具围绕自己的真实兴趣进行创作。每个项目都产出一个可以分享的实际作品。

- **总课时**：8节
- **建议节奏**：每周1节
- **每节时长**：45-60分钟
- **适合年龄**：8岁左右

## 学习准备

### 工具准备
- [ ] 电脑或平板电脑
- [ ] 手机（用于语音交互和拍照）
- [ ] 网络连接
- [ ] 豆包App或网页版（doubao.com）已登录
- [ ] Claude Code已安装（第13-16课使用，需家长协助）

### 材料准备
- [ ] 游泳训练和比赛的照片、视频（第9、14课）
- [ ] 原一的画作拍照存档（第10、15课）
- [ ] 作文本或纸张
- [ ] 文件夹（保存作品）

---

## 学习建议

### 给家长
1. **提前准备素材** - 每课前按照准备清单整理好照片、视频等素材
2. **陪伴学习** - 建议全程陪伴，特别是Claude Code课程
3. **鼓励表达** - 引导孩子用自己的话描述，不强求完美
4. **记录成长** - 用照片或视频记录孩子的创作过程

### 给小朋友
1. **大胆创作** - 这是你的作品，想怎么做就怎么做
2. **告诉AI你想要什么** - 说得越清楚，AI帮你做得越好
3. **保存好作品** - 每次做完都要保存，后面会用到
4. **分享给家人** - 让大家看看你做的东西有多酷

## 课程特色

与第一学期不同，第二学期的重点是**做项目**：
- 每个课都产出一个完整的、可以分享的作品
- 作品全部围绕原一的真实兴趣（游泳、画画、钢琴、羽毛球）
- 前半学期用豆包做内容（画册/文档），后半学期用Claude Code做网页
- 最终整合为个人网站
