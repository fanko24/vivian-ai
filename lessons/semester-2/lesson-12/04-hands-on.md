# 动手实践：制作羽毛球小教练手册

## 30秒魔法时刻

在豆包 AI 绘画中输入：**"一个小女孩做羽毛球发球动作，卡通风格，动作分解图，白色背景"**

几秒钟就画出可爱的动作分解图，是不是很神奇？现在来做完整的教学手册吧！

## 第 1 步：列出技术清单

列出3-5个最重要的羽毛球技术：

| 序号 | 技术名称 | 一句话描述 |
|------|---------|-----------|
| 1 | 握拍 | "像和球拍握手一样拿住它" |
| 2 | | |
| 3 | | |

> 写不出来可以让豆包帮忙！输入："羽毛球初学者最需要学的基本技术有哪些？"

## 第 2 步：用豆包整理技术说明

先用自己的话说一遍，然后请豆包润色：

> "我是8岁的小朋友，正在写羽毛球教学手册。请帮我改写得更清楚好懂：[你的描述]"

**示例**：
- 原始："握拍就是拿拍子的方法，不能太紧也不能太松。"
- 润色后："握拍就像和球拍'交朋友'——不要太用力捏它，也不能太松让它掉下去。"

## 第 3 步：用 AI 绘画制作动作示意图

描述模板：
> "一个小女孩做羽毛球____动作，____（具体描述），卡通风格，白色背景"

参考描述：
- **握拍**："小女孩的手拿羽毛球拍，特写握拍姿势，卡通风格"
- **发球**："小女孩做发球动作，左手拿球右手举拍，侧面视角，分三步，卡通风格"
- **接球**："小女孩做回球动作，球拍向前挥出，卡通风格"
- **步伐**："小女孩在球场上移动，用箭头标注方向，俯视角度"

> 每张图可以多试几次，选最清楚的！

## 第 4 步：用 Seedance 生成演示视频（可选）

在豆包中找到视频生成功能：
1. 输入："一个小女孩在球场上做正手发球，从准备到挥拍，侧面视角"
2. 等待生成（1-2分钟）
3. 查看效果，不完美可以多试几次

## 第 5 步：整理成完整手册

手册结构：
```
《羽毛球小教练手册》 作者：刘原一
一、前言（为什么要学羽毛球）
二、基本技术（每个配说明+示意图）
三、小贴士（你自己的经验分享）
四、演示视频（如果有的话）
```

## 第 6 步：写"教练寄语"

用自己的话写一段话鼓励想学羽毛球的小朋友：
- 你学羽毛球最大的收获是什么？
- 有什么小窍门想分享？

### 完成清单

- [ ] 手册封面和标题
- [ ] 教练寄语

## 小贴士
1. 用自己的话先说，再让 AI 润色
2. 动作描述越具体，AI 画得越准确
3. 记住安全口诀！
