# 动手实践：Yanny的画画作品集

## 30秒魔法时刻

看看这本AI帮忙做的电子作品集是什么样的！（家长展示一个提前做好的示例）

今天Yanny也要把自己画的画整理成一本漂亮的作品集！

---

## 第1步：挑选和拍照（5分钟）

1. 从你的画作中选出5-8幅最喜欢的
2. 请家长帮忙把每幅画拍成清晰的照片
3. 把照片传到电脑或平板上

**拍照小技巧（家长帮忙操作）**：
- **光线**：在白天拍照最好，把画放在窗边有自然光的地方，但不要让阳光直接照在画上（容易反光）。如果是晚上拍，打开房间的灯，再用手机的手电筒从侧面照一下画，会更清楚
- **角度**：手机要 **正对着画**，不要歪着拍。可以站在画的正前方，手机和画保持平行。画可以平放在桌上拍，也可以靠墙立起来拍
- **避免反光**：如果画上有蜡笔或彩笔的光泽，容易反光。可以试试稍微调整手机的角度（往左或往右偏一点点），找到一个不反光的位置再按快门
- **手机设置**：打开手机相机，点击屏幕上的画面让焦点对准画（会出现一个黄色的小方框），等画面变清晰了再按拍照按钮。如果手机有 **"文档模式"**（有些手机叫"扫描"），可以用这个模式拍，会更清楚
- **检查照片**：拍完一张后放大看看，文字和线条都清楚才算拍好了。如果模糊了就重拍一张

**把照片传到电脑**：
- 用手机拍照后，通过微信"文件传输助手"发到电脑微信上
- 或者用数据线连接手机和电脑，把照片复制过来
- 把所有照片保存在桌面的一个文件夹里，文件夹名字叫"Yanny的画作照片"

**挑选小建议**：
- 选你最喜欢的画
- 可以选不同时间画的，能看出你的进步
- 也可以选不同类型的（人物、动物、风景……）

---

## 第2步：让豆包帮忙写介绍（10分钟）

**打开豆包**：
1. 在浏览器地址栏输入 **doubao.com**，按回车键打开豆包网页
2. 如果还没登录，点击页面右上角的 **"登录"** 按钮，用手机号或微信扫码登录
3. 登录成功后，点击页面左上角的 **"开始新对话"**（或聊天框上方的 **"+"** 按钮），打开一个全新的聊天窗口

**上传画作照片**：
1. 看到聊天输入框旁边有一个 **"+"** 号按钮或者 **📎（回形针）** 图标，点击它
2. 在弹出的菜单中选择 **"上传图片"**（或"选择文件"）
3. 在弹出的窗口中，找到桌面上"Yanny的画作照片"文件夹，选择一幅画的照片
4. 等照片显示在聊天框上方，说明上传成功了

**输入要求**：照片上传后，在聊天输入框里打上下面这段话，然后点击 **发送按钮**（或按 **回车键**）：

> "豆包，这是我画的一幅画，请你看看我画了什么，帮我写一段短短的介绍，就像画展上的那种说明文字。用小朋友能看懂的语言写。"

如果你记得当时画的是什么，可以多告诉豆包一些信息，这样介绍会更准确：

> "这幅画我画的是我家的小猫在阳台上晒太阳。我用了橙色和黄色，因为我觉得很温暖。请帮我写一段介绍。"

**期待的结果**：豆包会回复一段关于这幅画的介绍文字。

**检查和修改**：读一读豆包写的介绍，看看说得对不对。如果不对，在聊天框里告诉豆包：
> "不对哦，这幅画画的不是小狗，是小猫。请重新写一下。"

**复制保存介绍**：
1. 用鼠标选中豆包写的介绍文字（从第一个字拖到最后一个字，文字变蓝）
2. 点击鼠标右键，选择 **"复制"**（或按 **Ctrl+C** / **Command+C**）
3. 打开记事本或文档，点击右键选择 **"粘贴"**（或按 **Ctrl+V** / **Command+V**）
4. 在文字前面标注是哪幅画的介绍，比如写上"画1的介绍："
5. 每幅画都重复以上步骤：上传照片 → 让豆包写介绍 → 复制保存

---

## 第3步：用豆包画封面（5分钟）

给你的作品集设计一张漂亮的封面！

**在豆包里画封面图**：
1. 还是在豆包的聊天页面里（也可以新建一个对话）
2. 在聊天输入框里，输入封面图的描述，然后点击 **发送按钮**
3. 等待大约 **10-30秒**，豆包会生成封面图片

你可以这样输入：

> "请帮我画一张作品集的封面。画上有一个快乐的小女孩在画画，旁边散落着彩色的画笔和颜料，背景是五颜六色的彩虹，上面写着'Yanny的画画世界'。风格要可爱温馨。"

**等待生成**：豆包会显示"正在生成中……"，稍等一下图片就会出现。

**如果不满意**：在聊天框里告诉豆包哪里要改，比如：
> "请把彩虹画得更大一点"
> "请把小女孩画得更可爱"
豆包会重新画一张。

**保存封面图片**：
1. 看到满意的封面图后，用鼠标 **右键点击** 图片
2. 在弹出的菜单中选择 **"图片另存为……"**
3. 在弹出的保存窗口中，选择保存到 **桌面**
4. 把文件名改成 **"作品集封面"**
5. 点击 **"保存"** 按钮

---

## 第4步：组装作品集（5分钟）

现在把所有素材组装成电子作品集！

**打开软件**：
1. 在电脑上找到 **WPS演示** 或 **PowerPoint**（PPT）的图标，双击打开
2. 如果看到"新建"页面，选择一个 **空白演示文稿**，点击创建

**做封面（第1页）**：
1. 点击幻灯片中间写着"单击此处添加标题"的地方，输入作品集的名字，比如 **"Yanny的画画世界"**
2. 点击下方"单击此处添加副标题"的地方，输入今天的日期，比如 **"2026年4月"**
3. 插入封面图：点击顶部菜单栏的 **"插入" → "图片"**，在弹出的窗口中找到桌面上的"作品集封面"图片，点击 **"插入"**
4. 用鼠标拖动图片到合适的位置，拖动图片角上的小圆点调整大小

**做作品页（第2页起）**：
1. 点击左侧幻灯片缩略图区域，再点击顶部菜单的 **"新建幻灯片"**（或"插入"→"新幻灯片"），选择一个有标题和内容的版式
2. 在标题栏输入这幅画的名字，比如 **"阳光下的猫咪"**、**"我的家人"**
3. 插入画作照片：点击 **"插入" → "图片"**，选择对应的画作照片插入
4. 插入介绍文字：点击 **"插入" → "文本框" → "横向文本框"**，在页面上拖出一个方框，把之前保存的介绍文字粘贴进去
5. 重复以上步骤，每幅画做一个页面

**调整排版**：
- 推荐排版方式一：**左边放画的照片，右边放介绍文字**
- 推荐排版方式二：**上面放画，下面放介绍**
- 用鼠标点击文字，在顶部工具栏把字号改成 **28号或32号**，字大一点更好看
- 点击文字后可以在顶部选择字体，推荐用 **"微软雅黑"** 或 **"楷体"**
- 用鼠标拖动图片和文字框调整位置，让每页看起来整齐美观

**别忘了保存**：点击左上角的 **"文件" → "保存"**（或按 **Ctrl+S** / **Command+S**），文件名叫"Yanny的画画世界"，保存到桌面。

## 任务清单

- [ ] 选出了5-8幅最喜欢的画
- [ ] 每幅画都拍了清晰的照片
- [ ] 豆包帮忙写了每幅画的介绍
- [ ] 用豆包画了一张封面
- [ ] 组装成了完整的电子作品集

---

> 记住安全口诀：不告诉AI真实姓名和家庭住址哦！

---

## 成品标准

完成的作品应该包含：

- [ ] 封面页（标题"Yanny的画画世界" + 日期 + AI画的封面图）
- [ ] 5-8页画作页（每页一幅画）
- [ ] 每页都有画作照片和介绍文字
- [ ] 每页都有小标题（画的名字）
- [ ] 文字用小朋友能看懂的语言
- [ ] 排版整齐，字号够大（28号以上）

---

## 分享你的作品

1. 把画册导出为 PDF：在 PPT 或 WPS 中点击"文件" → "导出" → 选择 PDF 格式
2. 也可以导出为图片：每页另存为图片，挑几张好看的发微信
3. 把文件通过微信发给奶奶、姥姥等家人，手机上直接打开就能看！

> 也可以用彩色打印机打印出来，做成纸质版送给家人！

做完以后翻翻看，这可是属于你的第一本作品集呢！
