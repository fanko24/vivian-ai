# 第12课：Yanny的画画作品集

## 课程信息
- **模块**：第二学期 - Yanny的AI创作工作室
- **时长**：45 分钟
- **难度**：⭐⭐
- **核心体验**：用AI帮自己的画作写介绍，制作电子作品集

## 学习目标
1. 学会观察和描述自己的画作
2. 学会用豆包帮忙写作品介绍
3. 学会把多幅作品整理成一本作品集

## 准备工作
- [ ] 平板电脑或电脑（已安装豆包或能访问 doubao.com）
- [ ] 豆包账号已登录
- [ ] Yanny的5-8幅画作提前拍照（家长协助）
- [ ] 画作照片已导入到电脑/平板中
- [ ] PPT软件或WPS已安装
- [ ] 家长陪伴

## 时间安排

| 环节 | 时间 | 内容 |
|------|------|------|
| 30秒魔法时刻 | 1 分钟 | 看看AI帮忙做的作品集是什么样的 |
| 故事开场 | 5 分钟 | 薇薇安整理画画作品的故事 |
| 知识讲解 | 8 分钟 | 怎么欣赏和描述自己的画 |
| 动手实践 | 25 分钟 | 和豆包一起做作品集 |
| 分享展示 | 4 分钟 | 展示自己的作品集 |
| 课后整理 | 2 分钟 | 保存和完善 |

> 本课工具：**豆包** (doubao.com) - 免费AI助手，支持对话和AI绘画
