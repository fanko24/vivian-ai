# 动手实践：制作原一的专属日历

## 30秒魔法时刻

家长打开 Claude Code，在屏幕上输入：

> "帮我做一个可爱的日历网页，有12个月，粉色主题。"

几秒钟后，一个漂亮的日历出现在屏幕上！

"看！这就是用说话让电脑帮你做东西！"

---

## 第一步：列出我的每周活动（5分钟）

拿出一张纸，写下原一每周的活动：

- 🏫 学校上课（周一到周五）
- 🏊 游泳训练（周___ 、周___）
- 🎹 钢琴练习（周___）
- 🏸 羽毛球（周___）
- 🎨 画画课（周___）

把每项活动选一个代表颜色：
- 游泳 = 蓝色
- 钢琴 = 粉色
- 羽毛球 = 绿色
- 画画 = 黄色
- 学校 = 橙色

---

## 第二步：用豆包生成可爱小图标（5分钟）

1. 打开豆包（doubao.com）
2. 输入："帮我画一个可爱的游泳小图标，蓝色，卡通风格"
3. 分别生成每个活动的图标
4. 保存到电脑上

---

## 第三步：告诉 Claude Code 做日历（15分钟）

家长在 Claude Code 中输入原一的需求：

> "帮我做一个可爱的日历网页，有12个月。每个月标记上我的活动：
> 游泳用蓝色，钢琴用粉色，羽毛球用绿色，画画用黄色。
> 标题写'原一的专属日历'，风格要温馨有趣。
> 我生日那天要特别标记出来！"

等 Claude Code 生成日历后，在浏览器中查看效果。

---

## 第四步：调整和优化（15分钟）

看看日历，告诉家长你想改什么：

可能的调整建议：
- "标题再大一点！"
- "每个日期格子里放上活动的小图标"
- "我生日那天画一个大蛋糕"
- "整体颜色再粉一点"
- "月份之间的间距再大一些"

家长把这些调整告诉 Claude Code，每次改一点点，看看效果。

---

## 第五步：保存和打印（5分钟）

1. 满意后，把日历网页保存为 PDF
2. 用彩色打印机打印出来
3. 贴在原一房间的墙上！

记住安全口诀：不随便下载陌生文件哦！
