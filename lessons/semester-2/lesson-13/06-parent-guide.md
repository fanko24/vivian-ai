# 家长指南：第13课

## 重要说明

这是原一第一次接触 Claude Code。

**核心原则：孩子只负责"说"，家长负责"操作"。**

原一不需要写代码，也不需要碰键盘。她只需要描述自己想要的日历长什么样，家长负责把她的需求输入到 Claude Code 中。

---

## 课前准备（提前一天完成）

### 安装 Claude Code
1. 确认电脑已安装 Node.js
2. 在终端运行：`npm install -g @anthropic-ai/claude-code`
3. 测试是否正常运行：打开终端，输入 `claude`，确认可以启动

### 准备材料
- 确认原一的每周活动时间表
- 确认原一的生日日期
- 准备好打印机（或知道如何保存为PDF）

### 测试流程
- 试着用 Claude Code 生成一个简单网页，确认流程顺畅
- 这样上课时不会因为技术问题打断孩子的体验

---

## 课堂中的角色

| 步骤 | 家长做什么 | 孩子做什么 |
|------|-----------|-----------|
| 魔法时刻 | 打开终端，输入演示命令 | 看效果，发出惊叹 |
| 列活动 | 帮忙记录 | 说出自己的活动安排 |
| 生成图标 | 协助操作豆包 | 描述想要的图标 |
| 做日历 | 在Claude Code中输入孩子的话 | 描述想要的日历 |
| 调整 | 把调整需求输入Claude Code | 说出想改的地方 |

---

## 引导技巧

1. **鼓励具体描述**：当孩子说"好看一点"时，引导她说"什么颜色""多大""放在哪里"
2. **允许试错**：第一次做出来的不完美很正常，这正是学习"迭代"的好机会
3. **及时赞美**："你描述得太清楚了！""Claude Code 听懂了你的话！"
4. **控制时间**：调整环节不要超过15分钟，避免孩子疲劳

---

## 常见问题

**Q：Claude Code 生成速度慢怎么办？**
A：先跟孩子解释"魔法师在认真工作"，同时准备好备用活动。

**Q：孩子不知道怎么描述怎么办？**
A：给她看一些日历的图片作为参考，问她"你喜欢哪个？为什么？"
