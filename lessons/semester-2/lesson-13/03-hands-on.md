# 动手实践：制作我的钢琴专辑

## 30秒魔法时刻

在豆包中输入：**"帮我写一段关于月光下弹钢琴的歌词，温柔安静的风格"**

看着 AI 几秒钟就写出优美的歌词，是不是很神奇？现在来制作完整的专辑吧！

## 第 1 步：选一首最爱的钢琴曲

先在心里想一想你最喜欢弹的钢琴曲，然后把答案填在下面的表格里：

| 项目 | 你的回答 |
|------|---------|
| 曲子名字 | |
| 喜欢的原因 | |
| 弹的时候心情 | |
| 像什么画面 |

填好之后，把你的回答告诉豆包，让它帮你更好地理解这首曲子：

1. 打开浏览器，在地址栏输入 **doubao.com**，按回车
2. 点击页面中间的输入框（光标会闪动的那个框）
3. 输入下面这段话（把横线换成你自己的答案）：

> "我叫____，今年8岁，正在学钢琴。我最喜欢的曲子是《____》，因为____。弹这首曲子的时候我心情____，我想到的画面是____。你能帮我更详细地描述一下这首曲子的感觉吗？"

4. 按回车键或点击"发送"按钮（蓝色箭头）
5. 等几秒钟，豆包就会回答你，认真读一读它的描述
6. 如果觉得说得好，**用鼠标选中文字 → 右键 → 复制**，保存到备忘录里备用

## 第 2 步：用豆包创作歌词或介绍

接下来我们正式请豆包帮忙写内容。

### 操作步骤

1. 如果刚才的对话还在，**继续在同个对话框里**操作；如果关了就重新打开 doubao.com
2. 在输入框里输入提示词（二选一）：

**写歌词**：
> "我是一名8岁的小女孩，正在学钢琴。我最喜欢的曲子是《____》，给我一种____的感觉。请帮我把这种感觉写成优美的歌词，要有主歌和副歌。"

**写介绍**：
> "帮我为钢琴曲《____》写一段100字左右的介绍，描述给人的感觉，用小朋友能听懂的语言。"

3. 点击"发送"按钮，等待豆包回复（大约5-10秒）
4. 仔细读一遍豆包写的内容：
   - 如果喜欢 → 选中文字 → 右键 → "复制"
   - 如果想改 → 输入"请再改一改，我希望更____一些"（比如更活泼、更温柔）
   - 如果不满意 → 点击输入框上面的"新建对话"，重新开始
5. 打开电脑上的**备忘录**或**WPS/Word文档**，粘贴保存好

> 小技巧：告诉豆包越多细节，它写得就越好！比如"像星星闪烁一样"、"像春天的小溪"这样的描述。

## 第 3 步：用 AI 创作歌曲（可选）

如果你想让歌词变成真正能听的歌曲，可以用 Suno 来做！

### 注册和登录

1. 打开浏览器，在地址栏输入 **suno.com**，按回车
2. 第一次用需要注册：
   - 点击页面上的"Sign Up"（注册）按钮
   - 可以选择"使用 Google 账号登录"或"使用微软账号登录"
   - 如果没有账号，请爸爸妈妈帮你注册一个
3. 登录成功后，你会看到主页面

### 创作歌曲

1. 点击页面左侧的 **"Create"**（创建）按钮
2. 你会看到一个编辑页面，上面有输入框
3. 找到"Lyrics"（歌词）输入框，把第2步豆包帮你写的歌词**粘贴**进去：
   - 在输入框里点击一下 → 右键 → "粘贴"
4. 找到"Style of Music"（音乐风格）输入框，输入风格关键词：
   - 钢琴安静曲 → 输入 **"Lullaby, Piano, Gentle"**
   - 活泼欢快曲 → 输入 **"Pop, Happy, Piano"**
   - 温柔浪漫曲 → 输入 **"Classical, Soft, Romantic"**
5. 给歌曲起个名字，填在"Title"（标题）框里
6. 点击 **"Create"**（生成）按钮
7. 等待大约30秒到1分钟，会生成两首歌曲供你选择
8. 点击播放按钮 ▶️ 听一听效果
9. 选一首你喜欢的，点击 **下载按钮**（⬇️图标）保存到电脑

> 没有 Suno 也没关系！可以让豆包把歌词改编成一首小诗，一样很棒。

## 第 4 步：用豆包 AI 绘画设计封面

现在来画一张美美的专辑封面！

### 操作步骤

1. 打开 **doubao.com**，确保你在首页
2. 找到输入框下方的一排功能按钮，点击 **"AI绘画"** 图标（画笔/调色盘样子的图标）
   - 如果在首页没看到，就点击左侧菜单栏里的"AI绘画"
3. 在绘画描述框里输入你想画的画面，参考以下模板（选一个改一改）：

- "一个小女孩坐在钢琴前，窗外是满天星星的夜空，月光照在琴键上，梦幻水彩风格"
- "一个小女孩在花园里弹钢琴，周围飞舞着彩色的蝴蝶，童话插画风格"
- "一个女孩坐在海边的大钢琴前，夕阳映在海面上，温暖治愈风格"

4. 选择画面比例：点击"横图"或"竖图"（专辑封面推荐选**正方形**）
5. 点击 **"开始绘画"** 或 **"生成"** 按钮
6. 等待大约10-20秒，豆包会生成几张图片
7. 浏览生成的图片，点击你最喜欢的那一张放大看
8. 满意的话，点击图片下方的 **"下载"** 或 **"保存"** 按钮，保存到电脑
9. 如果不满意，换一个描述词重新生成（可以多试几次！）

> 小提示：描述词越具体，画出来的效果越好。可以说清楚颜色、光线、风格。

## 第 5 步：写专辑寄语

用你自己的话写一段"专辑寄语"（50-100字），可以回答这些问题：
- 你为什么喜欢这首曲子？
- 弹这首曲子的时候你想到什么？
- 你想把这张专辑送给谁？

写好以后，也可以请豆包帮你润色一下：
1. 在豆包对话框里输入：**"请帮我润色这段专辑寄语，让它读起来更温馨：[粘贴你写的寄语]"**
2. 对比一下原文和润色后的版本，选你更喜欢的那一版
3. 复制保存到文档里

### 参考格式

```
《专辑名字》 演奏者：Yanny

寄语：这是我学钢琴以来最喜欢的一首曲子……

封面：[AI绘制的图片]

曲目：1. 《曲子名》— [一段介绍]
      2. 《曲子名》— [一段介绍]
```

## 第 6 步：整理成完整专辑

现在把所有内容整理到一起！

### 操作步骤

1. 打开 **WPS** 或 **Word** 文档软件
2. 新建一个空白文档
3. 按照下面顺序排版：
   - 第1页：**封面页** — 插入AI画的封面图（点击"插入" → "图片" → 选刚才保存的图片），写上专辑名和你的名字
   - 第2页：**曲目介绍** — 粘贴曲目名称和介绍文字
   - 第3页：**歌词或寄语** — 粘贴歌词和你的寄语
4. 调整字体大小和颜色让页面好看（标题可以用大一点、彩色字体）
5. 点击 **"文件" → "保存"**，给文件起个名字，比如"我的钢琴专辑"
6. 如果想发给别人，点击 **"文件" → "另存为" → 选择PDF格式"**，这样谁都能打开看

> 可以请爸爸妈妈帮你排版，一起把专辑做漂亮！

---

## 成品标准

完成的作品应该包含：

- [ ] 一张专辑封面图（用AI绘画生成的图片）
- [ ] 一个好听的专辑名称
- [ ] 至少一首曲目的介绍（用豆包写的）
- [ ] 歌词或专辑寄语（自己写的或和豆包一起润色的）
- [ ] 音乐文件（如果用了Suno，把生成的音频文件也保存好）

---

## 分享你的作品

1. 把专辑内容整理成文档，导出为 PDF
2. 如果用 Suno 创作了音乐，把音乐文件一起打包
3. 把文件通过微信发给奶奶、姥姥等家人
4. 家人在手机上打开就能看到封面、读到介绍，还能听音乐！

> 也可以用彩色打印机打印封面和介绍，做成实体专辑卡！

## 小贴士
1. 不要害怕表达真实感受
2. 封面可以多试几次，选最喜欢的
3. 专辑名字可以很有创意
4. 记住安全口诀！
