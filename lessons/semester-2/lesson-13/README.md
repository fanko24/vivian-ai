# 第13课：我的钢琴练习日记

## 课程信息
- **模块**：第二学期 - Yanny的AI创作工作室
- **时长**：50 分钟
- **难度**：⭐⭐
- **核心体验**：用真实练习视频和照片，做一本钢琴成长日记

## 学习目标
1. 学会从视频中截取精彩画面做素材
2. 学会用口述的方式让AI帮忙整理感受成文字
3. 学会把视频截图和文字组合成电子日记

## 准备工作
- [ ] 平板电脑或电脑（已安装豆包或能访问 doubao.com）
- [ ] 豆包账号已登录
- [ ] **提前准备3-5段钢琴练习或表演的视频**
- [ ] **准备几张钢琴练习的照片**（如果有）
- [ ] 准备好Word或WPS文档软件
- [ ] 家长陪伴

## 时间安排

| 环节 | 时间 | 内容 |
|------|------|------|
| 30秒魔法时刻 | 1 分钟 | 看看用真实视频截图做的钢琴日记 |
| 故事开场 | 5 分钟 | 薇薇安的钢琴日记故事 |
| 知识讲解 | 8 分钟 | 怎么用视频截图和AI做日记 |
| 动手实践 | 25 分钟 | 和豆包一起做钢琴练习日记 |
| 分享展示 | 4 分钟 | 看看自己做的日记 |
| 课后整理 | 2 分钟 | 整理和保存作品 |

> 本课工具：**豆包** (doubao.com) - 免费AI助手，能帮你整理口述内容
