# 第13课：Yanny的专属日历

**用时**：50分钟
**工具**：Claude Code + 豆包（doubao.com）
**难度**：⭐⭐⭐

---

## 学习目标

1. 第一次认识 Claude Code —— 理解"用说话的方式编程"
2. 学会用自然语言描述网页的样子
3. 制作一个实用又漂亮的个性化日历

## 准备清单

- [ ] 电脑已安装 Claude Code（家长提前完成）
- [ ] 打开豆包网页（doubao.com）
- [ ] 准备好Yanny的活动清单（游泳、钢琴、羽毛球、画画、学校）
- [ ] 打印机就绪（或准备好保存为PDF）
