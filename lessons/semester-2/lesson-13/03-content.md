# 知识小课堂

## 1. 什么是 Claude Code？

Claude Code 是一个 AI 编程助手。

它有一个超级本领：**你用中文告诉它你想要什么，它就帮你写代码！**

打个比方：
- 你 = 建筑设计师（说想法）
- Claude Code = 建筑工人（把想法变成现实）

你不需要学写代码，只需要学会**清楚地描述你想要什么**。

---

## 2. 如何清楚描述需求？

告诉 Claude Code 你的想法时，要包含这四个部分：

| 部分 | 例子 |
|------|------|
| **做什么** | 做一个日历 |
| **长什么样** | 可爱风格，粉色和蓝色为主 |
| **有什么内容** | 12个月，标记我的活动 |
| **特别要求** | 游泳用蓝色，钢琴用粉色 |

秘诀：越具体，Claude Code 做出来的东西就越接近你想要的！

---

## 3. 描述 → 看结果 → 调整

和 Claude Code 合作有三个步骤：

```
第一步：描述你想要什么
   ↓
第二步：看它做出来的效果
   ↓
第三步：说出要调整的地方
   ↓
（重复，直到满意！）
```

这就像画画一样——先画个草图，看看哪里需要改，再慢慢修改到满意为止。

不用一次说完美，每次调整一点点，结果就会越来越好！
