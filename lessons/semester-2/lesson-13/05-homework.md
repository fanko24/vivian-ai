# 课后作业

## 必做作业：日历上墙（10分钟）

把你做好的专属日历打印出来，贴在房间墙上。

用彩色笔在这周的格子里：
- 标记出这周的活动（游泳、钢琴、羽毛球、画画）
- 在已完成的活动中打一个勾 ✓
- 在今天的位置画一颗小星星 ⭐

完成后拍一张照片给爸爸妈妈看！

## 选做作业：设计生日特别页（15分钟）

你的生日是哪一天？用 Claude Code 为你的生日月份设计一个特别版本：

要求：
- 生日那天的格子要特别大、特别醒目
- 可以加上气球、蛋糕等装饰
- 写上一句给自己生日的话

完成后保存为单独的一页，可以打印出来当生日装饰！
