# 第14课：羽毛球训练手册

## 课程信息
- **模块**：第二学期 - Yanny的AI创作工作室
- **时长**：50 分钟
- **难度**：⭐⭐⭐
- **核心体验**：用真实训练照片和AI标注，做一本羽毛球教学手册

## 学习目标
1. 学会把技能分解成简单的教学步骤
2. 学会在真实照片上用AI辅助标注动作要领
3. 学会把照片和文字组合成教学手册

## 准备工作
- [ ] 平板电脑或电脑（已安装豆包或能访问 doubao.com）
- [ ] 豆包账号已登录
- [ ] **提前选好5-8张羽毛球训练的照片或视频截图**
- [ ] 准备好Word或WPS文档软件
- [ ] 家长陪伴

## 时间安排

| 环节 | 时间 | 内容 |
|------|------|------|
| 30秒魔法时刻 | 1 分钟 | 看看用真实照片做的训练手册 |
| 故事开场 | 5 分钟 | 薇薇安当教练 |
| 知识讲解 | 8 分钟 | 怎么标注照片做教学手册 |
| 动手实践 | 25 分钟 | 和豆包一起做训练手册 |
| 分享展示 | 4 分钟 | 看看自己做好的手册 |
| 课后整理 | 2 分钟 | 整理和保存作品 |

> 本课工具：**豆包** (doubao.com) - 免费AI助手，能看懂照片并帮你写要点
