# 动手实践：制作羽毛球小教练手册

## 30秒魔法时刻

在豆包 AI 绘画中输入：**"一个小女孩做羽毛球发球动作，卡通风格，动作分解图，白色背景"**

几秒钟就画出可爱的动作分解图，是不是很神奇？现在来做完整的教学手册吧！

## 第 1 步：列出技术清单

先想想你学羽毛球时学了哪些动作，试着填在表格里：

| 序号 | 技术名称 | 一句话描述 |
|------|---------|-----------|
| 1 | 握拍 | "像和球拍握手一样拿住它" |
| 2 | | |
| 3 | | |

如果不太确定有哪些技术，或者想不出来，可以请豆包帮忙：

1. 打开浏览器，在地址栏输入 **doubao.com**，按回车
2. 点击页面中间的输入框
3. 输入这段话：

> "我是8岁的小朋友，正在学羽毛球，想写一本小教练手册。请告诉我羽毛球初学者最需要学的5个基本技术，每个技术用一句话简单说明。"

4. 点击"发送"按钮，等几秒钟看豆包的回答
5. 从豆包列出的技术里，选出你最熟悉的3-5个填到表格里
6. 把豆包的回答**复制保存**到备忘录里，后面会用到

## 第 2 步：用豆包整理技术说明

现在把每个技术写得更详细、更有趣。

### 先自己写一遍

对着表格里的每个技术，用自己的话说一说：
- 这个动作怎么做？（手怎么放、脚怎么站）
- 有什么小窍门？
- 新手容易犯什么错？

### 再请豆包帮你润色

1. 打开 **doubao.com**，在输入框里输入：

> "我是8岁的小朋友，正在写羽毛球教学手册。请帮我改写得更清楚好懂：[粘贴你自己写的描述]"

2. 点击"发送"，等豆包回复
3. 对比豆包润色前后的文字，选你更喜欢的版本
4. 每个技术都重复一遍，**复制保存**到文档里

**示例**：
- 原始："握拍就是拿拍子的方法，不能太紧也不能太松。"
- 润色后："握拍就像和球拍'交朋友'——不要太用力捏它，也不能太松让它掉下去。"

> 小技巧：你可以告诉豆包"请用比喻来说明"，这样写出来的文字更有趣！

## 第 3 步：用 AI 绘画制作动作示意图

给每个技术配上一张可爱的示意图吧！

### 操作步骤

1. 打开 **doubao.com**
2. 找到输入框下方的一排功能按钮，点击 **"AI绘画"** 图标（画笔/调色盘样子的图标）
   - 如果在首页没看到，就点击左侧菜单栏里的"AI绘画"
3. 在绘画描述框里，用下面的模板来写描述：

> "一个小女孩做羽毛球____动作，____（具体描述动作细节），卡通风格，白色背景"

4. 参考下面的示例，选择你要画的技术，把描述粘贴进去：

- **握拍**："小女孩的手拿羽毛球拍，特写握拍姿势，卡通风格，白色背景"
- **发球**："小女孩做发球动作，左手拿球右手举拍，侧面视角，分三步，卡通风格"
- **接球**："小女孩做回球动作，球拍向前挥出，卡通风格，白色背景"
- **步伐**："小女孩在球场上移动，用箭头标注方向，俯视角度，卡通风格"

5. 点击 **"开始绘画"** 或 **"生成"** 按钮
6. 等待大约10-20秒，豆包会生成几张图片
7. 浏览生成的图片，选最清楚、动作最好看的一张
8. 点击你选中的图片，然后点击 **"下载"** 或 **"保存"** 按钮，保存到电脑
9. 给每个技术都画一张，每张图用"技术名称"命名保存（比如"握拍示意图.png"）

> 每张图可以多试几次，换不同的描述词，选最清楚的那张！

## 第 4 步：用 Seedance 生成演示视频（可选）

如果你有手机上的豆包App，可以用视频生成功能做一个演示动画！

### 找到视频生成功能

1. 打开手机上的 **豆包App**（如果没装，请爸爸妈妈在应用商店搜索"豆包"下载）
2. 在首页向下滑动，找到 **"AI视频"** 或 **"视频生成"** 入口
   - 也可能在底部菜单栏的"发现"或"更多"里面
3. 如果看到 **"Seedance"** 字样，点击进入

### 生成演示视频

1. 在描述框里输入你想要的视频内容：

> "一个小女孩在球场上做正手发球，从准备到挥拍，侧面视角，动作清晰"

2. 选择视频风格（如果有选项）：选"写实"或"卡通"
3. 点击 **"生成"** 按钮
4. 等待1-2分钟，视频会自动生成
5. 点击播放按钮 ▶️ 看看效果
6. 如果满意，点击 **"保存到相册"** 或 **"分享"** 发给自己
7. 如果不满意，换一个描述词再试一次

> 这个功能是可选的，没有做也没关系，手册里有图片就够啦！

## 第 5 步：整理成完整手册

现在把所有内容整理成一本漂亮的手册！

### 操作步骤

1. 打开 **WPS** 或 **Word** 文档软件
2. 新建一个空白文档
3. 按照下面的结构来排版：

```
第1页 —— 封面
┌─────────────────────────┐
│   [AI绘制的封面图]        │
│                          │
│   《羽毛球小教练手册》     │
│      作者：Yanny          │
└─────────────────────────┘

第2页 —— 前言
  为什么要学羽毛球？
  （用几句话说说你的感受）

第3页起 —— 基本技术
  每个技术占一页：
  - 技术名称（大字标题）
  - 文字说明（豆包润色过的）
  - 动作示意图（AI画的图片）

最后一页 —— 小贴士 + 教练寄语
```

4. 插入图片的方法：点击菜单栏的 **"插入" → "图片"** → 选择你保存的AI绘画图片
5. 调整图片大小：点击图片，拖动角落的小圆点来缩放
6. 给标题选一个醒目的字体和颜色
7. 点击 **"文件" → "保存"**，文件名写成"羽毛球小教练手册"

## 第 6 步：写"教练寄语"

在手册最后一页，用你自己的话写一段话鼓励想学羽毛球的小朋友：
- 你学羽毛球最大的收获是什么？
- 有什么小窍门想分享？
- 你想对新学羽毛球的小朋友说什么？

写好以后也可以请豆包帮你润色：
1. 在豆包里输入：**"请帮我润色这段教练寄语，让它读起来更鼓励人：[粘贴你写的寄语]"**
2. 选你最喜欢的版本，复制粘贴到手册的最后一页

## 成品标准

完成的作品应该包含：

- [ ] 手册封面（有标题、你的名字、AI绘制的封面图）
- [ ] 至少3个羽毛球技术的文字说明（自己写的 + 豆包润色）
- [ ] 每个技术配一张AI绘制的动作示意图（至少3张）
- [ ] 教练寄语（鼓励其他小朋友的一段话）
- [ ] 整理成完整的文档或PDF文件

## 分享你的作品

1. 把手册导出为 PDF：在 WPS 或 Word 中点击"文件" → "另存为" → 选择 PDF 格式
2. 把 PDF 文件通过微信发给奶奶、姥姥等家人
3. 家人在手机上直接打开就能看！

> 也可以用彩色打印机打印出来，做成纸质版送给家人！

---

## 小贴士
1. 用自己的话先说，再让 AI 润色
2. 动作描述越具体，AI 画得越准确
3. 记住安全口诀！
