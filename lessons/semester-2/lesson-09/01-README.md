# 第9课：我的游泳日记

## 课程信息
- **模块**：第二学期 - 原一的AI创作工作室
- **时长**：45 分钟
- **难度**：⭐⭐
- **核心体验**：用豆包把游泳经历变成漂亮的电子画册

## 学习目标
1. 学会用豆包把口语描述整理成日记文字
2. 学会用文字描述画面，让AI帮忙画插图
3. 学会把文字和图片组合成电子画册

## 准备工作
- [ ] 平板电脑或电脑（已安装豆包或能访问 doubao.com）
- [ ] 豆包账号已登录
- [ ] 手机里的游泳照片和视频准备好（方便回忆）
- [ ] 准备好PPT软件或WPS（家长提前安装好）
- [ ] 家长陪伴

## 时间安排

| 环节 | 时间 | 内容 |
|------|------|------|
| 30秒魔法时刻 | 1 分钟 | 看看AI生成的游泳日记画册 |
| 故事开场 | 5 分钟 | 薇薇安的游泳故事 |
| 知识讲解 | 8 分钟 | AI怎么帮忙写日记和画画 |
| 动手实践 | 25 分钟 | 和豆包一起做游泳日记 |
| 分享展示 | 4 分钟 | 看看自己做的画册 |
| 课后整理 | 2 分钟 | 整理和保存作品 |

> 本课工具：**豆包** (doubao.com) - 免费AI助手，支持对话和AI绘画

---

让我们一起把游泳的故事变成漂亮的电子画册吧！
