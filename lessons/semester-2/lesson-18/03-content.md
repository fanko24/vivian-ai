# 知识小课堂：多版块网页设计

## 1. 网页可以展示多媒体

有的网页能播放音乐、看视频、还能滑来滑去看照片——这就是**多媒体**！

| 多媒体类型 | 网页里怎么展示 |
|-----------|--------------|
| 🎵 音乐 | 音乐播放器，点一下就播放 |
| 📸 照片 | 照片画廊，可以一张张浏览 |
| 🎬 视频 | 嵌入视频播放窗口 |
| ✨ 动画 | 图片和文字可以动起来 |

这节课，我们要在一个网页里展示两种才艺——钢琴和羽毛球！

---

## 2. 用 Claude Code 描述多版块布局

做多个版块的网页，就像装修一个有多个房间的房子。先想好哪里是"钢琴室"，哪里是"运动场"。

**跟 Claude Code 说话的秘诀：**

| 要说清楚 | 举例 |
|---------|------|
| 分几个版块 | "分成钢琴区和运动区两个版块" |
| 每个版块的风格 | "钢琴区优雅黑金色，运动区活力蓝绿色" |
| 每个版块放什么 | "每个区有介绍文字+照片画廊" |
| 怎么导航 | "最上面有导航栏，点击跳到对应版块" |

---

## 3. 配色和风格选择

不同的主题适合不同的颜色和风格：

| 主题 | 适合的配色 | 感觉 |
|------|-----------|------|
| 🎹 钢琴 | 黑白 + 金色 | 优雅、经典 |
| 🏸 羽毛球 | 蓝色 + 绿色 | 活力、运动 |
| 🌸 春天 | 粉色 + 绿色 | 清新、可爱 |
| 🎨 画画 | 彩色、鲜艳 | 活泼、创意 |

**小秘诀**：虽然两个版块风格不同，但整体网页要协调。可以用相同的字体和类似的布局，只是颜色不同。

---

## 4. 真实照片和视频让网页更有说服力

在网页上展示自己的才艺时，真实照片和视频比AI生成的图更有说服力：

- 真实照片证明"我真的会"
- 视频能展示动态的才艺（弹琴、打球）
- 别人看了会更相信你的本领

记住安全口诀：在网上分享内容时，不要放太多个人信息哦！
