# 家长指导：第18课 - Yanny的才艺展示网页

## 课前准备（提前一天完成）

### 收集材料

1. **钢琴照片和视频**：在手机里找3-5张照片 + 1-2段视频
   - 日常练琴的照片
   - 表演或汇报演出的照片/视频
   - 把文件传到电脑上，放到一个文件夹里

2. **羽毛球照片和视频**：找3-5张照片 + 1-2段视频
   - 训练动作照片
   - 训练或比赛视频
   - 把文件传到电脑上，放到同一个文件夹

3. **学琴/学球记录**：整理学习时间和进展

### 环境确认

- Claude Code 已安装，终端能正常启动
- 建一个新的文件夹，比如 `才艺网页`，把所有素材放进去
- 在这个文件夹里打开终端，启动 Claude Code

## 课堂中的角色

| 步骤 | 家长做什么 | 孩子做什么 |
|------|-----------|-----------|
| 魔法时刻 | 打开终端，输入演示命令 | 看效果，发出惊叹 |
| 规划结构 | 帮忙画草图 | 说出想要哪些版块 |
| 生成网页 | 在 Claude Code 中输入需求 | 描述想要的网页 |
| 添加内容 | 协助操作文件 | 选择照片、描述感受 |
| 调整美化 | 输入调整提示词 | 说出想改的地方 |

---

## 引导技巧

1. **鼓励选择**：照片让孩子自己选，尊重她的审美
2. **引导描述**：当孩子说"好看一点"时，引导她说"什么颜色""多大""放在哪里"
3. **双版块协调**：引导孩子思考两个版块风格如何区分又保持整体协调
4. **真实素材**：强调用真实照片和视频，比AI生成的更有说服力

---

## 常见问题

**Q：视频文件太大怎么办？**
A：可以让 Claude Code 帮忙处理，或者用视频截图代替。如果确实无法嵌入，就只放照片。

**Q：两个版块风格差异太大怎么办？**
A：让 Claude Code 统一字体和整体布局，只在颜色和装饰元素上做区分。

**Q：孩子想加的内容太多怎么办？**
A：提醒她"先做最重要的部分，其他可以课后慢慢加"，控制课堂节奏。

## 安全注意事项

1. 网页中的介绍不要写具体学校名称和家庭住址
2. 分享网页时只发给熟悉的人（家人、老师）
3. 照片如果包含其他小朋友，分享前需征得对方家长同意
