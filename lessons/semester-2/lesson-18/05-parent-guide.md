# 家长指南：第18课 - Yanny的钢琴音乐展示页

## 课前准备（提前一天完成）

### 收集材料

1. **钢琴照片**：在手机或电脑里找3-5张照片
   - 日常练琴的照片
   - 表演或比赛的照片
   - 和钢琴老师的合影
   - 把照片传到电脑上，放到一个文件夹里

2. **AI音乐文件**：找到第13课用豆包生成的音乐
   - 如果是音频文件（.mp3），直接准备好
   - 如果找不到了，可以让豆包重新生成一段钢琴曲

3. **学琴记录**：整理一下Yanny学过的曲子
   - 从第一首到现在，按时间顺序列出来
   - 每首大概什么时候开始学的

### 环境确认

- Claude Code 已安装，终端能正常启动
- 建一个新的文件夹，比如 `钢琴网页`，把照片和音乐文件都放进去
- 在这个文件夹里打开终端，启动 Claude Code

## 课堂中的角色

| 步骤 | 家长做什么 | 孩子做什么 |
|------|-----------|-----------|
| 魔法时刻 | 打开终端，输入演示命令 | 看效果，发出惊叹 |

| 生成网页 | 在 Claude Code 中输入需求 | 描述想要的网页 |
| 添加内容 | 协助操作文件 | 选择照片、描述感受 |
| 调整美化 | 输入调整提示词 | 说出想改的地方 |

---

## 引导技巧

1. **鼓励选择**：照片让孩子自己选，尊重她的审美
2. **引导描述**：当孩子说"好看一点"时，引导她说"什么颜色""多大""放在哪里"
3. **连接情感**：做成长记录时，引导孩子回忆每首曲子的故事
4. **音乐体验**：播放AI音乐时，和孩子一起安静地听完，分享感受

---

## 常见问题

**Q：音乐文件格式不对怎么办？**
A：可以让 Claude Code 帮忙处理，或者用豆包重新生成一段钢琴曲。如果确实无法嵌入，就让 Claude Code 做一个模拟的播放器样式。

**Q：照片太多或太大怎么办？**
A：告诉 Claude Code："帮我把照片缩小到合适的尺寸"，它会自动处理。

**Q：孩子想加的内容太多怎么办？**
A：提醒她"先做最重要的部分，其他可以课后慢慢加"，控制课堂节奏。
