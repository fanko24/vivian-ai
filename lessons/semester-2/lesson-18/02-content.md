# 知识小课堂：让网页"活"起来

## 1. 网页可以展示多媒体

有的网页能播放音乐、看视频、还能滑来滑去看照片——这就是**多媒体**！网页不只是文字和图片，还能展示声音、动画和视频。

| 多媒体类型 | 网页里怎么展示 |
|-----------|--------------|
| 🎵 音乐 | 音乐播放器，点一下就播放 |
| 📸 照片 | 照片画廊，可以一张张浏览 |
| 🎬 视频 | 嵌入视频播放窗口 |
| ✨ 动画 | 图片和文字可以动起来 |

这节课，我们要让钢琴网页"活"起来——能听音乐、能看照片！

---

## 2. 用 Claude Code 描述页面布局

做网页前，先想清楚页面要分成几个部分——就像装修房子，先想好哪里是客厅、哪里是卧室。

**跟 Claude Code 说话的秘诀：**

| 要说清楚 | 举例 |
|---------|------|
| 分几个区 | "上面是标题区，中间是音乐区，下面是照片区" |
| 每个区放什么 | "音乐区放一个播放器，能播放我的曲子" |
| 想要什么风格 | "优雅的钢琴风格，黑白色调配金色装饰" |

---

## 3. 音乐播放器是什么

网页里的音乐播放器就像一个小遥控器：有播放按钮 ▶️、暂停按钮 ⏸，还能显示曲子名字和播放进度。

你只需要告诉 Claude Code："帮我加一个音乐播放器"，它就知道怎么做了！

---

## 4. 配色和风格选择

不同的主题适合不同的颜色和风格：

| 主题 | 适合的配色 | 感觉 |
|------|-----------|------|
| 🎹 钢琴 | 黑白 + 金色 | 优雅、经典 |
| 🌸 春天 | 粉色 + 绿色 | 清新、可爱 |
| 🏊 游泳 | 蓝色 + 白色 | 清爽、活力 |
| 🎨 画画 | 彩色、鲜艳 | 活泼、创意 |

钢琴主题用黑白色调配金色点缀，就像真正的三角钢琴一样优雅！

记住安全口诀：在网上分享内容时，不要放太多个人信息哦！
