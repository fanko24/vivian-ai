# 薇薇安的钢琴小舞台

## 故事开始

薇薇安刚练完一首新曲子，开心地在琴键上弹了一遍又一遍。

"妈妈，你看我弹得是不是越来越好了？"薇薇安转过头问。

妈妈走过来，笑着说："进步可大了！要是能把你的钢琴学习过程都记录下来就好了——从第一次摸琴键到现在能弹一整首曲子。"

"第13课的时候，我用豆包做过一个钢琴专辑PDF呀！"薇薇安想起了之前的课。

"那如果把你的钢琴故事做成一个网页呢？"妈妈说，"网页上可以放音乐，别人打开就能听！还能放你的照片，看到你弹琴的样子。"

## 新的挑战

"像一个小舞台一样！"薇薇安眼睛亮了起来，"有自己的音乐、自己的照片，还有我从零开始学钢琴的故事。"

她打开电脑，看着屏幕想：这个网页要有哪些"节目"呢？

"首先要有自我介绍——大家好，我是Yanny，欢迎来到我的钢琴小舞台！"

"然后要有一个音乐区，把之前用AI创作的曲子放上去，点一下就能听。"

"还要有照片区，放我练琴和表演的照片。"

"最后是成长记录，写上我从第一节课到现在学过的每一首曲子！"

## 动手吧

薇薇安打开 Claude Code，对着屏幕说："帮我做一个钢琴展示网页，要有音乐播放器、照片画廊，还要记录我的成长故事！"

Claude Code 马上开始工作了。

几分钟后，一个漂亮的钢琴网页出现在屏幕上——优雅的配色、飘动的音符、能播放音乐的按钮……

"这就是我的钢琴小舞台！"薇薇安高兴地拍手，"我要把它分享给钢琴老师看！"
