# 动手实践：制作钢琴音乐展示页

## 30秒魔法时刻

家长打开 Claude Code，输入：

> "帮我做一个钢琴展示网页，要有标题'我的钢琴小舞台'，黑色背景配金色文字，中间有一个音乐播放器，能播放一首钢琴曲。"

几秒钟后，一个优雅的钢琴网页出现在屏幕上！

"看！这就是你今天要做的东西——一个属于你的钢琴小舞台！"

---

## 第1步：规划网页结构（5分钟）

拿出一张纸，画出你的网页草图。

**参考结构：**

```
我的钢琴小舞台
├── 介绍区：我是谁、学钢琴多久了
├── 音乐区：AI创作的曲子，有播放按钮
├── 照片区：练琴和表演的照片
└── 成长记录：学过的曲子列表
```

**跟爸爸妈妈讨论：**
- 介绍区写什么？（名字、学琴时间、最喜欢的曲子）
- 音乐区放几首曲子？（1-3首）
- 照片区选哪些照片？（3-5张最有代表性的）
- 成长记录写哪些曲子？（从第一首到现在）

> 小提示：先画草图再动手，做起来更快！

---

## 第2步：用 Claude Code 生成网页框架（10分钟）

### 2.1 打开终端并启动 Claude Code

1. **Mac电脑**：按 `Cmd + 空格`，输入 `终端`，按回车
2. 在终端里输入 `claude`，按回车
3. 等几秒钟，看到输入提示就可以开始了

### 2.2 输入需求

把下面这段话输入 Claude Code（把 ___ 换成你自己的内容）：

```
帮我做一个钢琴展示网页，要求：
1. 标题叫"Yanny的钢琴小舞台"
2. 优雅的钢琴风格，黑色背景配金色文字和装饰
3. 分成4个区域：
   - 介绍区：我叫___，学钢琴已经___了，最喜欢弹___
   - 音乐区：放一个音乐播放器，能播放我的AI音乐
   - 照片区：展示3-5张我弹钢琴的照片
   - 成长记录：列出我学过的曲子
4. 页面要有音符飘动的动画效果
5. 最上面有导航栏，可以跳到每个区域
6. 把所有代码写在一个 index.html 文件中
```

### 2.3 预览网页

1. Claude Code 生成完后，找到 `index.html` 文件
2. 双击打开，在浏览器里看看效果
3. 觉得哪里不满意，记住等一下调整

---

## 第3步：添加音乐和照片（10分钟）

### 3.1 添加AI音乐

找到第13课用豆包生成的AI音乐文件，把它放到和 `index.html` 同一个文件夹里。

然后告诉 Claude Code：

```
在音乐区添加一个播放器，播放我的AI音乐。
音乐文件名叫 ___（填你的文件名）。
显示曲子的名字叫"我的第一首AI钢琴曲"。
加一个漂亮的播放按钮和进度条。
```

> 如果音乐文件找不到，可以让 Claude Code 先做一个模拟播放器的样子。

### 3.2 添加钢琴照片

把准备好的钢琴照片也放到同一个文件夹里，然后告诉 Claude Code：

```
在照片区添加一个照片画廊，展示我的钢琴照片。
照片横排排列，每张照片下面写一句说明。
鼠标放到照片上，照片会稍微放大。
```

把照片文件名逐个告诉 Claude Code，比如：
- `练琴.jpg` —— "我每天坚持练琴"
- `表演.jpg` —— "第一次上台表演"
- `和老师.jpg` —— "和钢琴老师在一起"

---

## 第4步：调整配色和布局（5分钟）

看看网页，跟爸爸妈妈讨论哪些地方想调整。

**常见的调整提示词：**

| 想改什么 | 跟 Claude Code 说 |
|---------|-----------------|
| 标题不够醒目 | "把标题改大一些，加上金色的发光效果" |
| 想加音符装饰 | "在标题旁边加上浮动的音符emoji" |
| 照片太小 | "每张照片放大到300像素宽" |
| 想加背景音乐 | "页面打开时自动播放一段轻柔的背景钢琴音" |
| 成长记录太简单 | "成长记录用时间线的样式展示，每首曲子标上学琴时间" |
| 整体不够优雅 | "加上钢琴键盘的装饰条，黑色和白色相间" |

**每次只改1-2个地方，改完刷新浏览器看效果。**

---

## 成品标准

完成的钢琴展示网页应该包含：

- [ ] 标题写着"Yanny的钢琴小舞台"，金色醒目
- [ ] 介绍区有Yanny的钢琴学习简介
- [ ] 音乐区有播放器，能播放AI音乐（或显示播放器样式）
- [ ] 照片区展示了3-5张钢琴照片
- [ ] 成长记录列出了学过的曲子
- [ ] 整体风格优雅，黑白配色加金色装饰
- [ ] 有音符飘动或其他小动画
- [ ] 网页在浏览器中能正常打开

## 分享你的作品

1. 告诉 Claude Code："请把所有图片和音乐内嵌到 HTML 文件中"
2. 把生成的 HTML 文件通过微信发给钢琴老师
3. 老师在手机上点击文件 → 选择"用浏览器打开"就能看到你的钢琴小舞台啦！

> 也可以请全家人一起围在电脑前，打开网页，一边听音乐一边看照片！
