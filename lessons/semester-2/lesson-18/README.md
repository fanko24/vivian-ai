# 第18课：Yanny的才艺展示网页

**用时**：55分钟
**工具**：Claude Code + 豆包（doubao.com）
**难度**：⭐⭐⭐⭐

---

## 学习目标

1. 学会用 Claude Code 制作包含多个才艺版块的展示网页
2. 了解网页中如何展示多种多媒体内容（视频播放器、照片画廊）
3. 学会规划一个多版块的网页结构

## 准备清单

- [ ] Claude Code 已安装并能正常启动
- [ ] **收集3-5张钢琴练习/表演的照片和视频**
- [ ] **收集3-5张羽毛球训练的照片和视频**
- [ ] 打开豆包网页（doubao.com）备用
- [ ] 准备一张纸和彩色笔（画网页草图用）

## 时间安排

| 环节 | 时间 | 内容 |
|------|------|------|
| 30秒魔法时刻 | 3 分钟 | 看一个做好的才艺展示网页 |
| 规划网页 | 7 分钟 | 画出网页结构草图 |
| 知识讲解 | 5 分钟 | 多版块网页的设计 |
| 动手实践 | 30 分钟 | 用 Claude Code 制作网页 |
| 作品分享 | 10 分钟 | 展示和互相欣赏 |

---

> 本课要把你的钢琴和羽毛球照片、视频都放进一个漂亮的网页！
