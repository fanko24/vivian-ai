# 第18课：Yanny的钢琴音乐展示页

**用时**：55分钟
**工具**：Claude Code + 豆包（doubao.com）
**难度**：⭐⭐⭐⭐

---

## 学习目标

1. 学会用 Claude Code 制作包含音乐和照片的展示网页
2. 了解网页中如何展示多媒体内容（音乐播放器、照片画廊）
3. 设计一个有风格的钢琴主题网页

## 准备清单

- [ ] Claude Code 已安装并能正常启动
- [ ] 收集3-5张钢琴练习/表演的照片（存在电脑上）
- [ ] 找到第13课用豆包生成的AI音乐文件
- [ ] 打开豆包网页（doubao.com）备用
- [ ] 准备一张纸和彩色笔（画网页草图用）

## 时间安排

| 环节 | 时间 | 内容 |
|------|------|------|
| 30秒魔法时刻 | 3 分钟 | 看一个做好的钢琴展示网页 |
| 规划网页 | 7 分钟 | 画出网页结构草图 |
| 知识讲解 | 5 分钟 | 多媒体网页的奥秘 |
| 动手实践 | 30 分钟 | 用 Claude Code 制作网页 |
| 作品分享 | 10 分钟 | 展示和互相欣赏 |

---

> 本课要用到第13课的AI音乐和你的钢琴照片，把它们变成一个漂亮的网页！
