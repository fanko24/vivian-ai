# 动手实践：制作钢琴音乐展示网页

## 30秒魔法时刻

家长打开 Claude Code，输入：

> "帮我做一个钢琴展示网页，标题叫'我的钢琴小舞台'，黑色背景配金色文字，中间有一个音乐播放器，有音符飘动的动画。"

几秒钟后，一个优雅的钢琴网页出现在屏幕上！

"看！这就是你今天要做的东西——一个属于你的钢琴小舞台！"

---

## 第1步：规划网页结构（5分钟）

拿出一张纸，画出你的网页草图：

```
我的钢琴小舞台
├── 介绍区：我是谁、学钢琴多久了
├── 音乐区：AI创作的曲子，有播放按钮
├── 照片区：练琴和表演的照片
└── 成长记录：学过的曲子列表
```

**跟爸爸妈妈讨论：**
- 介绍区写什么？（名字、学琴时间、最喜欢的曲子）
- 音乐区放几首曲子？（1-3首）
- 照片区选哪些照片？（3-5张最有代表性的）
- 成长记录写哪些曲子？（从第一首到现在）

> 小提示：先画草图再动手，做起来更快！

---

## 第2步：用豆包准备内容（10分钟）

### 2.1 写钢琴介绍

1. 打开浏览器，输入 `doubao.com`，按回车
2. 在对话框中输入：

> "我是一个8岁的小女孩，学钢琴已经___（填时间）了。我最喜欢弹___（填曲子名字），我的钢琴老师叫___。帮我写一段100字以内的钢琴小介绍，活泼可爱一点。"

3. 看看豆包写的喜不喜欢，不满意就说：
   - "加上我最近在学___这首曲子"
   - "语气再活泼一点"
4. 满意后复制保存下来

### 2.2 写成长记录

继续在豆包里输入：

> "我学过的钢琴曲子有：___（列出曲子名）。帮我按时间顺序排列，每首曲子加一句可爱的点评。"

把豆包生成的列表保存好。

### 2.3 用豆包画钢琴装饰图（可选）

1. 点击豆包左侧导航栏的 **"图像生成"**
2. 输入：

> "儿童插画风格，一个小女孩坐在钢琴前弹琴，黑色钢琴，金色音符飘在空中，温暖灯光，色彩明亮"

3. 在图片上 **右键** → **"另存为"**，保存到桌面

---

## 第3步：用Claude Code生成网页（15分钟）

### 3.1 打开终端并启动 Claude Code

1. **Mac电脑**：按 `Cmd + 空格`，输入 `终端`，按回车
   **Windows电脑**：按 `Win + R`，输入 `cmd`，按回车
2. 在终端里输入 `claude`，按回车
3. 等几秒钟，看到输入提示就可以开始了

### 3.2 输入需求

把下面这段话输入 Claude Code（把 ___ 换成你自己的内容）：

> "帮我做一个钢琴展示网页，要求：
> 1. 标题叫「Yanny的钢琴小舞台」
> 2. 优雅的钢琴风格，黑色背景配金色文字和装饰
> 3. 分成4个区域：
>    - 介绍区：我叫___，学钢琴已经___了，最喜欢弹___
>    - 音乐区：放一个音乐播放器的样式
>    - 照片区：展示3-5张我弹钢琴的照片
>    - 成长记录：列出我学过的曲子
> 4. 页面要有音符飘动的动画效果
> 5. 最上面有导航栏，可以跳到每个区域
> 6. 把所有代码写在一个 index.html 文件中"

### 3.3 预览网页

1. Claude Code 生成完后，在终端输入 `open index.html`（Mac）或双击 `index.html` 文件
2. 浏览器会打开网页，看看效果
3. 觉得哪里不满意，记住等一下调整

---

## 第4步：添加照片并美化（10分钟）

### 4.1 添加钢琴照片

1. 把准备好的钢琴照片（3-5张）放到和 `index.html` 同一个文件夹里
2. 回到 Claude Code，输入：

> "在照片区添加一个照片画廊，展示我的钢琴照片。照片横排排列，每张下面写一句说明。鼠标放到照片上会稍微放大。
> 照片文件名：___（填你的文件名）"

3. 也可以把照片文件直接 **拖到终端窗口** 里，Claude Code 会自动识别

### 4.2 美化调整

看看网页，跟爸爸妈妈讨论哪些地方想调整：

| 想改什么 | 跟 Claude Code 说 |
|---------|-----------------|
| 标题不够醒目 | "把标题改大一些，加上金色的发光效果" |
| 想加音符装饰 | "在标题旁边加上浮动的音符emoji" |
| 照片太小 | "每张照片放大到300像素宽" |
| 成长记录太简单 | "成长记录用时间线样式，每首曲子标上学琴时间" |
| 整体不够优雅 | "加上钢琴键盘的装饰条，黑白相间" |

**每次只改1-2个地方，改完刷新浏览器看效果。**

### 4.3 内嵌图片

调整满意后，告诉 Claude Code：

> "请把所有图片内嵌到 HTML 文件中，让这个网页可以单独打开"

---

## 第5步：预览和分享（5分钟）

### 最终检查

打开 `index.html`，逐项检查：
- [ ] 标题「Yanny的钢琴小舞台」金色醒目
- [ ] 介绍区有你的钢琴学习简介
- [ ] 音乐区有播放器样式
- [ ] 照片区展示了3-5张钢琴照片
- [ ] 成长记录列出了学过的曲子
- [ ] 有音符飘动的动画
- [ ] 所有图片已内嵌，文件可以独立打开

### 分享你的作品

1. 把 `index.html` 通过微信发给钢琴老师
2. 老师在手机上点击文件 → 选择"用浏览器打开"就能看到你的钢琴小舞台啦！

> 也可以请全家人围在电脑前，打开网页一起欣赏！
