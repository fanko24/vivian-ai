# 动手实践：制作才艺展示网页

## 30秒魔法时刻

家长打开 Claude Code，输入：

> "帮我做一个才艺展示网页，标题叫'Yanny的才艺秀'，有两个版块：钢琴区和运动区。钢琴区用黑金配色，运动区用蓝绿配色。每个版块有照片画廊。"

几秒钟后，一个漂亮的才艺展示网页出现在屏幕上！

"看！这就是你今天要做的东西——一个展示你所有才艺的网页！"

---

## 第1步：规划网页结构（7分钟）

拿出一张纸，画出你的网页草图：

```
Yanny的才艺秀
├── 标题区：网页名称 + 简短介绍
├── 钢琴区：
│   ├── 介绍：学钢琴多久了、最喜欢的曲子
│   ├── 照片画廊：3-5张弹钢琴的照片
│   └── 视频区：1-2段弹钢琴的视频（如果有）
├── 运动区（羽毛球）：
│   ├── 介绍：学羽毛球多久了、最擅长的动作
│   ├── 照片画廊：3-5张训练照片
│   └── 视频区：1-2段训练视频（如果有）
└── 页脚：一句话总结
```

**跟爸爸妈妈讨论：**
- 钢琴区选哪些照片？哪些视频？
- 运动区选哪些照片？哪些视频？
- 每个区域写什么介绍文字？

---

## 第2步：用豆包准备内容（10分钟）

### 2.1 写钢琴区介绍

1. 打开浏览器，输入 **doubao.com**
2. 在对话框中输入：

> "我是一个8岁的小女孩，学钢琴已经___了。我最喜欢弹___。帮我写一段80字以内的钢琴区介绍，活泼可爱一点。"

3. 满意后复制保存

### 2.2 写运动区介绍

继续在豆包里输入：

> "我还在学打羽毛球，最擅长___（填动作）。帮我写一段80字以内的运动区介绍，充满活力。"

### 2.3 写照片说明

让豆包帮每张照片写一句说明：

> "我有这些照片：[简单描述每张照片的内容]。请帮每张照片写一句10-15个字的说明。"

---

## 第3步：用Claude Code生成网页（15分钟）

### 3.1 打开终端并启动 Claude Code

1. **Mac电脑**：按 `Cmd + 空格`，输入 `终端`，按回车
   **Windows电脑**：按 `Win + R`，输入 `cmd`，按回车
2. 在终端里输入 `claude`，按回车

### 3.2 输入需求

把下面这段话输入 Claude Code（把 ___ 换成你自己的内容）：

> "帮我做一个才艺展示网页，要求：
> 1. 标题叫「Yanny的才艺秀」
> 2. 分成两个大版块：钢琴区和运动区
> 3. 钢琴区：黑金配色，优雅风格，有照片画廊
> 4. 运动区：蓝绿配色，活力风格，有照片画廊
> 5. 每个版块有介绍文字和照片说明
> 6. 页面最上面有导航栏，可以跳到每个版块
> 7. 把所有代码写在一个 index.html 文件中"

### 3.3 预览网页

1. Claude Code 生成完后，在终端输入 `open index.html`（Mac）或双击文件
2. 浏览器会打开网页，看看效果

---

## 第4步：添加照片和视频并美化（10分钟）

### 4.1 添加钢琴照片

1. 把钢琴照片（3-5张）放到和 `index.html` 同一个文件夹里
2. 回到 Claude Code，输入：

> "在钢琴区的照片画廊里添加我的钢琴照片，横排排列，每张下面有说明。鼠标放上去会稍微放大。照片文件名：___"

### 4.2 添加运动照片

同样方式添加羽毛球训练照片：

> "在运动区的照片画廊里添加我的羽毛球照片，横排排列，每张下面有说明。照片文件名：___"

### 4.3 添加视频（如果有）

> "在___区添加一个视频播放区，视频文件名：___"

### 4.4 美化调整

| 想改什么 | 跟 Claude Code 说 |
|---------|-----------------|
| 标题不够醒目 | "把标题改大一些，加上发光效果" |
| 两个版块分不清 | "钢琴区和运动区之间加一条分隔线" |
| 照片太小 | "每张照片放大到300像素宽" |
| 想加背景音乐 | "在页面加一个背景音乐播放按钮" |

**每次只改1-2个地方，改完刷新浏览器看效果。**

### 4.5 内嵌图片

调整满意后：

> "请把所有图片内嵌到 HTML 文件中，让这个网页可以单独打开"

---

## 第5步：预览和分享（5分钟）

### 最终检查

- [ ] 标题「Yanny的才艺秀」醒目
- [ ] 钢琴区有介绍、照片（和视频）
- [ ] 运动区有介绍、照片（和视频）
- [ ] 两个版块风格不同但协调
- [ ] 有导航栏可以跳转
- [ ] 所有图片已内嵌

### 分享你的作品

1. 把 `index.html` 发给钢琴老师和羽毛球教练
2. 发到家庭群里，让大家看看你的才艺秀
3. 请全家人围在电脑前一起欣赏！

---

## 成品标准

- [ ] 标题「Yanny的才艺秀」
- [ ] 钢琴区有介绍 + 3-5张真实照片
- [ ] 运动区有介绍 + 3-5张真实照片
- [ ] 两个版块风格各异
- [ ] 有导航栏
- [ ] 所有图片已内嵌，文件可独立打开

记住安全口诀！分享网页时只发给认识的人哦！
