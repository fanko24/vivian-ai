# 第16课：Yanny的专属日历

> 用豆包生成插图，在PPT中制作个性化日历，彩色打印出来

## 课程信息

- **时长**：50分钟
- **工具**：豆包（doubao.com）+ PPT/WPS
- **难度**：⭐⭐⭐

## 学习目标

1. 规划日历内容，整理自己的重要日期和活动
2. 用豆包AI绘画为每个月份生成主题插图
3. 在PPT中排版制作完整日历，导出PDF打印

## 准备清单

- [ ] 列出Yanny的重要日期（生日、节日、活动等）
- [ ] 打开豆包（doubao.com）
- [ ] 打开 PPT 或 WPS 演示
- [ ] 准备彩色打印机（可选，没有也能完成）
- [ ] 家长陪伴

## 时间安排

| 环节 | 时间 |
|------|------|
| 30秒魔法时刻 | 3分钟 |
| 整理重要日期 | 7分钟 |
| 知识讲解 | 5分钟 |
| AI绘画+PPT排版 | 30分钟 |
| 作品分享 | 5分钟 |
