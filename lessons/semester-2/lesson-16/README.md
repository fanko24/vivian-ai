# 第16课：Yanny的成长日历

> 用真实活动照片替代AI插画，在PPT中制作个性化日历

## 课程信息

- **时长**：50分钟
- **工具**：豆包（doubao.com）+ PPT/WPS
- **难度**：⭐⭐⭐

## 学习目标

1. 规划日历内容，整理自己的重要日期和活动
2. 用真实活动照片替代AI插画，让日历更个人化
3. 在PPT中排版制作完整日历，导出PDF打印

## 准备清单

- [ ] 列出Yanny的重要日期（生日、节日、活动等）
- [ ] **准备12张活动照片**（每月一张代表照片，可以是游泳、钢琴、羽毛球、画画或日常生活）
- [ ] 打开豆包（doubao.com）
- [ ] 打开 PPT 或 WPS 演示
- [ ] 准备彩色打印机（可选）
- [ ] 家长陪伴

## 时间安排

| 环节 | 时间 |
|------|------|
| 30秒魔法时刻 | 3分钟 |
| 整理日期和照片 | 7分钟 |
| 知识讲解 | 5分钟 |
| 豆包写月份语+PPT排版 | 30分钟 |
| 作品分享 | 5分钟 |
