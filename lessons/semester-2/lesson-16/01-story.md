# 薇薇安遇见"会写代码的魔法师"

## 故事

薇薇安的房间里贴满了各种纸条——

"周三下午游泳！""周五要练钢琴！""周六画画课！"

可是纸条总是掉下来，有时候还会搞混。

"要是有个大日历就好了，上面标好我所有的事情……"薇薇安叹了口气。

这时，小机器人豆豆飞了过来："薇薇安，我认识一个新朋友，他叫 Claude Code！"

"Claude……Code？"薇薇安歪着头，"名字好奇怪呀。"

"他是个会写代码的魔法师！"豆豆兴奋地说，"你只要用嘴巴告诉他你想要什么，他就能帮你变出来！"

"真的吗？"薇薇安不太相信，"我又不会写代码。"

"你不需要会写代码呀！"豆豆拉着薇薇安坐到电脑前，"你只要像跟我聊天一样，告诉 Claude Code 你想要什么就行了。"

薇薇安半信半疑地对屏幕说："帮我做一个漂亮的日历吧！要有12个月，每个月标记上我的活动。"

话音刚落，Claude Code 就开始工作了！屏幕上跳动着一行行神奇的文字。

几秒钟后——一个超级可爱的日历出现在屏幕上！

"哇！！！"薇薇安瞪大了眼睛，"这就是我说的日历！"

"你看，"豆豆笑着说，"你只是说了你想要什么，Claude Code 就帮你写出了所有的代码。这就是'用说话来编程'！"

"太酷了！"薇薇安高兴地跳起来，"我想让标题再大一点，颜色换成粉色！"

Claude Code 马上就改好了。

"我就是一个小小设计师！"薇薇安骄傲地说。

豆豆点点头："没错，你说想法，Claude Code 帮你实现。你们是最好的搭档！"
