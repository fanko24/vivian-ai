# 日历制作三步法

## 1. 规划日历内容

做日历的第一步不是画图，而是**整理日期**。

就像写作文先列提纲一样：
- 先把一年中最重要的日子找出来
- 分成几类：节日、生日、活动、比赛
- 想好每个月想配什么主题的图

**日历要标什么？**

| 类别 | 举例 |
|------|------|
| 我的生日 | Yanny的生日、家人生日 |
| 传统节日 | 春节、中秋节、端午节 |
| 我的活动 | 钢琴考级、游泳比赛、画画课 |
| 特别日子 | 开学第一天、暑假开始 |

## 2. AI绘画提示词公式

让豆包画出适合日历的插图，要用这个公式：

**季节 + 场景 + 主角 + 风格**

| 部分 | 示例 |
|------|------|
| 季节 | 春天的花园 |
| 场景 | 小女孩在樱花树下放风筝 |
| 主角 | 可爱的卡通小女孩 |
| 风格 | 儿童插画风格，色彩明亮温暖 |

**关键技巧**：每张图的风格保持一致，都加上"儿童插画风格"，这样12个月看起来才像同一套日历！

## 3. PPT排版秘诀

日历排版有3个要点：

1. **页面设置**：用竖版（A4），每页一个月
2. **上半部分**：放月份插图，占页面的40-50%
3. **下半部分**：放日历表格，标好日期和重要日子

页面方向用**纵向（A4）**，图片在上，日历格在下。

## 小结

记住三步口诀：**整理日期 → 画插图 → 排版打印！**
