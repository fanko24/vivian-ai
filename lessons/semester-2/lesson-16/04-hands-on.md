# 动手实践：制作Yanny的专属日历

## 30秒魔法时刻

家长打开 Claude Code，在屏幕上输入：

> "帮我做一个可爱的日历网页，有12个月，粉色主题。"

几秒钟后，一个漂亮的日历出现在屏幕上！

"看！这就是用说话让电脑帮你做东西！"

---

## 第一步：列出我的每周活动（5分钟）

拿出一张纸，和Yanny一起填写下面这个表格，把每周要做的事情写下来：

| 星期 | 活动名称 | 代表颜色 | 时间段 |
|------|----------|----------|--------|
| 周一 | 🏫 上学 | 橙色 | 白天 |
| 周二 | 🏫 上学 | 橙色 | 白天 |
| 周三 | 🏫 上学 + ___ | 橙色 + ___ | 白天 + ___ |
| 周四 | 🏫 上学 | 橙色 | 白天 |
| 周五 | 🏫 上学 | 橙色 | 白天 |
| 周六 | ___ 、___ | ___ | ___ |
| 周日 | ___ 、___ | ___ | ___ |

**常见活动参考**（选Yanny实际参加的填进去）：
- 🏊 游泳训练 → 蓝色
- 🎹 钢琴练习 → 粉色
- 🏸 羽毛球 → 绿色
- 🎨 画画课 → 黄色
- 📚 课外阅读 → 紫色

**别忘了这些重要日子**：
- Yanny的生日是 ___ 月 ___ 日（用红色标记）
- 爸爸/妈妈的生日是 ___ 月 ___ 日
- 开学第一天是 ___ 月 ___ 日
- 寒暑假开始的日子是 ___ 月 ___ 日

> 小提示：不用填得太满，先挑3-4个最重要的活动就好！

---

## 第二步：用豆包生成可爱小图标（5分钟）

我们要为每个活动生成一个小图标，后面会放到日历上。

**操作步骤**：

1. **打开豆包**
   - 在浏览器地址栏输入 `doubao.com`，按回车
   - 如果还没登录，用手机号登录（家长帮忙）

2. **找到AI绘画功能**
   - 在豆包聊天框旁边，找到「AI绘画」按钮（通常是一个小画笔图标）
   - 点击进入AI绘画页面

3. **逐个生成活动图标**
   - 在输入框中输入下面的提示词，每生成一个就保存一个：

   | 活动 | 输入内容（直接复制） |
   |------|---------------------|
   | 🏊 游泳 | "画一个可爱的游泳小女孩图标，蓝色水花背景，卡通简笔画风格，适合做小图标" |
   | 🎹 钢琴 | "画一个可爱的钢琴图标，粉色音符装饰，卡通简笔画风格，适合做小图标" |
   | 🏸 羽毛球 | "画一个可爱的羽毛球拍图标，绿色背景，卡通简笔画风格，适合做小图标" |
   | 🎨 画画 | "画一个可爱的调色盘图标，黄色星星装饰，卡通简笔画风格，适合做小图标" |
   | 🎂 生日 | "画一个可爱的生日蛋糕图标，红色气球装饰，卡通简笔画风格，适合做小图标" |

4. **保存图片**
   - 看到满意的图片后，点击图片右下角的「下载」按钮（一个向下的箭头）
   - 把图片保存到桌面上，建一个文件夹叫 `日历图标`
   - 给每个图片改个容易认的名字，比如 `游泳.png`、`钢琴.png`、`生日蛋糕.png`

5. **检查一下**
   - 打开桌面上的 `日历图标` 文件夹，看看是不是每个活动的图标都有了
   - 一共需要 5 个图标（游泳、钢琴、羽毛球、画画、生日蛋糕）

> 小提示：如果生成的图片不满意，就点「重新生成」，多试几次，挑最喜欢的！

---

## 第三步：告诉 Claude Code 做日历（15分钟）

这一步需要家长操作电脑，Yanny在旁边说出自己的想法。

### 3.1 打开终端

- **Mac电脑**：按 `Command + 空格`，输入 `终端`，按回车
- **Windows电脑**：按 `Win + R`，输入 `cmd`，按回车
- 打开后会看到一个黑底白字的窗口

### 3.2 启动 Claude Code

1. 在终端里输入 `claude`，按回车
2. 等几秒钟，会看到 Claude Code 的欢迎界面
3. 光标闪烁时，说明可以输入了

### 3.3 输入Yanny的需求

把下面这段话**完整复制**，粘贴到 Claude Code 中：

> "帮我做一个2026年的日历网页，要求如下：
> 1. 有完整的12个月，每个月单独一个区域
> 2. 标题写「Yanny的专属日历」，要大字、粉色
> 3. 整体风格温馨可爱，粉色系主题
> 4. 以下活动需要用颜色标记：
>    - 游泳训练（蓝色）→ 周___ 和 周___
>    - 钢琴练习（粉色）→ 周___
>    - 羽毛球（绿色）→ 周___
>    - 画画课（黄色）→ 周___
> 5. 我生日是 ___月___日，那天要画一个生日蛋糕，用红色特别标记
> 6. 页面最上面有导航栏，点击月份名字可以跳到对应月份
> 7. 请把所有代码写在一个 index.html 文件中"

**注意**：要把上面的 `___` 替换成第一步里填好的实际内容！

### 3.4 等待生成

- Claude Code 会开始写代码，屏幕上会滚动显示文字
- 这个过程大约需要 30-60 秒，耐心等待
- 当看到提示符重新出现，说明生成完毕了

### 3.5 在浏览器中查看

1. Claude Code 会告诉你文件保存在哪里（通常是当前文件夹下的 `index.html`）
2. 打开「访达」（Mac）或「文件资源管理器」（Windows）
3. 找到 `index.html` 文件
4. 双击打开它，浏览器会自动启动并显示日历
5. 看看效果！是不是很神奇？

> 如果页面显示不正常，别着急，下一步我们会调整！

## 第四步：调整和优化（15分钟）

看看日历，和Yanny一起讨论哪里需要改。每次只改1-2个地方，改完刷新浏览器看效果。

**具体调整方法**：回到 Claude Code 的终端窗口，输入下面的提示词：

### 如果标题不好看
输入：
> "把标题「Yanny的专属日历」改大一些，加上可爱的阴影效果，字体用圆滚滚的"

### 如果颜色不够粉
输入：
> "整体背景改成浅粉色 #FFE4E1，月份标题用深粉色 #FF69B4"

### 如果想让活动标记更明显
输入：
> "每个有活动的日期格子，用对应的颜色画一个小圆点，鼠标放上去能显示活动名称"

### 如果想让生日更醒目
输入：
> "我生日那天（___月___日）的格子要特别大，加一个生日蛋糕的emoji 🎂，背景用红色渐变"

### 如果想加上图标
输入：
> "在每个月的月份标题旁边，加上对应季节的小装饰，比如1月加雪花❄️，7月加太阳☀️"

### 如果想让月份排列更好看
输入：
> "12个月排成3行4列的网格布局，每个月份卡片有圆角和阴影"

### 如果想打印出来更好看
输入：
> "加一个打印样式，打印时只显示日历内容，去掉背景颜色，用黑白线条风格"

**调整小技巧**：
- 每次改完，在浏览器中按 `F5` 或 `Cmd+R` 刷新页面看效果
- 不满意就继续输入新的调整要求
- 可以告诉 Claude Code："改回刚才的样子" 来撤销修改

---

## 第五步：保存和分享（5分钟）

### 保存 HTML 文件

1. 在 Claude Code 的终端中输入：
   > "请把所有样式和内容都写在一个 index.html 文件中，不依赖外部网络资源"

2. 找到保存的文件：
   - **Mac**：打开「访达」→ 左侧点击当前文件夹 → 找到 `index.html`
   - **Windows**：打开「文件资源管理器」→ 找到当前文件夹 → 找到 `index.html`

3. 双击 `index.html` 确认能正常打开

### 发给家人看（微信分享）

1. 打开微信电脑版
2. 找到要分享的家人聊天窗口
3. 把 `index.html` 文件直接拖到聊天框里
4. 点击「发送」
5. 家人在手机上点击文件 → 选择「用浏览器打开」就能看啦

### 打印纸质版

1. 在浏览器中打开 `index.html`
2. 按 `Ctrl+P`（Mac 按 `Cmd+P`）
3. 在打印设置中选择「另存为 PDF」或直接连接打印机
4. 保存/打印出来，贴在Yanny的房间墙上！

## 成品标准

完成的作品应该包含：

- [ ] 标题写着「Yanny的专属日历」，字体又大又可爱
- [ ] 完整的12个月份（1月到12月），每个月都能看到
- [ ] 每周的活动用不同颜色标记（游泳蓝色、钢琴粉色等）
- [ ] Yanny的生日那天有特别标记（红色 + 生日蛋糕）
- [ ] 页面风格温馨可爱，以粉色为主色调
- [ ] 能在浏览器中正常打开，不报错
- [ ] 文件是一个独立的 `index.html`，双击就能看
- [ ] 可以通过微信发给家人，家人也能正常打开
