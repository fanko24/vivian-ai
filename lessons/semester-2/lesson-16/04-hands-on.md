# 动手实践：制作个人网站

## 30 秒魔法时刻

爸爸妈妈先展示一个完整的个人网站示例，让Yanny看到最终效果："这就是你今天要做的——一个完全属于你的网站！"

---

## 第 1 步：规划网站结构（10 分钟）

拿出一张 A4 纸，横着放，画出你的网站地图。

**画网站地图的方法：**
1. 在纸的中间画一个大框，写上网站名字"你好，我是Yanny"
2. 从中间向右画几条线，每条线连到一个板块
3. 每个板块画一个框，写上板块的名字
4. 在每个板块框里，简单写上要放什么内容

**参考网站地图：**

```
你好，我是Yanny！（首页）
  ├── 自我介绍（头像 + 一段话 + 我的爱好）
  ├── 游泳世界（比赛时间线 + 照片）
  ├── 我的画作（线上画展）
  ├── 钢琴角（练习心得 + 喜欢的曲子）
  └── 羽毛球天地（小教练手册）
```

**跟爸爸妈妈讨论每个板块放什么：**
- 自我介绍：你想让别人第一眼了解你什么？（名字、年龄、最喜欢的几件事）
- 游泳世界：放第 13 课做的比赛时间线
- 我的画作：放第 15 课做的线上画展
- 钢琴角：放第 14 课做的日历里跟钢琴相关的内容，加上你喜欢的曲子
- 羽毛球天地：可以简单写几句你打球的故事

> 小提示：板块不要超过 5-6 个，太多了网站会太长，看不完。

## 第 2 步：用豆包准备素材

**写自我介绍：**

1. 打开豆包（doubao.com）
2. 输入这段提示词：

```
我是一个 8 岁的女孩，叫Yanny，上一年级。
我喜欢游泳、画画、弹钢琴和打羽毛球。
我的性格是：开朗、爱笑、喜欢交朋友。
帮我写一段有趣的自我介绍，100 字以内，要让人一看就觉得我很可爱！
```

3. 看看豆包写的喜不喜欢，不满意就说：
   - "太长了，帮我缩短到 50 个字"
   - "加上我还会画画这个爱好"
   - "语气再活泼一点"
4. 满意后复制保存下来

**做头像：**

1. 继续在豆包里输入：

```
帮我画一张可爱的小女孩头像，卡通风格，短头发，
穿着彩色衣服，背景是浅粉色，圆圆的脸蛋，大大的眼睛，
风格要温馨可爱，像绘本里的插画。
```

2. 豆包会生成一张图片
3. 不满意就说："头发换成长头发"或"衣服换成蓝色"
4. 满意后点击图片，保存到电脑上

**准备其他素材：**
- 找 1-2 张你最满意的个人照片（游泳、画画、弹琴的照片都可以）
- 想一句网站的欢迎语，比如"欢迎来到我的小天地！"

## 第 3 步：用 Claude Code 制作网站（25 分钟）

**打开终端：**
1. 爸爸妈妈按 `Cmd + 空格`（Mac）或点击搜索按钮，输入"终端"或"Terminal"
2. 打开终端应用，会看到一个黑色或白色的窗口，有一个闪烁的光标

**启动 Claude Code：**
1. 在终端里输入 `claude`，然后按回车键
2. 等几秒钟，出现欢迎信息和输入框，就可以开始说话了

**告诉 Claude Code 你的需求：**

在 Claude Code 的输入框里，输入这段话：

```
帮我做一个个人网站，叫"你好，我是Yanny"。
风格温馨可爱，浅色背景。
板块：首页（自我介绍+头像）、游泳世界、我的画作、钢琴角、羽毛球天地。
最上面要有导航栏，可以点击跳到每个板块。
首页显示我的自我介绍和头像，其他板块先留着空位，我后面再填充内容。
```

按 `Shift + Enter` 发送，等待 Claude Code 生成网页框架。

**预览网页：**
1. Claude Code 生成完成后，在浏览器中打开生成的 HTML 文件
2. Mac 用户：在 Finder 中找到文件，双击打开
3. 或者在终端输入 `open index.html`（Mac）/ `start index.html`（Windows）
4. 看看整体框架是不是你想要的，不满意就跟 Claude Code 说

## 第 4 步：整合之前的项目

现在要把之前几课做的项目放到网站里。

**先找到之前的项目文件：**

| 来源课程 | 项目内容 | 文件在哪里找 | 放到哪个板块 |
|----------|---------|-------------|-------------|
| 第 13 课 | 游泳比赛时间线 | 电脑上第 13 课生成的 HTML 文件 | 游泳世界 |
| 第 14 课 | 钢琴日历 | 电脑上第 14 课生成的 HTML 文件 | 钢琴角 |
| 第 15 课 | 线上画展 | 电脑上第 15 课生成的 HTML 文件 | 我的画作 |

**怎么整合——方法一（推荐）：**

把之前的 HTML 文件拖到终端窗口里，告诉 Claude Code：

```
这是我的游泳时间线网页：[把第13课的HTML文件拖进来]
这是我的画展网页：[把第15课的HTML文件拖进来]
这是我的日历网页：[把第14课的HTML文件拖进来]

请把游泳时间线的内容放到"游泳世界"板块，
把画展的内容放到"我的画作"板块，
把日历里跟钢琴相关的内容放到"钢琴角"板块。
羽毛球天地的内容我自己来写。
```

**怎么整合——方法二：**

如果文件太多，也可以一个一个来：

1. 先整合游泳时间线：
```
请把我的游泳时间线内容加到"游泳世界"板块。
[把第13课的HTML文件拖进来]
```

2. 再整合画展：
```
请把我的画展内容加到"我的画作"板块。
[把第15课的HTML文件拖进来]
```

3. 最后整合日历：
```
请把日历内容加到"钢琴角"板块。
[把第14课的HTML文件拖进来]
```

**填写羽毛球天地：**

告诉 Claude Code：
```
在"羽毛球天地"板块，帮我写一些简单的内容：
- 我什么时候开始学打羽毛球的
- 我最喜欢的羽毛球动作
- 给初学者 3 个小建议
```

然后把你自己的实际情况告诉 Claude Code，让它帮你写。

**每整合一个板块，都去浏览器刷新看看效果。**

## 第 5 步：美化与调整

检查网站：每个板块都有内容吗？导航能跳转吗？风格统一吗？跟 Claude Code 说你想调整的地方。

**具体调整示例：**
- "导航栏加上点击后变色的效果，让人知道当前在哪个板块"
- "自我介绍那段话旁边放上我的头像"
- "每个板块之间加一条分隔线或漂亮的装饰"
- "页面最上面加一个大的横幅图片，写上'欢迎来到我的小天地'"
- "整体字体再圆润可爱一点"
- "背景加上淡淡的渐变色"
- "页面底部加一行字：'这是Yanny用AI制作的个人网站'"
- "每个板块标题前面加一个小图标（游泳加浪花、画画加画笔……）"
- "导航栏固定在页面最上方，往下滑的时候也能看到"
- "加入一些小动画，比如页面滚动时内容慢慢出现"

**每次只改一两处，改完刷新看看效果**，满意了再改下一处。

**如果不确定要改什么，可以问自己：**
- 第一眼看到这个网站，你觉得好看吗？
- 有没有什么地方感觉空空的？
- 颜色搭配舒服吗？
- 字太小还是太大？

## 第 6 步：毕业庆祝

网站完成后：

1. 从头到尾浏览一遍，每个板块都点一点、看一看
2. 请爸爸妈妈也看一看，听听他们的建议
3. 如果想改，就用 Claude Code 继续调整
4. 满意了就保存好——这就是你的毕业作品！

请爸爸妈妈拿出毕业证书，一起填写。

## 成品标准

完成的个人网站应该包含：

- [ ] 首页有自我介绍和头像
- [ ] 至少 3 个内容板块（游泳世界、我的画作、钢琴角等）
- [ ] 最上方有导航栏，点击可以跳转到对应板块
- [ ] 整体风格统一（颜色、字体、间距协调）
- [ ] 之前的项目内容已整合进来（时间线、画展、日历等）
- [ ] 每个板块都有实际内容，不是空白
- [ ] 网页可以在浏览器中正常打开和浏览
- [ ] 导航栏能正常跳转到每个板块
