# 动手实践：制作个人网站

## 30 秒魔法时刻

爸爸妈妈先展示一个完整的个人网站示例，让原一看到最终效果："这就是你今天要做的——一个完全属于你的网站！"

---

## 第 1 步：规划网站结构（10 分钟）

拿出一张纸，画出网站地图：

```
你好，我是刘原一！（首页）
  ├── 自我介绍
  ├── 游泳世界（比赛时间线）
  ├── 我的画作（线上画展）
  ├── 钢琴角（专辑介绍）
  └── 羽毛球天地（小教练手册）
```

跟爸爸妈妈讨论每个板块放什么内容。

## 第 2 步：用豆包准备素材

**写自我介绍**：告诉豆包"我是一个 8 岁的女孩，喜欢游泳、画画、弹钢琴和打羽毛球，帮我写一段有趣的自我介绍。"

**做头像**：让豆包画一张可爱的头像，风格要温馨。

## 第 3 步：用 Claude Code 制作网站（25 分钟）

打开 Claude Code，告诉它：

```
帮我做一个个人网站，叫"你好，我是刘原一"。
风格温馨可爱，浅色背景。
板块：首页（自我介绍+头像）、游泳世界、我的画作、钢琴角、羽毛球天地。
最上面要有导航栏，可以点击跳到每个板块。
```

然后把之前项目的内容（第 13-15 课）逐步发给 Claude Code 整合。

## 第 4 步：整合之前的项目

| 来源课程 | 项目内容 | 放到哪个板块 |
|----------|---------|-------------|
| 第 13 课 | 游泳时间线/羽毛球手册 | 游泳世界 / 羽毛球天地 |
| 第 14 课 | 钢琴专辑介绍 | 钢琴角 |
| 第 15 课 | 线上画展 | 我的画作 |

## 第 5 步：美化与调整

检查网站：每个板块都有内容吗？导航能跳转吗？风格统一吗？跟 Claude Code 说你想调整的地方。

## 第 6 步：毕业庆祝！

请爸爸妈妈拿出毕业证书，一起填写。

- [ ] 导航栏正常工作
- [ ] 毕业证书已填写
