# 动手实践：制作Yanny的专属日历

## 30秒魔法时刻

爸爸妈妈在电脑上打开一本做好的AI日历PDF，让Yanny翻一翻：

"看！这是一个小朋友用AI做的日历！每个月都有漂亮的插图，还标了她的生日和活动。等一下，你也能做一本属于自己的日历！"

Yanny睁大眼睛："好漂亮！我也要做！"

---

## 第1步：整理重要日期（7分钟）

拿出一张纸，和爸爸妈妈一起把最重要的日子写下来：

| 类别 | 要填的内容 |
|------|-----------|
| 我的生日 | Yanny：___ 月 ___ 日 |
| 家人生日 | 妈妈：___ 月 ___ 日、爸爸：___ 月 ___ 日 |
| 特别活动 | 钢琴考级：___ 月、游泳比赛：___ 月、其他：___ |
| 重要节日 | 春节：___ 月、中秋节：___ 月、儿童节：6月1日 |

> 小提示：挑最重要的5-8个日子就好，不用全部都写！

---

## 第2步：用豆包画月份插图（15分钟）

每个月选一个主题，让豆包帮你画一张漂亮的插图。

### 2.1 打开豆包图像生成

1. 打开浏览器，输入 `doubao.com`，按回车
2. 登录你的豆包账号
3. 点击左侧导航栏的 **"图像生成"**

### 2.2 写提示词画画

每个月选一个季节主题，按照这个公式写提示词：

> **"儿童插画风格，" + 季节场景 + 主角动作 + "色彩明亮"**

|------|-----------|

| 2月 | 儿童插画风格，红灯笼和烟花，春节喜庆场景，色彩明亮 |

### 2.3 保存图片

1. 看到满意的图片，在图片上 **右键** → **"另存为"**
2. 保存到桌面新建的"日历插图"文件夹
3. 文件名改成"X月-主题.png"，比如"1月-雪人.png"

> 小提示：12个月太多？可以先选6个月做半年历，下次再补！

---

## 第3步：在PPT中排版日历（15分钟）

### 3.1 准备工作

1. 打开 PPT 或 WPS
2. 点击 **"新建"** → 选择 **"空白演示文稿"**
3. 点击 **"设计"** → **"幻灯片大小"** → 选择 **"纵向 A4"**

### 3.2 制作封面（第1页）

1. 点击 **"插入"** → **"图片"** → 选一张最喜欢的月份插图
2. 把图片拉大，铺满整页
3. 点击 **"插入"** → **"文本框"** → 在页面上方画一个框
4. 输入标题：**"Yanny的专属日历 2026"**，字号设为 48-60
5. 再插入一个文本框，在下方写："我的重要日子都在这里！"

### 3.3 制作月份页（每页一个月）

每个月份页分两部分：

**上半部分——放插图：**
1. 点击 **"插入"** → **"图片"** → 选择对应月份的插图
2. 把图片调整到占页面上半部分（大约40-50%）

**下半部分——画日历表格：**
1. 点击 **"插入"** → **"表格"** → 选择 **7列 × 5行**
2. 第一行填星期：周日、周一、周二、周三、周四、周五、周六
3. 下面几行填日期数字（对照日历填写）

**标记重要日子：**
- 生日 → 点那个格子，把文字颜色改成 **红色**，后面写"🎂 生日快乐！"
- 活动 → 文字颜色改成 **蓝色**，后面写活动名，如"🎹 钢琴考级"
- 节日 → 文字颜色改成 **金色**，后面写节日名，如"🧧 春节"

**顶部写月份名：**
- 在表格上方插入文本框，写月份名（如"一月"），字号 32-40

### 3.4 制作封底（最后一页）

1. 插入一个好看的背景颜色
2. 写一句你的话："这是我做的第一本日历！——Yanny"

---

## 第4步：导出PDF（3分钟）

1. 点击左上角 **"文件"** → **"导出"** → 选择 **"PDF格式"**
2. 选择保存到 **桌面**，点击 **"导出"**
3. 等几秒钟，PDF就生成好了
4. 双击打开PDF翻一遍，检查每个月的插图和日期对不对

---

## 成品标准

- [ ] 封面有标题和插图
- [ ] 每个月份页有插图 + 日历表格
- [ ] 重要日期已用颜色标记
- [ ] 有封底，写了结束语
- [ ] 已导出为PDF文件

---

## 成果整理与分享

1. 把日历PDF发给家人，让大家看看你的作品
2. 如果家里有彩色打印机，打印出来贴在墙上，每天都能看到
3. 把电子版保存好，年底再回头看看这一年的重要日子

记住安全口诀：日历上不要写家庭住址和学校全名哦！
