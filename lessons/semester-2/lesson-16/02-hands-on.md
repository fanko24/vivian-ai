# 动手实践：制作Yanny的成长日历

## 30秒魔法时刻

爸爸妈妈在电脑上打开一本做好的成长日历PDF，让Yanny翻一翻：

> "看！这是一个小朋友用真实照片做的日历！每个月都有一张她自己的活动照片，下面标了她的生日和重要活动。比AI画的图更有意义，因为每一张照片都是真实的！"

---

## 第1步：整理重要日期和选照片（10分钟）

### 列出重要日子

拿出一张纸，和爸爸妈妈一起把最重要的日子写下来：

| 类别 | 要填的内容 |
|------|-----------|
| 我的生日 | Yanny：___ 月 ___ 日 |
| 家人生日 | 妈妈：___ 月 ___ 日、爸爸：___ 月 ___ 日 |
| 特别活动 | 钢琴考级：___ 月、游泳比赛：___ 月、其他：___ |
| 重要节日 | 春节：___ 月、中秋节：___ 月、儿童节：6月1日 |

### 给每个月选一张照片

翻看手机里的照片，给每个月选一张 **代表照片**：

| 月份 | 选什么照片 | 一句话描述 |
|------|-----------|-----------|
| 1月 | 冬天的活动照片 | |
| 2月 | 春节或假期照片 | |
| 3月 | 游泳训练/比赛照片 | |
| 4月 | 春天户外活动照片 | |
| 5月 | 钢琴练习照片 | |
| 6月 | 儿童节或学期末照片 | |
| 7月 | 暑假活动照片 | |
| 8月 | 羽毛球训练照片 | |
| 9月 | 开学照片 | |
| 10月 | 国庆假期照片 | |
| 11月 | 画画作品照片 | |
| 12月 | 冬天或年末照片 | |

> 小提示：如果某个月没有合适的照片，可以用一幅画的作品照片代替，或者让豆包帮忙写一句温馨的话！

---

## 第2步：让豆包为每个月写一句话（8分钟）

### 打开豆包

1. 打开浏览器，输入 **doubao.com**
2. 登录后看到聊天界面

### 让豆包写月份语

输入：

> "我在做一本成长日历，每个月有一张我的活动照片。请你帮我为每个月写一句温馨的话，50字以内，描述这个月的活动感受。以下是我的活动：[列出每个月的活动]"

比如豆包可能写：
- 3月："春天来了，我在泳池里像一条小鱼！每一次划水都在变快！"
- 8月："暑假最开心的事就是打羽毛球！挥拍的姿势越来越帅了！"

### 保存月份语

把豆包写的12句话复制保存到文档。

---

## 第3步：在PPT中排版日历（20分钟）

### 准备工作

1. 打开 PPT 或 WPS
2. 点击 **"新建"** → 选择 **"空白演示文稿"**
3. 点击 **"设计"** → **"幻灯片大小"** → 选择 **"纵向 A4"**

### 制作封面（第1页）

1. 插入一张最好看的活动照片作为封面背景
2. 在上方插入文本框，输入：**"Yanny的成长日历 2026"**（48-60号字）
3. 下方写："我的每一个月都很精彩！"

### 制作月份页（每页一个月）

每个月份页分三部分：

**上半部分——放真实照片：**
1. 点击 **"插入" → "图片"** → 选择对应月份的活动照片
2. 把图片调整到占页面上方约40%

**中间——月份语：**
1. 在照片下方插入文本框
2. 粘贴豆包为这个月写的那句话
3. 字号20-24号，可以用彩色字体

**下半部分——日历表格：**
1. 点击 **"插入" → "表格"** → 选择 **7列 × 5行**
2. 第一行填星期：周日、周一……周六
3. 下面几行填日期数字（对照日历填写）

**标记重要日子：**
- 生日 → 文字颜色改成 **红色** + "🎂"
- 活动 → 文字颜色改成 **蓝色** + 活动名
- 节日 → 文字颜色改成 **金色** + 节日名

**顶部写月份名：**
- 在照片上方插入文本框，写月份名（如"三月"），字号32-40

### 制作封底

1. 新建一页
2. 写一句你的话："这是我用真实照片做的日历，每个月都是真实的我！——Yanny"

---

## 第4步：导出PDF（2分钟）

1. 点击 **"文件" → "导出"** → 选择 **"PDF格式"**
2. 保存到桌面，双击打开翻一遍检查

---

## 成品标准

- [ ] 封面有标题和真实照片
- [ ] 每个月份页有真实活动照片 + 月份语 + 日历表格
- [ ] 重要日期已用颜色标记
- [ ] 有封底
- [ ] 已导出为PDF文件

---

## 成果整理与分享

1. 把日历PDF发给家人
2. 如果有彩色打印机，打印出来贴在墙上
3. 年底再翻看，回忆每个月的精彩瞬间

记住安全口诀：日历上不要写家庭住址和学校全名哦！
