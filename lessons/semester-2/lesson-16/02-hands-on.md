# 动手实践：制作Yanny的专属日历

## 30秒魔法时刻

爸爸妈妈展示一本做好的AI日历（PDF或打印版），让Yanny翻一翻："看，这是一个小朋友用AI画的日历！每个月都有漂亮的插图，还标了她的生日和活动。等一下，你也能做一本！"

---

## 第一步：整理重要日期（7分钟）

拿出一张纸，把最重要的日子写下来：

| 类别 | 要填的内容 |
|------|-----------|
| 我的生日 | Yanny：___ 月 ___ 日 |
| 家人生日 | 妈妈：___ 月 ___ 日、爸爸：___ 月 ___ 日 |
| 特别活动 | 钢琴考级：___ 月、游泳比赛：___ 月、其他：___ |
| 重要节日 | 春节：___ 月、中秋节：___ 月、儿童节：6月1日 |

> 小提示：挑最重要的5-8个日子就好！

---

## 第二步：用豆包画月份插图（15分钟）

1. 打开 `doubao.com`，登录豆包账号，点击"AI绘画"
2. 每个月选一个季节主题，用公式写提示词：
   **"儿童插画风格，" + 季节场景 + 主角动作 + "色彩明亮"**
3. 月度主题参考：
   - **春（2-4月）**：春节红灯笼 / 樱花放风筝 / 油菜花田野
   - **夏（5-7月）**：池塘蜻蜓 / 游乐园气球 / 海滩沙堡
   - **秋（8-10月）**：吃西瓜 / 背书包上学 / 国庆烟花
   - **冬（11-12月）**：踩落叶 / 圣诞树和礼物
4. 每张保存为"X月-主题.png"。12个月太多？先选6个月做半年历

---

## 第三步：在PPT中排版日历（15分钟）

### 准备工作
打开 PPT/WPS，新建空白文稿，页面设为**纵向 A4**

### 封面（第1页）
- 插入一张喜欢的插图作背景
- 上方大标题（48-60号字）：**"Yanny的专属日历 2026"**
- 下方写："我的重要日子都在这里！"

### 月份页（每页一个月）
1. **上半部分**：插入月份插图，占页面40-50%
2. **下半部分**：插入7列×5-6行表格，填写日期
3. **标记重要日子**：
   - 生日 → 红色格子 + "🎂 生日快乐！"
   - 活动 → 对应颜色 + 活动名（如"🎹 钢琴考级"）
   - 节日 → 金色格子 + 节日名（如"🧧 春节"）
4. 顶部写月份名（32-40号字）

### 封底（最后一页）
写一句你的话："这是我做的第一本日历！——Yanny"

---

## 第四步：导出PDF（3分钟）
1. 文件 → 导出 → 创建PDF，保存到桌面
2. 打开PDF翻一遍看看效果
3. 有彩色打印机就打印出来，贴在墙上！

---

## 成品标准

- [ ] 封面 + 6-12页月份页 + 封底
- [ ] 每页有插图 + 日历表格，重要日期已标颜色
- [ ] AI插图风格统一，已导出PDF

---

## 分享你的作品

把日历PDF发给家人，打印出来贴墙上每天看！记住安全口诀：日历上不要写家庭住址和学校全名哦。
