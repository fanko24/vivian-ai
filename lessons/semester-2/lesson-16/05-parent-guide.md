# 家长指导手册：第16课 - Yanny的专属日历

## 课前准备

1. **整理Yanny的重要日期**：提前回忆Yanny的生日、家人生日、钢琴考级、游泳比赛等活动日期
2. 确认电脑已安装 PPT 或 WPS（免费版即可）
3. 准备彩色打印机（可选，导出PDF即可）
4. 如果打印，准备A4纸或稍厚的彩喷纸

## 课堂引导

### 帮忙整理日期
- 可以只记住月份，不一定要精确到具体哪一天
- 活动类标注大致时间即可（如"暑假"）
- 提醒孩子想想：哪些日子是你最期待的？

### 引导选主题图
- 春天的月份画花、风筝、小动物
- 夏天的月份画海滩、西瓜、冰淇淋
- 秋天的月份画落叶、丰收、开学
- 冬天的月份画雪人、圣诞、新年

### AI绘画提示词
- 每张图都要加上"儿童插画风格"，保持风格统一
- 避免具体人脸，用"小女孩""小朋友"等泛指
- 色调保持温暖明亮

### PPT排版协助
- 帮孩子设置PPT页面为纵向A4
- 插入表格：插入 → 表格 → 7列×5-6行
- 填日期是很好的数学练习，让孩子自己算
- 导出PDF：文件 → 导出 → 创建PDF

## 打印和使用建议

- **打印**：彩色单面打印，A4纸，选"实际大小"
- **装订**：打孔+挂绳挂墙上 / 放活页夹 / 磁铁贴吸冰箱
- **日常使用**：每月翻新页，过完一天划掉或贴贴纸

## 安全注意事项

1. 日历上不要写家庭住址、学校全名
2. 分享日历照片时注意遮挡个人日期信息
3. 只发给熟悉的家人

## 学习观察

- **时间观念**：能否理解"月份"和"日期"的关系
- **规划能力**：能否区分重要日期和普通日期
- **审美能力**：对插图风格和颜色搭配的判断
- **数学能力**：填日历表格时对日期排列的理解

**温馨提示**：这本日历是孩子对"时间"概念的第一次动手实践，做好后可以用一整年！
