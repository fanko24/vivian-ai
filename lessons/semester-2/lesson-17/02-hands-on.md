# 动手实践：制作游泳比赛时间线网页

## 30秒魔法时刻

家长打开 Claude Code，输入：

> "帮我做一个时间线风格的网页，蓝色水波纹背景，三个时间节点，清爽活泼的风格。"

一个漂亮的时间线骨架瞬间出现！

"今天我们要把Yanny的游泳故事放进去！"

---

## 第一步：整理素材（10分钟）

这一步需要家长帮忙整理文件，Yanny来选照片和视频。

### 1.1 创建文件夹

1. 在电脑桌面上，新建一个文件夹，名字叫 `游泳时间线`
2. 在这个文件夹里面，再新建3个子文件夹：
   - `第1次比赛`
   - `第2次比赛`
   - `第3次比赛`

最终结构像这样：
```
游泳时间线/
├── 第1次比赛/
├── 第2次比赛/
└── 第3次比赛/
```

### 1.2 挑选照片和视频

和Yanny一起，按顺序回忆每次比赛，每次比赛挑选：
- **2-3张照片**：建议选「赛前准备」「比赛中」「赛后开心」各一张
- **1个短视频**：选最精彩的一段（不超过30秒，如果没有视频也没关系）

### 1.3 给文件改名字

把选好的照片和视频复制到对应的文件夹，并按规则改名：

**第一次比赛文件夹**（示例）：
```
第1次比赛/
├── 01-准备.jpg       （赛前的照片）
├── 02-比赛中.jpg     （比赛中的照片）
├── 03-赛后.jpg       （赛后的照片）
└── 04-视频.mp4       （短视频，可选）
```

**第二次比赛文件夹**（同样格式）：
```
第2次比赛/
├── 01-准备.jpg
├── 02-比赛中.jpg
├── 03-赛后.jpg
```

**第三次比赛文件夹**（同样格式）：
```
第3次比赛/
├── 01-准备.jpg
├── 02-比赛中.jpg
├── 03-赛后.jpg
```

> 小提示：照片用 `.jpg` 或 `.png` 格式都行，视频用 `.mp4` 格式。如果照片太大（超过5MB），可以让家长帮忙压缩一下。

### 1.4 记录比赛信息

拿出一张纸，填写每次比赛的基本信息：

| | 第一次比赛 | 第二次比赛 | 第三次比赛 |
|---|-----------|-----------|-----------|
| 比赛名称 | | | |
| 日期 | | | |
| 我的成绩 | | | |
| 最难忘的事 | | | |
| 我的心情 | | | |

> 小提示：如果有的信息记不清了，可以问爸爸妈妈，或者翻翻朋友圈找记录。

---

## 第二步：写每场比赛的小故事（10分钟）

Yanny口述，家长帮忙记录，然后用豆包把故事润色得更生动。

### 2.1 先自己写草稿

拿出一张纸，Yanny回答下面的问题（每次比赛都回答一遍）：

**第一次比赛：**
- 比赛前你心里在想什么？
- 比赛中最难忘的一瞬间是什么？
- 比赛后你的心情怎样？

把Yanny说的原话记下来就好，不用很完整。

### 2.2 用豆包润色故事

1. **打开豆包**：浏览器输入 `doubao.com`，按回车
2. **输入润色要求**：在对话框中输入下面这段话（把Yanny的原话替换进去）：

> "请帮我把这段小朋友的话润色成一段可爱的小故事（2-3句话），保留小朋友的语气和感受，语言简单温暖：
> 【把Yanny的原话粘贴在这里】"

3. **等待豆包回复**：几秒钟后，豆包会给出一段润色后的故事
4. **读给Yanny听**：问Yanny「这样说好不好？有没有想改的地方？」
5. **如果需要修改**：继续对豆包说「帮我改得更___一些」（比如更勇敢、更开心、更搞笑）

### 2.3 保存三段故事

把三次比赛的故事分别写下来或打字保存。例如：

**第一次比赛的故事（示例）**：
> "第一次参加比赛，我站在跳台上心跳得好快。但跳进水里的那一刻，什么都不怕了！虽然没拿名次，但我特别骄傲，因为我勇敢地完成了比赛！"

**第二次比赛的故事（示例）**：
> "这一次我不紧张了！我听到爸爸妈妈在岸上给我加油，游得比上次快了好多。出来的时候，教练说我进步特别大！"

**第三次比赛的故事（由Yanny来写）**：
> ________________________________________________
> ________________________________________________
> ________________________________________________

> 小提示：故事不用很长，每段2-3句话就够了。最重要的是写出真实的感受！

---

## 第三步：告诉 Claude Code 做网页（15分钟）

这一步需要家长操作终端，Yanny在旁边说出自己的想法。

### 3.1 把素材放到正确位置

1. 打开桌面的 `游泳时间线` 文件夹
2. 确认3个子文件夹里的照片和视频都在
3. 在终端中，用 `cd` 命令进入这个文件夹：
   - 输入 `cd ~/Desktop/游泳时间线`，按回车
   - （如果文件夹不在桌面，把路径换成实际位置）

### 3.2 打开终端并启动 Claude Code

1. **打开终端**：
   - Mac：按 `Command + 空格`，输入 `终端`，按回车
   - Windows：按 `Win + R`，输入 `cmd`，按回车
2. **进入素材文件夹**：输入 `cd ~/Desktop/游泳时间线`，按回车
3. **启动 Claude Code**：输入 `claude`，按回车
4. 等几秒钟，看到光标闪烁就可以输入了

### 3.3 输入Yanny的需求

把下面这段话**完整复制**，把 `___` 替换成实际内容，粘贴到 Claude Code 中：

> "帮我在当前文件夹下做一个时间线风格的网页，要求如下：
> 1. 标题写「Yanny的游泳故事」，要大字、蓝色
> 2. 背景用蓝色的水波纹风格，清爽活泼
> 3. 从上到下展示我参加的三次游泳比赛，用一条竖线连起来
> 4. 每次比赛包含：比赛名称、日期、我的成绩、一段小故事、照片
> 5. 第一次比赛：___（填比赛名称），___月___日，成绩___，故事是___
> 6. 第二次比赛：___，___月___日，成绩___，故事是___
> 7. 第三次比赛：___，___月___日，成绩___，故事是___
> 8. 照片文件在当前文件夹的子文件夹中：第1次比赛/、第2次比赛/、第3次比赛/
> 9. 请把所有代码和内容写在一个 index.html 文件中
> 10. 页面最下面加一句话：「每一次跳入水中，都是一次勇敢的冒险！Yanny加油💪」"

### 3.4 等待生成

- Claude Code 会开始写代码，屏幕上会滚动显示文字
- 这个过程大约需要 30-60 秒，耐心等待
- 看到提示符重新出现，说明生成完毕

### 3.5 在浏览器中查看

1. 打开「访达」（Mac）或「文件资源管理器」（Windows）
2. 进入 `游泳时间线` 文件夹
3. 找到 `index.html` 文件，双击打开
4. 浏览器会自动启动并显示时间线网页
5. 检查一下：三场比赛都显示了吗？照片能看到吗？故事对不对？

> 如果照片没有显示，别着急，下一步会调整！

---

## 第四步：添加细节和调整（10分钟）

看看效果，和Yanny一起讨论哪里需要改。每次只改1-2个地方，改完刷新浏览器看效果。

**具体调整方法**：回到 Claude Code 的终端窗口，输入下面的提示词：

### 如果照片没有显示
输入：
> "照片文件在子文件夹中，请检查图片路径。第1次比赛的照片在 '第1次比赛/01-准备.jpg' 这样的路径"

### 如果想让成绩更醒目
输入：
> "每次比赛的成绩用大大的数字显示，放在故事前面，加上奖牌emoji 🏅"

### 如果想加上每场比赛的标签
输入：
> "在第一次比赛旁边加一个标签写着「勇敢的起点」
> 在第二次比赛旁边加一个标签写着「越来越好」
> 在第三次比赛旁边加一个标签写着「继续加油」"

### 如果想让时间线更好看
输入：
> "时间线的竖线用蓝色渐变，每个节点用圆形的水滴💧图标，节点之间加上虚线连接"

### 如果想加庆祝效果
输入：
> "在最后一次比赛旁边加上一个小小的庆祝动画，比如缓缓飘落的星星✨"

### 如果字体不够可爱
输入：
> "所有文字用圆滚滚的字体，标题用大号加粗，故事文字稍微小一点但容易阅读"

### 如果想在手机上也能看
输入：
> "让这个网页在手机上也能正常显示，照片自适应屏幕大小"

**调整小技巧**：
- 每次改完，在浏览器中按 `F5` 或 `Cmd+R` 刷新页面看效果
- 不满意就继续输入新的调整要求
- 可以告诉 Claude Code："改回刚才的样子" 来撤销修改

---

## 第五步：预览和分享（5分钟）

### 最终检查

1. 在浏览器中打开 `index.html`，按 `F11`（Mac 按 `Cmd+Ctrl+F`）全屏预览
2. 逐项检查：
   - 三场比赛的故事都完整吗？
   - 每张照片都能正常显示吗？
   - 标题和文字有没有错别字？
   - 页面滚动是否流畅？
3. 如果有问题，回到第四步继续调整

### 保存最终版本

1. 在 Claude Code 中输入：
   > "请确认 index.html 文件中所有内容都是完整的，可以直接双击打开使用"
2. 确认 `游泳时间线` 文件夹里有 `index.html` 和3个子文件夹

### 发给家人看（微信分享）

1. 打开微信电脑版
2. 找到要分享的家人聊天窗口
3. 把整个 `游泳时间线` 文件夹压缩成 zip：
   - **Mac**：右键文件夹 → 选择「压缩"游泳时间线"」
   - **Windows**：右键文件夹 → 选择「发送到」→「压缩(zipped)文件夹」
4. 把压缩后的文件拖到微信聊天框发送
5. 家人收到后解压，双击 `index.html` 就能看啦

> 也可以在浏览器中打开网页，按 `Ctrl+P`（Mac 按 `Cmd+P`）保存为 PDF，打印出来贴在墙上！

## 成品标准

完成的作品应该包含：

- [ ] 标题写着「Yanny的游泳故事」，又大又醒目
- [ ] 完整的3次比赛记录（第一次、第二次、第三次）
- [ ] 每次比赛都有：日期、比赛名称、成绩
- [ ] 每次比赛都有一段Yanny自己写的小故事（2-3句话）
- [ ] 每次比赛至少配有1张照片
- [ ] 时间线样式：从上到下排列，有一条竖线连接三次比赛
- [ ] 蓝色水波纹风格的背景，清爽活泼
- [ ] 页面底部有一句鼓励的话
- [ ] 能在浏览器中正常打开，照片都能显示
- [ ] 文件是一个独立的 `index.html`，双击就能看
- [ ] 可以通过微信发给家人，家人也能正常打开
