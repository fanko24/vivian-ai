# 动手实践：制作游泳比赛时间线网页

## 30秒魔法时刻

家长打开 Claude Code，输入：

> "帮我做一个时间线风格的网页，蓝色水波纹背景，三个时间节点，清爽活泼的风格。"

一个漂亮的时间线骨架瞬间出现在屏幕上！

"看！今天我们要把Yanny的游泳故事放进去，变成一个可以分享的网页！"

---

## 第1步：整理素材（10分钟）

这一步需要家长帮忙整理文件，Yanny来选照片和视频。

### 1.1 创建文件夹

1. 在电脑桌面上，新建一个文件夹，名字叫 `游泳时间线`
2. 在里面再新建3个子文件夹：`第1次比赛`、`第2次比赛`、`第3次比赛`

### 1.2 挑选照片并改名

和Yanny一起回忆每次比赛，每次挑选2-3张照片，复制到对应文件夹并改名：
- `第1次比赛/01-准备.jpg`、`02-比赛中.jpg`、`03-赛后.jpg`
- `第2次比赛/01-准备.jpg`、`02-比赛中.jpg`、`03-赛后.jpg`
- `第3次比赛/01-准备.jpg`、`02-比赛中.jpg`、`03-赛后.jpg`

> 小提示：照片用 .jpg 或 .png 格式都行。

### 1.4 记录比赛信息

拿出一张纸，填写每次比赛的信息：

| | 第一次 | 第二次 | 第三次 |
|---|-------|-------|-------|
| 比赛名称 | | | |
| 日期 | | | |
| 我的成绩 | | | |
| 最难忘的事 | | | |
| 我的心情 | | | |

---

## 第2步：用豆包润色故事（10分钟）

Yanny口述比赛经历，家长记录原话，然后用豆包把故事润色得更生动。

### 2.1 写故事草稿

Yanny回答下面的问题（每次比赛都回答一遍）：比赛前你在想什么？比赛中最难忘的一瞬间？比赛后的心情？把原话记下来就好。

### 2.2 用豆包润色

1. 打开浏览器，输入 `doubao.com`，按回车
2. 在对话框中输入：

> "请帮我把这段小朋友的话润色成一段可爱的小故事（2-3句话），保留小朋友的语气和感受，语言简单温暖：
> 【把Yanny的原话粘贴在这里】"

3. 等几秒钟，豆包会给出一段润色后的故事
4. 读给Yanny听，问："这样说好不好？有没有想改的地方？"
5. 想修改就继续说："帮我改得更___一些"（比如更勇敢、更开心）

### 2.3 保存三段故事

把三次比赛的故事分别保存好。每段2-3句话就够了，最重要的是写出真实的感受！

---

## 第3步：用Claude Code生成时间线网页（15分钟）

这一步需要家长操作终端，Yanny在旁边说出自己的想法。

### 3.1 打开终端并启动 Claude Code

1. **打开终端**：Mac 按 `Command + 空格` 输入 `终端`；Windows 按 `Win + R` 输入 `cmd`
2. **进入素材文件夹**：输入 `cd ~/Desktop/游泳时间线`，按回车
3. **启动 Claude Code**：输入 `claude`，按回车，等几秒钟

### 3.2 输入需求

把下面这段话完整复制，把 `___` 替换成实际内容，粘贴到 Claude Code：

> "帮我在当前文件夹下做一个时间线风格的网页，要求如下：
> 1. 标题写「Yanny的游泳故事」，要大字、蓝色
> 2. 背景用蓝色的水波纹风格，清爽活泼
> 3. 从上到下展示我参加的三次游泳比赛，用一条竖线连起来
> 4. 每次比赛包含：比赛名称、日期、我的成绩、一段小故事、照片
> 5. 第一次比赛：___（填比赛名称），___月___日，成绩___，故事是___
> 6. 第二次比赛：___，___月___日，成绩___，故事是___
> 7. 第三次比赛：___，___月___日，成绩___，故事是___
> 8. 照片文件在当前文件夹的子文件夹中
> 9. 请把所有代码写在一个 index.html 文件中
> 10. 页面最下面加一句话：「每一次跳入水中，都是一次勇敢的冒险！Yanny加油」

### 3.3 等待生成并预览

- Claude Code 会开始写代码，大约需要 30-60 秒，耐心等待
- 看到提示符重新出现，说明生成完毕
- 在终端输入 `open index.html`（Mac）或在文件夹中双击 `index.html` 打开
- 浏览器会显示你的时间线网页！

> 如果照片没有显示，别着急，下一步会调整！

---

## 第4步：添加照片并调整（10分钟）

### 4.1 让Claude Code内嵌照片

在 Claude Code 中输入：

> "请把所有图片内嵌到 HTML 文件中，让这个网页可以单独打开，不需要网络也能看"

Claude Code 会把照片嵌入到HTML文件里，这样分享给别人也能看到图片。

### 4.2 美化调整

看看效果，每次只改1-2个地方，改完在浏览器中按 `F5` 或 `Cmd+R` 刷新：

| 想改什么 | 跟 Claude Code 说 |
|---------|-----------------|
| 成绩不够醒目 | "每次比赛的成绩用大大的数字显示，加上奖牌emoji" |
| 想加标签 | "第一次比赛旁边加标签「勇敢的起点」，第二次「越来越好」，第三次「继续加油」" |
| 时间线不够好看 | "竖线用蓝色渐变，节点用水滴图标" |
| 字体不够可爱 | "文字用圆滚滚的字体，标题大号加粗" |
| 手机上不好看 | "让网页在手机上也能正常显示，照片自适应屏幕" |

> 调整小技巧：不满意就说"改回刚才的样子"来撤销修改。

---

## 第5步：预览和分享（5分钟）

### 最终检查

打开 `index.html`，检查：三场比赛故事完整吗？照片都能显示吗？文字有没有错别字？有问题回到第4步继续调整。

### 微信分享给家人

1. 告诉 Claude Code：
   > "请确认 index.html 文件中所有内容都是完整的，可以直接双击打开使用"
2. 把 `index.html` 文件通过微信发给奶奶、姥姥等家人
3. 家人在手机上点击文件 → 选择"用浏览器打开"就能看到你的游泳故事啦！

> 也可以按 `Ctrl+P`（Mac 按 `Cmd+P`）保存为 PDF，打印出来贴在墙上！

---

## 成品标准

- [ ] 标题写着「Yanny的游泳故事」，又大又醒目
- [ ] 完整的3次比赛记录
- [ ] 每次比赛都有日期、名称、成绩和小故事
- [ ] 每次比赛至少配有1张照片
- [ ] 时间线从上到下排列，有竖线连接
- [ ] 蓝色水波纹风格，清爽活泼
- [ ] 页面底部有一句鼓励的话
- [ ] 所有图片已内嵌，文件可以独立打开
- [ ] 可以通过微信发给家人

---

## 成果整理与分享

1. 确认 `index.html` 文件已保存在 `游泳时间线` 文件夹
2. 通过微信发给家人，让大家都看到你的勇敢故事
3. 打印出来做成纸质版，贴在房间里

记住安全口诀：不要在网页上写家庭住址等隐私信息！
