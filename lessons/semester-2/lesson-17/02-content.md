# 知识小课堂

## 1. 用时间线讲故事

什么是时间线？就是按照**时间的顺序**，把发生的事情一个个排列出来。

```
第一次比赛 ──── 第二次比赛 ──── 第三次比赛
 2024年          2025年          2025年
 "紧张但勇敢"   "拿到奖杯"     "冠军！"
```

时间线网页的好处：
- 一眼就能看出事情的发展顺序
- 每个时间点可以放照片、视频、文字
- 像一本翻开的相册，但更有条理

---

## 2. 把照片和视频放进网页

网页上可以展示很多种内容：

| 内容类型 | 怎么放进去 |
|---------|-----------|
| 照片 | 把图片文件放在网页文件夹里 |
| 视频 | 把视频文件放进去，或者用网络链接 |
| 文字 | 直接写在描述里告诉 Claude Code |

告诉 Claude Code 时可以这样说：
"在第一次比赛的位置放一张叫 photo1.jpg 的照片，下面写上我的感受。"

---

## 3. 网页分享

做完的网页怎么给别人看呢？

- **保存成文件**：把整个文件夹发给别人
- **发链接**：如果放到网上，给别人一个网址就能看
- **投屏展示**：连上电视或投影仪，全家人一起看

就像把你画的画贴在墙上给大家欣赏一样，网页也可以分享给想分享的人！

记住安全口诀：网页上不放个人隐私信息哦！
