# 第17课：Yanny的三次游泳比赛

**用时**：55分钟
**工具**：Claude Code + 豆包（doubao.com）
**难度**：⭐⭐⭐

---

## 学习目标

1. 用 Claude Code 制作一个时间线风格的网页
2. 学会把真实照片和视频放进网页里
3. 用网页来讲自己的故事

## 准备清单

- [ ] 电脑已安装 Claude Code
- [ ] 打开豆包网页（doubao.com）
- [ ] 准备好三次游泳比赛的照片和视频（每次2-3张）
- [ ] 了解三次比赛的日期、名称和成绩
