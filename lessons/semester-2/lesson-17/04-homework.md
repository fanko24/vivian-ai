# 课后作业

## 必做作业：分享你的游泳故事（5分钟）

把你做好的时间线网页分享给这些人：

1. **爷爷奶奶**（或外公外婆）——让他们看看Yanny有多棒
2. **游泳教练**——感谢教练的指导

分享方式（选一种）：
- 在电脑上当面展示给家人看
- 家长帮忙把网页文件发给教练
- 连上电视，全家人一起看

分享的时候，试着用自己的话介绍每一场比赛的故事！

---

## 选做作业：添加训练日常专区（15分钟）

除了比赛，平时的训练也很了不起！

用 Claude Code 在网页底部加一个"训练日常"专区：

- 放 2-3 张训练时的照片
- 写一句话描述平时的训练（比如"每周游___米"）
- 加上你的训练口号或座右铭

这样整个网页就不只是比赛记录，而是完整的游泳故事啦！

---

## 完成打卡

- [ ] 网页已经分享给家人
- [ ] 网页已经分享给游泳教练
- [ ] 自己讲解了每场比赛的故事
- [ ] （选做）训练日常专区已完成
