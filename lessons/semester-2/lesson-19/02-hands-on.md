# 动手实践：制作线上画展网页

## 30秒魔法时刻

家长打开 Claude Code，输入：

> "帮我做一个画展网页，标题叫'Yanny的线上画展'，浅米色背景像美术馆一样，每幅画有漂亮的画框，鼠标放上去显示画作名字和介绍。"

几秒钟后，一个漂亮的线上美术馆出现在屏幕上！

"看！这就是你今天要做的东西——一个属于你的线上画展！"

---

## 第1步：挑选画作（5分钟）

1. 打开电脑里保存画作的文件夹
2. 挑选 **5-8幅** 最满意的作品
3. 把选好的画复制到一个新文件夹，命名为 **"画展素材"**
4. 给每幅画重新命名：`01-向日葵的微笑.jpg`、`02-彩虹小猫.jpg`……

**选画小贴士：**
- 选不同类型的画（水彩、蜡笔、铅笔画各几张）
- 选不同主题的画（动物、风景、人物等）
- 每幅画都要是你真正喜欢的

---

## 第2步：用豆包写画作介绍（10分钟）

### 2.1 先自己想想每幅画

拿出一张纸，给每幅画写3个信息：

| 编号 | 画的名字 | 用的材料 | 一句话介绍 |
|------|---------|---------|-----------|
| 1 | | | |
| 2 | | | |
| 3 | | | |
| 4 | | | |
| 5 | | | |

**写介绍的小技巧：** 想想你画这幅画时的心情，用"我觉得……""这幅画让我想到……"开头，一两句话就够了。

### 2.2 用豆包润色介绍

1. 打开浏览器，输入 `doubao.com`，按回车
2. 在对话框中输入：

> "我画了一幅画，叫___（填画的名字），用的是___（填材料：水彩/蜡笔/铅笔）。我画的时候觉得___（写你的感受）。帮我润色成一句话的画作介绍，要温暖可爱。"

3. 豆包会给出一段介绍，读一读喜不喜欢
4. 不满意就说："帮我改得更___一些"
5. 每幅画都重复这个步骤，把所有介绍保存好

---

## 第3步：用Claude Code生成画展网页（20分钟）

### 3.1 打开终端并启动 Claude Code

1. **Mac电脑**：按 `Cmd + 空格`，输入 `终端`，按回车
   **Windows电脑**：按 `Win + R`，输入 `cmd`，按回车
2. 在终端里输入 `claude`，按回车
3. 等几秒钟，看到输入提示就可以开始了

### 3.2 输入需求

把下面这段话输入 Claude Code（把括号里的内容换成你自己的）：

> "请帮我做一个画展网页，要求：
> 1. 标题是「Yanny的线上画展」
> 2. 下面写「小画家 · 8岁 · 热爱画画」
> 3. 中间是画作展示区，一排3幅画
> 4. 每幅画要有漂亮的画框效果，深棕色画框
> 5. 鼠标放在画上时，显示画的名字和介绍
> 6. 背景用浅米色，整体风格像真正的美术馆
> 7. 网页底部写「感谢你来看我的画展！」
> 8. 把所有代码写在一个 index.html 文件中"

### 3.3 添加画作

1. 把画作图片放到和 `index.html` 同一个文件夹里
2. 回到 Claude Code，输入：

> "这些是我的画作，请把它们放到画廊里。
> 每幅画的信息：
> - 第1幅：名字「___」，___画，介绍「___」
> - 第2幅：名字「___」，___画，介绍「___」
> - 第3幅：名字「___」，___画，介绍「___」
> ……（把所有画作信息都写上）"

3. 也可以把图片文件直接 **拖到终端窗口** 里，Claude Code 会自动识别文件路径

---

## 第4步：调整画框效果并内嵌图片（10分钟）

### 4.1 查看并调整

看看网页效果，跟爸爸妈妈讨论想改什么：

| 想改什么 | 跟 Claude Code 说 |
|---------|-----------------|
| 画框不够粗 | "画框再粗一点，用深棕色，加一点阴影效果" |
| 画之间的间距太窄 | "画和画之间的间距再大一些" |
| 标题不够大 | "标题字体再大一些，加上装饰" |
| 想加背景图案 | "加一个浅灰色的格子背景图案" |
| 鼠标效果不好看 | "鼠标放上去的时候，画作慢慢放大，有一个柔和的阴影" |
| 想加音乐 | "页面打开时播放一段轻柔的背景音乐" |

**每次只改1-2个地方，改完在浏览器中按 `F5` 或 `Cmd+R` 刷新看效果。**

### 4.2 内嵌图片

调整满意后，告诉 Claude Code：

> "请把所有图片内嵌到 HTML 文件中，让这个网页可以单独打开，不需要网络也能看"

Claude Code 会把所有画作嵌入到HTML文件里。

---

## 第5步：预览和分享（5分钟）

### 最终检查

打开 `index.html`，逐项检查：
- [ ] 标题「Yanny的线上画展」和副标题显示正确
- [ ] 所有画作都显示了
- [ ] 每幅画有画框效果
- [ ] 鼠标放上去能看到名字和介绍
- [ ] 背景颜色和整体风格好看
- [ ] 底部文字显示正常
- [ ] 所有图片已内嵌，文件可以独立打开

### 分享你的作品

1. 把 `index.html` 通过微信发给家人
2. 告诉朋友："我的画在网上办画展了！"
3. 家人在手机上点击文件 → 选择"用浏览器打开"就能看到你的画展啦！

> 也可以请全家人围在电脑前，像真正的画展开幕式一样，一幅一幅欣赏！

---

## 成品标准

- [ ] 标题区：画展名称「Yanny的线上画展」+ 小画家信息
- [ ] 5-8幅画作展示，有深棕色画框效果
- [ ] 每幅画有名字和一句话介绍
- [ ] 鼠标悬停显示详情，有放大效果
- [ ] 美术馆风格的整体设计（浅米色背景）
- [ ] 所有图片已内嵌，文件可独立打开
- [ ] 网页底部有结束语

---

## 成果整理与分享

1. 确认 `index.html` 文件已保存在"画展素材"文件夹
2. 通过微信发给家人和朋友
3. 这个画展永远在线！以后画了新画，还可以回来添加

记住安全口诀！网页上不要出现家庭住址和学校名称哦！
