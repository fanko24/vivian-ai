# 动手实践：制作线上画展网页

## 30秒魔法时刻

爸爸妈妈打开一个做好的画展网页，让Yanny看看：

"看！这是一个小朋友用Claude Code做的网上画展。每幅画都有漂亮的画框，点一下还能看到介绍。今天你的画也能变成这样！"

---

## 第一步：挑选画作（5分钟）

1. 打开电脑里保存画作的文件夹
2. 挑选 **5-8幅** 最满意的作品
3. 把选好的画复制到一个新文件夹，命名为"画展素材"
4. 给每幅画重新命名：`01-向日葵的微笑.jpg`、`02-彩虹小猫.jpg`……

**选画小贴士：**
- 选不同类型的画（水彩、蜡笔、铅笔画各几张）
- 选不同主题的画（动物、风景、人物等）
- 每幅画都要是你真正喜欢的

---

## 第二步：画网页草图（5分钟）

在纸上画出你想要的网页样子：

```
┌─────────────────────────────┐
│     ✨ Yanny的线上画展 ✨      │
│     小画家 · 8岁 · 爱画画      │
├─────────────────────────────┤
│  ┌────┐  ┌────┐  ┌────┐    │
│  │ 画1 │  │ 画2 │  │ 画3 │    │
│  │名字 │  │名字 │  │名字 │    │
│  └────┘  └────┘  └────┘    │
│  ┌────┐  ┌────┐  ┌────┐    │
│  │ 画4 │  │ 画5 │  │ 画6 │    │
│  │名字 │  │名字 │  │名字 │    │
│  └────┘  └────┘  └────┘    │
├─────────────────────────────┤
│   感谢你来看我的画展！         │
└─────────────────────────────┘
```

**在草纸上标注：**
- 画展叫什么名字
- 一排放几幅画（2幅或3幅）
- 喜欢什么颜色的背景

---

## 第三步：准备画作介绍（5分钟）

给每幅画写3个信息，写在纸上或让家长帮忙打字：

**表格模板：**

| 编号 | 画的名字 | 用的材料 | 一句话介绍 |
|------|---------|---------|-----------|
| 1 | | | |
| 2 | | | |
| 3 | | | |
| 4 | | | |
| 5 | | | |
| 6 | | | |
| 7 | | | |

**写介绍的小技巧：**
- 想想你画这幅画时的心情
- 用"我觉得……""这幅画让我想到……"开头
- 一两句话就够了，不用写太长

---

## 第四步：用Claude Code制作网页（20分钟）

### 4.1 启动Claude Code

1. 打开终端
2. 输入 `claude`，按回车
3. Claude Code启动成功！

### 4.2 告诉Claude Code你的想法

输入以下内容（把括号里的内容换成你自己的）：

```
请帮我做一个画展网页，要求：
1. 标题是"Yanny的线上画展"
2. 下面写"小画家 · 8岁 · 热爱画画"
3. 中间是画作展示区，一排3幅画
4. 每幅画要有漂亮的画框效果
5. 鼠标放在画上时，显示画的名字和介绍
6. 背景用浅米色，整体风格像真正的美术馆
7. 网页底部写"感谢你来看我的画展！"
```

### 4.3 添加画作

把画作的图片文件拖到终端里，告诉Claude Code：

```
这些是我的画作，请把它们放到画廊里。
[把图片文件拖进来]

每幅画的信息：
- 第1幅：名字"向日葵的微笑"，水彩画，介绍"向日葵像太阳一样温暖"
- 第2幅：……（把你在第三步写的表格内容输入进去）
```

### 4.4 调整和美化

看看效果，告诉Claude Code调整：

```
请帮我调整：
1. 画框再粗一点，用深棕色
2. 画和画之间的间距再大一些
3. 标题字体再大一些
4. 加一个浅灰色的背景图案
```

---

## 第五步：预览和保存（5分钟）

1. 在浏览器中打开HTML文件，检查效果
2. 翻一翻所有画，看看排列和介绍对不对

**检查清单：**
- [ ] 标题和副标题显示正确
- [ ] 所有画作都显示了
- [ ] 每幅画有画框效果
- [ ] 鼠标放上去能看到名字和介绍
- [ ] 背景颜色和整体风格好看
- [ ] 底部文字显示正常

---

## 成品标准

完成的画展网页应该包含：

- [ ] 标题区：画展名称 + 小画家信息
- [ ] 5-8幅画作展示，有画框效果
- [ ] 每幅画有名字和一句话介绍
- [ ] 鼠标悬停显示详情
- [ ] 美术馆风格的整体设计
- [ ] 能在浏览器中正常打开

---

## 分享你的作品

1. 把画展网页截图发给家人
2. 告诉朋友："我的画在网上办画展了！"
3. 也可以把画展链接分享给老师和同学

> 记住安全口诀！网页上不要出现家庭住址和学校名称。
