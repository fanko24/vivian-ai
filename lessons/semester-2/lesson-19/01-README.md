# 第19课：Yanny的线上画展

> 用Claude Code制作画展网页，展示自己的画作

## 课程信息

- **时长**：55分钟
- **工具**：Claude Code + 豆包（doubao.com）
- **难度**：⭐⭐⭐⭐

## 学习目标

1. 用Claude Code制作一个漂亮的画展网页
2. 学会在网页中展示图片画廊
3. 为每幅画写介绍，配上画框效果

## 准备清单

- [ ] Claude Code 已安装并能正常启动
- [ ] 收集5-8张自己的画作照片（存在电脑上）
- [ ] 打开豆包（doubao.com）备用
- [ ] 准备一张纸和彩色笔（画网页草图用）
- [ ] 家长陪伴

## 时间安排

| 环节 | 时间 |
|------|------|
| 30秒魔法时刻 | 3分钟 |
| 规划网页 | 7分钟 |
| 知识讲解 | 5分钟 |
| 动手实践 | 30分钟 |
| 作品分享 | 10分钟 |
