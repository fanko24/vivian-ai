# CLAUDE.md

本文件为 Claude Code (claude.ai/code) 在此仓库中工作时提供指导。

## 项目概述

Vivian AI 是专为8岁一年级小女孩设计的AI启蒙课程。内容全部使用中文编写，通过"先体验再学习"的方式，先让孩子感受AI的神奇，再逐步学会使用AI工具。

## 课程文件结构

每节课包含6个标准文件：

| 文件 | 用途 | 行数限制 |
|------|------|----------|
| `01-README.md` | 课程概览、学习目标、准备清单 | ≤40行 |
| `02-story.md` | 故事引入，激发孩子兴趣 | ≤60行 |
| `03-content.md` | 知识讲解（3-5个概念） | ≤60行 |
| `04-hands-on.md` | 动手实践步骤（含30秒魔法时刻） | ≤80行 |
| `05-homework.md` | 课后作业（1必做+1选做） | ≤50行 |
| `06-parent-guide.md` | 家长指导 | ≤60行 |

## 写作规范

- **语言**：全部使用简体中文
- **目标读者**：8岁一年级儿童及其家长
- **语气**：亲切友好、鼓励性强、适合儿童理解
- **格式**：适当使用emoji增加趣味性，但不过量
- **AI工具推荐**：首选豆包（doubao.com），字节跳动旗下AI助手，全平台免费
- **安全提示**：第1课教安全口诀，后续课只提醒"记住安全口诀"
- **避免术语**：不使用"神经网络""深度学习"等抽象概念

## 内容要求

1. 每课以"30秒魔法时刻"开场，先展示AI效果再教操作
2. 语言要简单易懂，一年级孩子能听懂
3. 课后作业控制在10-15分钟，接受非文字形式（语音、绘画、截图）
4. 新增或修改课程时更新 `docs/progress.md`
5. 动手实践内容需符合 `docs/safety-guide.md` 的安全指南
