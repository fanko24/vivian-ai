# Vivian AI 课程总览

> 专为8岁小女孩设计的AI启蒙课程

## 课程简介

Vivian AI 是一套专为8岁左右一年级小女孩设计的AI启蒙课程。通过"先体验再学习"的方式，让孩子先感受到AI的神奇，再逐步学会使用AI，培养面向未来的数字素养。

## 设计理念

### 🎮 先体验再学习
- 每节课以"30秒魔法时刻"开场
- 先让孩子看到AI能做到什么，再学习怎么做
- 激发好奇心，让学习变成探索

### 📈 循序渐进
- 第一学期：体验AI，感受AI的各种能力
- 第二学期：用AI做项目，把兴趣变成作品

### 🎨 创意表达
- 每课都有实际作品产出
- 接受多种表达形式（绘画、语音、截图、视频）
- 降低书写要求，鼓励创意

### 🛡️ 安全第一
- 第1课建立安全口诀，后续课只需提醒
- 推荐工具都有内容过滤功能
- 强调家长陪伴

## 课程结构

### 第一学期：AI魔法探险（8周）

| 课时 | 课程名称 | 核心体验 |
|------|----------|----------|
| 第1课 | 和AI聊聊天 | 第一次与AI对话，感受AI的智能 |
| 第2课 | AI画梦工厂 | 用文字生成图片，体验AI绘画 |
| 第3课 | 和AI做朋友 | 深入学习对话技巧，学会问好问题 |
| 第4课 | AI绘画大师 | 进阶创意绘画，探索不同风格 |
| 第5课 | AI会说话 | 体验AI语音交互和朗读 |
| 第6课 | AI音乐工厂 | 用AI工具生成真正的歌曲 |
| 第7课 | AI视频魔术 | 用文字生成短视频 |
| 第8课 | AI小老师 | 用AI辅助学习，探索科学问题 |

### 第二学期：原一的AI创作工作室（8周）

| 课时 | 课程名称 | 核心体验 |
|------|----------|----------|
| 第9课 | 我的游泳日记 | 用豆包记录游泳经历，制作电子画册 |
| 第10课 | 原一的画画作品集 | 整理画作，用AI写作品介绍 |
| 第11课 | 我的钢琴第一张专辑 | 用AI创作音乐，设计专辑封面 |
| 第12课 | 羽毛球小教练手册 | 用AI画教学示意图，生成演示视频 |
| 第13课 | 原一的专属日历 | 用Claude Code制作个性化日历 |
| 第14课 | 原一的三次游泳比赛 | 用Claude Code制作时间线网页 |
| 第15课 | 原一的线上画展 | 用Claude Code制作画展网页 |
| 第16课 | 刘原一的个人网站 | 综合所有作品，制作个人网站 |

## 学习目标

完成本课程后，孩子将能够：

1. **认识AI** - 知道AI是什么，能做什么
2. **使用AI** - 会用AI对话、画画、创作音乐和视频
3. **创作表达** - 能用AI辅助创作自己的作品
4. **安全使用** - 知道保护个人信息，合理控制使用时间
5. **正确学习** - 理解AI是学习伙伴，不是答案机器
