## 工具使用指南

### 豆包使用方法

1. **注册账号**
   - 访问 https://www.doubao.com/ 或下载豆包App
   - 使用手机号注册登录

2. **开始对话**
   - 在输入框输入问题
   - 点击发送或按回车

3. **绘画功能**
   - 输入"帮我画一个..."或"画一张..."
   - 描述想要的图片
   - 也可以点击"AI绘画"功能入口

4. **语音功能**
   - 点击语音通话按钮
   - 开始语音对话

5. **音乐功能**
   - 在豆包中描述你想要的音乐风格和歌词
   - 等待生成

6. **视频功能**
   - 使用豆包的Seedance功能
   - 输入描述即可生成短视频

## 安全注意事项

1. **不要分享个人信息**
   - 不告诉AI真实姓名
   - 不说家庭住址
   - 不说电话号码

2. **家长陪伴**
   - 首次使用需要家长指导
   - 遇到问题及时询问

3. **内容安全**
   - 不问不适当的问题
   - 遇到奇怪的内容告诉家长

4. **开启青少年模式**
   - 豆包支持青少年模式
   - 可以过滤不适合小朋友的内容
