# Vivian AI - 8岁小女孩AI启蒙课程

> 让每个小女孩都能轻松踏入AI世界

## 项目介绍

Vivian AI 是专为8岁一年级小女孩设计的AI启蒙课程。通过有趣的故事、互动的实践和创意的项目，让孩子们在轻松愉快的氛围中了解AI、使用AI，成为小小AI专家。

## 课程特色

- **趣味故事引导**：每节课都有生动的故事引入
- **动手实践**：简单易懂的操作步骤
- **渐进式学习**：从对话到创作，逐步深入
- **亲子互动**：家长指导手册帮助陪伴学习

## 课程结构

### 第一学期：AI魔法探险（8周）

1. 和AI聊聊天
2. AI画梦工厂
3. 和AI做朋友
4. AI绘画大师
5. AI会说话
6. AI音乐工厂
7. AI视频魔术
8. AI小老师

- 8岁左右的一年级小女孩

## 课程理念

- **寓教于乐**：通过游戏化学习激发兴趣
- **循序渐进**：从简单到复杂，适合儿童认知发展
- **创意表达**：鼓励孩子用AI表达自己的想法
- **安全第一**：所有工具都经过筛选，确保安全
